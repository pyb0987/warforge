카드는 자리를 선택해서 놓을 수 있지만, 한번 놓으면 자유 재배치 불가.
배치를 바꾸려면 "배치 변경" 종류의 효과를 사용해야 한다.

**Why:** 게임 설계상 배치 순서가 전략적 의사결정. 자유 재배치를 허용하면 배치의 무게감이 사라짐.

**How to apply:**
- 시뮬레이터의 "고정 보드"는 실제 게임과 가까운 모델 (배치 외 의사결정인 상점/★합성/업그레이드만 빠짐)
- 시뮬 승률은 "해당 보드의 최대 성능"이 아니라 "실제 플레이에 가까운 결과"로 해석 가능
- 배치 관련 카드 효과 설계 시 고려 (배치 변경 = 희소한 전략적 자원)
