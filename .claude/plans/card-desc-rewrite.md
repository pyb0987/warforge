## 설명 표준 (Style Guide)

### 포맷
```
[트리거]: [대상에] [효과]. [조건]
```

### 규칙
1. **★별 독립성**: ★2/★3도 트리거+대상+효과를 완전히 포함. "★1과 같되 N→M" 형식 금지.
2. **트리거 명시**: 모든 설명의 첫 단어가 트리거. 생략 불가.
   - 라운드 시작 / 전투 시작 / 전투 종료 / 패배 시 / [반응] / [지속] / 판매 시
3. **대상 명시**: 위치 기반 표현으로 대상을 명시.
   - 이 카드 / 오른쪽 인접 카드 / 양쪽 인접 카드 / 필드 위 모든 카드 / 필드 위 모든 [테마] 카드 / [반응 대상 카드]
4. **키워드 첫 등장 시 괄호 보충** (글로서리 구현 전 임시):
   - "부화(유충 유닛 추가) 2기"
   - "변태(유충→상위 유닛 교체) 2기"
   - 글로서리 구현 후에는 키워드만 사용 + 호버로 정의 표시
5. **수치 명시**: "강화" → "ATK +5% 영구". 퍼센트/절대값/영구/전투한정 구분.
6. **발동 상한 포함**: "(최대 N/R)" 을 효과 끝에 표기.

### 예시 (before → after)

**ne_wild_pulse ★2 (before)**:
```
강화 ATK +5%
```

**ne_wild_pulse ★2 (after)**:
```
라운드 시작: 이 카드 ATK +5% 영구 강화 (최대 1/R)
```

**pr_transcend ★1 (before)**:
```
라운드 시작: 자기 부화 3기, 전체 포식종 부화 1기. [지속] 전투 중 포식종 사망마다 생존 ATK +3% 누적. 자기 적 처치 HP 10% 회복
```

**pr_transcend ★1 (after)**:
```
라운드 시작: 이 카드 부화(유충 추가) 3기 + 필드 위 모든 포식종 카드 부화 1기. [지속] 전투 중 포식종 유닛 사망 시 생존 포식종 전체 ATK +3% 누적(전투 한정). 이 카드 유닛이 적 처치 시 HP 10% 회복
```

## Risks

| Risk | Impact | Mitigation |
|---|---|---|
| ★2/★3 효과 합성 오류 — 기획문서 델타를 잘못 합성 | 카드 효과 변경 (치명적) | 기획문서 원본 + card_db.gd 코드 교차 검증. 모호하면 사용자 확인 |

| card_tooltip.gd 수정 시 기존 렌더링 깨짐 | UI 버그 | effects[] 직접 렌더링 카드 목록 확인 후 수정 |
