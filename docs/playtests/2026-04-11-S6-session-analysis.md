## 핵심 인사이트

### 1. 스팀펑크 제조 → 복리 성장 문제
증기 용광로의 "제조 시 개량" 효과가 복리로 작동:
- 제조 → 유닛 증가 → CP 증가
- 제조 → 개량(ATK 증가) → 기존 유닛 전체에 적용 → CP 증가
- 두 효과가 곱연산으로 결합 → R1 대비 74배

다른 테마(드루이드, 포식종, 군대)가 이 성장률에 근접할 수 있는지 검증 필요.

### 2. T5 풀 고갈 → ★3 달성 난이도
T5 풀: 카드당 11장. 상점 Lv6에서 T5 출현 비중이 높아, 리롤 1~2회로 같은 T5 카드 2~3장을 매 라운드 확보 가능. 4라운드 만에 ★3 완성은 TFT 기준으로도 매우 빠름.

### 3. 제국병기창 버그가 게임플레이에 영향
병기창의 핵심 컨셉인 "판매 흡수"가 작동하지 않아, ★3까지 만들었음에도 기대한 파워 스파이크를 경험하지 못함. 수정 시 밸런스 재검토 필요.

### 4. 정보 부족으로 인한 불확실성
R11~R13에서 다수의 "이 효과가 정확히 뭘 하는지 모르겠다" 피드백. 카드 효과 텍스트 리라이트(이전 세션에서 완료)가 이 문제를 얼마나 해결했는지 다음 플레이에서 확인 필요.
