# CP 밸런싱 원칙 변경

## 날짜
2026-03-26

## 변경 사항
CP(Combat Power)를 밸런싱의 1차 기준에서 2차 참고 지표로 격하.
전투 시뮬 승률을 1차 기준으로 채택.

## 변경 이유

### CP 공식의 한계
```
CP = (ATK / AS) × HP = DPS × HP
```
- Range(사거리)와 MS(이동속도)를 반영하지 않음
- 원거리 유닛의 "교전 전 무료 공격 시간"이 CP에 잡히지 않음
- Range가 높은 빌드(확산 스팀펑크 등)를 체계적으로 과소평가

### 실제 사례
- 스팀펑크 시뮬에서 확산 CP 136k vs 집중 CP 227k → CP만 보면 "확산 약함"
- 하지만 전쟁기계의 사거리 스케일링(#화기 8기당 +1)은 CP에 미반영
- SC1에서 사거리가 긴 유닛이 CP 이상의 전투 가치를 가진다는 건 검증된 패턴

### 기존 밸런싱의 문제
- CP 격차를 기준으로 드루이드 등의 수치를 조정하는 경우가 많았음
- Range가 다른 빌드 간 CP 비교는 오도할 수 있음

## 대안 비교
| 방법 | 장점 | 단점 |
|------|------|------|
| CP 수정 (Range 가중치 추가) | 하나의 공식으로 통일 | 어떤 가중치도 근사일 뿐, 복잡해짐 |
| ECP 도입 (내부용 별도 공식) | 표시용/분석용 분리 | 두 공식 유지 부담 |
| **승률 기준 전환** | 가장 정확, CP 한계 무관 | 시뮬 의존도 높아짐 |

## 결정
**승률 기준 전환** 채택. CP는 플레이어 UI + 원인 분석 참고 지표로 유지.
