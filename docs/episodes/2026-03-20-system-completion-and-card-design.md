# 2026-03-20: 시스템 완성 + 카드/유닛 설계 진입

## Topic 1: 테마 독립 시스템 완성

### Process
- 목표: 기획 완료까지 남은 스텝 식별 → 테마 독립 작업 우선 진행
- 보조 태그 시너지: 별도 시스템이 아닌 카드 효과의 한 유형으로 결론
- 업그레이드 구조: 5축 효과, 범용/태그한정 트레이드오프, 23~28종
- 적 파워 곡선: 프리셋형(라운드당 3~4개) + 랜덤 업그레이드 + 특성 보스
- 보스 보상: ⚡즉시/🔄영구/💥직접/📐구조 유형별 원칙 확정

### Negative Knowledge
- 보조 태그에 TFT식 브레이크포인트 시스템을 제안했으나, 유저는 단순히 카드 효과의 패시브 태그 참조로 충분하다고 판단
- 성장(개량/성장)을 카드의 주력 효과로 설계했으나, 너무 잦은 성장이 후반 카드 가치를 소멸시키는 문제 → 업그레이드가 주 축으로 전환

## Topic 2: 키워드 아키텍처 & 테마 확정

### Process
- 🌱배양은 임시 이벤트 이름. 스팀펑크에서는 "제조"로 대체
- Layer 1 (결과 범주, 테마 무관) + Layer 2 (키워드, 테마별) 구조 확정
- 멀티 테마 확정: 단일 스팀펑크가 아닌 4테마+중립
- 테마 선택: 스팀펑크(확정) + 드루이드/충군/군대 (연금술 탈락)
- 카드 풀 31→50장 (TFT/BG 비교 분석 후 결정)
- 중립 14장으로 굿스터프 덱 아키타입 지원

### Negative Knowledge
- 연금술 테마: 군집 적합도가 가장 낮아 탈락. "호문쿨루스 대량 생산"이 직관적이지 않음.
- 드루이드를 "물량 폭발" 테마로 잘못 설계 → 유저 수정: 드루이드=소수 정예, 충군=물량 폭발

## Topic 3: 발동 모델 & 성장 체계 전면 개편

### Process
- 기존 "발동 N회" 이중 해석 (라운드 시작 = N회 연속) → 유저 거부
- 수정: 모든 카드 1이벤트=1발동. "발동 N회" = 라운드당 반응 상한
- 라운드 시작 카드: 1회 발동, ★ 진화 = 효과 품질 향상 (횟수가 아님)
- 성장(개량/성장) 빈도: 테마 9장 중 2~3장만. 기본 5%, 조건부 8%.
- 성장의 주 축: 업그레이드 (카드에 부착, 영구)
- 효과 지속 유형 3종: 1회성(카드효과) / 영구규칙(업그레이드,보스) / 일시적(전투버프)
- 강화 수치 체계: 기본%(원래스탯기준) / 곱연산(명시) / 고정값(드문)

### Negative Knowledge
- 성장 15~25%로 설계 시 5라운드만에 100%+ 성장 → R1 T1 유닛이 R12 T5 유닛 압도
- 성장 상한(cap) 도입을 제안했으나 유저는 % 하향 + 빈도 감소만으로 충분하다고 판단
- 자기 효과로 자기가 반응하는 설계: 가능하지만 자기 루프 유발 → 트리거에서 자신 제외 설계

## Topic 4: 설계 공간 (Design Space) 구축

### Process
- 유저: 모든 카드가 이벤트 반응일 필요 없음. 다양한 트리거/효과 필요.
- 트리거 풀 구축: 이벤트 반응 / 시점 / 조건부 / 시간누적 / 일회성 임계점
- 효과 풀 구축: 기본 + 발라트로식 (대가형, 보드 활용, 복사/참조, ★합성 변형, 경제 연동)
- 유저 추가: 벤치 카드 수 조건, 골드 정확히 0, 같은 카드 2장, 상점 레벨업 트리거 등
- 핵심 원칙: 이 풀은 카드/업그레이드/부적/커맨더의 공유 재료
- 시스템 간 효과 강도 정의: 커맨더 > 부적 > 보스 보상 > 카드 > 업그레이드

### Negative Knowledge
- 카드 효과를 전부 체인 이벤트 반응으로 설계하면 다양성 부족
- 유저: "효과풀을 먼저 만드는 것이 카드 설계보다 우선"

## Topic 5: 유닛 풀 프레임워크 & 초기 설계

### Process
- 유닛은 테마 소속 없음 (방안 2). 어떤 카드에든 포함 가능하지만 테마별 특색 있음.
- 테마별 10종, 중립 10종, 총 50종
- 테마별 스탯 차별화: 충군(약/많) vs 드루이드(강/적) vs 스팀펑크/군대(균형)
- 범용 태그: #기계/#생체, #소형/#중형/#대형, #근접/#원거리
- 유닛 이름 = 시각화 가능한 개체 (부품 이름 ❌ → 로봇/생물 이름 ✅)
- 스팀펑크 18→10종 정리. 드루이드 10종 신규 설계.
- 전체 ATK 스케일 상향: ATK:HP 비율 ~1:8~10으로 전투 60초 내 완료 보장

### Negative Knowledge
- 유닛 이름이 "구리 기어", "고무 스프링" 등 부품 이름 → 스프라이트로 제작 불가. 유닛은 이동/공격 가능한 개체여야 함.
- ATK 1~4 수준(스팀펑크 초기)은 전투 교착 유발. 특히 양쪽이 같은 유닛 풀 사용 시.
