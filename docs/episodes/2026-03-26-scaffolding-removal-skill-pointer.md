# Scaffolding 제거: 카드 설계 스킬 포인터

## 제거 대상
CLAUDE.md의 `### 카드 설계` 섹션 (2줄):
```
카드 설계 규칙은 `.claude/skills/card-designer/SKILL.md`에 분리. 카드 관련 작업 시 자동 참조.
```

## 인코딩된 가정
"에이전트에게 스킬 파일 위치를 명시적으로 알려줘야 카드 설계 시 해당 스킬을 로드한다."

## 제거 근거
Claude Code 스킬 시스템은 `when-to-use` 필드 기반으로 자동 매칭한다.
card-designer 스킬의 when-to-use는 충분히 명확:
> "카드를 새로 만들거나, 기존 카드를 수정하거나, ★ 진화를 설계하거나, 카드 밸런스를 조정할 때"

명시적 포인터는 시스템이 제공하는 기능의 중복이며, CLAUDE.md 토큰을 소비한다.

## 검증 방법
다음 세션에서 카드 설계 작업 수행 시 card-designer 스킬이 자동 로드되는지 확인.
자동 로드 실패 시 포인터 복원.
