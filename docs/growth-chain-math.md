# 성장 체인 수학 검증

## 1. 전제 & 미결정 해석

시뮬레이션 전에 확정해야 할 설계 판단 3가지.

### 1.1 "발동 N회"의 의미

> **⚠ 이중 해석 폐기 (2026-03-20 업데이트).**
> 모든 카드는 트리거 조건 1회당 1회 발동. "발동 N회" = 라운드당 최대 N회 반응 상한.
> 라운드 시작 카드는 1회 발동. ★ 진화 = 효과 자체가 강해짐 (횟수 증가 아님).
> 추가 발동: ⚡충전(상한 +1) 또는 "추가 발동" 효과로만 가능.
> 최신 모델은 DESIGN.md의 "발동 모델" 섹션 참조.

### 1.2 신규 유닛 스탯 상속

| 해석 | 내용 |
|------|------|
| **현재 스탯 상속 (채택)** | 🌱배양으로 생긴 유닛은 카드의 현재(강화된) 스탯을 가짐 |
| 기본 스탯 고정 | 신규 유닛은 항상 기본 스탯으로 생성 |

**현재 스탯 상속 채택 근거:**
- 플레이어 멘탈 모델 단순화 (카드 스탯 = 유닛 스탯)
- 배양×강화의 곱셈적 성장이 "사기" 순간을 만듦
- 기본 스탯 고정 시 유닛별 개별 트래킹 필요 → 복잡도 증가

### 1.4 스탯 스케일 통일 (2층 아키텍처 기준)

| 항목 | 값 | 비고 |
|------|-----|------|
| ATK 범위 | 1~20 | 강화로 성장. +1당 체감 큼 (base 2 → 3 = +50%) |
| HP 범위 | 20~500 | 강화로 성장. +10당 안정적 (base 30 → 40 = +33%) |
| HP_SCALE | 불필요 | HP가 직접 전투에 사용됨 |
| 📈강화 기본 | (+1 ATK / +10 HP) | HP 절대 성장률이 높아 초중반 생존 기본값 |

**HP > ATK 성장 설계 의도:**
- 초중반: HP 성장이 전투 생존에 직접적 → HP 강화가 유리
- 후반: 적 HP가 급등 → ATK 강화(대장장이 등)로 돌파력 확보 필수
- 결과: HP는 "기본 생존", ATK는 "사기 돌파"로 역할 분리

### 1.3 이벤트 처리 순서

**BFS(큐) 방식 채택**: 이벤트 발생 → 왼→오 스캔으로 반응 수집 → 발동 → 생성된 새 이벤트를 큐에 추가 → 다음 이벤트 처리.

## 2. 성장 공식

### 단일 카드 파워

매 라운드 일정한 성장량(dᵤ: 유닛 증가, dₐ: ATK 증가)을 받는 카드의 파워:

```
카드 총 공격력(R) = (U₀ + R·dᵤ) × (A₀ + R·dₐ)
                  = U₀·A₀ + R(U₀·dₐ + A₀·dᵤ) + R²(dᵤ·dₐ)
                                                    ↑ 핵심: 2차항
```

**핵심 성질**: 배양(dᵤ)과 강화(dₐ)가 동시에 존재하면 **2차(quadratic) 성장**. 하나만 있으면 선형 성장.

```
dᵤ·dₐ = 0 → 선형 (배양만 or 강화만)
dᵤ·dₐ > 0 → 2차 (배양+강화 동시 → "복리 성장")
```

### 군대 총 파워

```
총 공격력 = Σ (카드별 유닛수 × 카드별 ATK)
총 체력   = Σ (카드별 유닛수 × 카드별 HP)
```

```

🌱배양(돌격) ×2:
  → 배양소 반응(2/2회): 자신 🌱배양 ×2        → 🌱배양(배양소) ×2
  → 훈련교관 반응(2/3회): 돌격부대 📈강화 ×2   → 돌격 ATK +2

  → 훈련교관 반응(1/3회): 배양소 📈강화 ×1     → 배양소 ATK +1

총: 7 활성화. 돌격 +2u/+2ATK. 배양소 +2u/+1ATK.
```

- 돌격부대: +2u, +2ATK
- 배양소: +2u, +1ATK
- 양성사, 훈련교관: 변화 없음

## 5. 적 파워 곡선 제안

### 설계 원칙

```
"업그레이드 없는 기본 플레이 = R4부터 고전, R6에서 패배"
"★2 빌드 = R6 돌파, R9에서 한계"
"★3 빌드 = R10+ '사기'로 돌파"
```

적 파워 = 약한보드와 최적화보드의 **중간** 정도를 기준으로 설정.

## 6. 경제 제약 검증

### ★2 달성 시점

```
★2 = 같은 카드 3장 합성. 카드 비용 3골드.

수입:
  R1~R5 총 수입 = 5라운드 × 6골드(평균) = 30골드
  시작 골드 = ?? (미정, 10 가정)
  총 가용: ~40골드

지출:
  코어 카드 4장: 12골드
  ★2용 추가 2장 (같은 카드): 6골드
  리롤 (같은 카드 찾기): 5~10골드
  총 최소: 23~28골드

→ R5까지 ★2 1장 달성 가능 ✓
→ 남은 12~17골드로 대장장이 등 추가 카드 구매 가능
```

### ★3 달성 시점

```
★3 = ★2 카드 3장 합성 = ★1 카드 9장 필요.
비용: 9 × 3 = 27골드 (카드 구매만)

R1~R9 총 수입 = 9 × 6 + 10(시작) = 64골드
★3 1장에 27골드 + 리롤 비용 → 총 ~35골드

→ R7~R8에서 ★3 1장 달성 가능 ✓
→ 남은 ~29골드로 나머지 보드 구축

⚠ 병목: 같은 카드 9장을 R7까지 상점에서 찾아야 함.
  → 상점 카드 풀 크기, 등장 확률이 미정.
  → 이 수치가 정해져야 ★3 달성 시점 정밀 계산 가능.
```

### 미정 경제 파라미터

| 파라미터 | 필요한 이유 | 영향 |
|----------|------------|------|
| 시작 골드 | R1 구매력 결정 | 초반 보드 완성도 |
| 상점 카드 수 | 리롤 없이 원하는 카드 볼 확률 | ★2/★3 달성 시점 |
| 카드 풀 크기 | 특정 카드 등장 확률 | ★2/★3 달성 난이도 |
| 라운드별 레어리티 분포 | 강한 카드 접근 시점 | 빌드 커브 |
| 카드 레어리티별 가격 차등 | 강한 카드의 경제 부담 | 빌드 선택 압박 |

## 7. 핵심 발견

### ✅ 잘 작동하는 것

1. **배양×강화 = 2차 성장**: 핵심 설계 의도인 "복리 성장"이 수학적으로 성립. R² 계수(dᵤ·dₐ)가 성장률을 결정.

2. **★ 업그레이드 = 체감되는 파워 점프**: R² 계수가 ★1(6) → ★2(15) → ★3(100)으로 급등. "사기"가 되는 순간이 수치적으로 존재.

3. **약한빌드 vs 최적빌드 차이 3~5배**: 시너지 구축이 보상받음. R3에서 3.3배 (37 vs 123).

4. **적 파워 곡선 실현 가능**: 2차 성장을 기반으로 적도 2차 스케일링하면 의도한 난이도 곡선 구현 가능.

### ⚠️ 밸런스 위험

**1. 체력 성장 부재**

물량 배양 빌드(3B)는 체력이 선형으로만 성장(12R + 29). 공격력(6R² + 18R + 15)과 체력의 격차가 라운드가 갈수록 벌어짐.

```
R5: 공격력 255, 체력 89 → 비율 2.9:1
R3: 공격력 123, 체력 65 → 비율 1.9:1
```

→ 적의 높은 ATK에 취약. 결집의깃발(+0/+1) 같은 HP 강화 카드가 보드에 빠지면 "유리대포" 빌드.
→ **설계 질문**: 이것을 의도된 트레이드오프로 둘 것인가, HP 성장 카드를 더 추가할 것인가?

**2. 유닛 수 폭발 (★3)**

★3 양성사가 매 라운드 돌격부대에 +10u. R12에 돌격부대 단일 카드에 80유닛.

```
R7:  30u
R9:  50u
R12: 80u
```

→ 문서의 "대규모 군집 20v20+"와 비교하면 한쪽 카드만 80u는 과도할 수 있음.
→ **완화 옵션**: (a) 카드별 유닛 상한 (예: 30u), (b) 배양 효율 체감 (10u 이후 2기→1기), (c) 유닛 수 자체를 더 적은 숫자로 리스케일.

**3. 매 라운드 동일한 체인 → 반복감**

보드가 변하지 않으면 매 라운드 동일한 체인이 동일한 순서로 발동. 성장량은 누적되지만, **체인 자체의 변화가 없다**.

→ 재미는 "체인 구축"(상점/배치 페이즈)에 있지, "체인 감상"(성장 페이즈)에는 제한적.
→ **완화 옵션**: (a) 라운드 이벤트(날씨, 돌발 등)로 변수 추가, (b) 적 구성 미리보기 → 배치 순서 조정 유도, (c) 체인 연출 속도를 빠르게 (발라트로식 빠른 점수 계산).

**4. 성장 격차의 눈덩이 효과**

R1에 좋은 시너지를 잡은 플레이어는 R3에 3배 강해짐. 초반 운이 런 전체를 결정할 위험.

→ 로그라이크 특성상 어느 정도의 런 분산은 의도적이지만, "R1~R2에 시너지 못 잡으면 무조건 실패"는 피해야 함.
→ **완화 옵션**: (a) R1 상점에서 기본 시너지 카드 보장, (b) 패배 후 성장 트리거(전쟁고아 등) 강화, (c) 유물로 뒤늦은 빌드 전환 지원.
