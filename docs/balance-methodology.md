## 발견된 밸런스 이슈

### 1. HP 성장 기본값 적용

기존 문제: 훈련교관 📈강화(+1/+0) → HP 성장 없음 → R7+ 즉사.

수정: 📈강화 기본값을 (+1/+10)으로 변경.
- 훈련교관: +1 ATK, +10 HP (기본 강화)
- 결집의깃발: +0 ATK, +15 HP (HP 특화)
- 대장장이: +2 ATK, +0 HP (ATK 특화)

설계 의도:
- HP 성장 = 기본값 (아무 체인이나 돌려도 쌓임)
- ATK 성장 = 투자 (대장장이 등 특화 카드 필요)
- 초중반: HP 우세 → 생존 기반 확보
- 후반: ATK 투자가 돌파의 열쇠

### 2. ★3 스파이크 = 의도된 "사기" 순간

R9~R10: ★3 달성 시 "사기" 돌입
R11~R13: 2~3라운드 지배 (핵심 쾌감)
R14~R15: 적이 따라잡음 → 최종 긴장

이 곡선은 의도된 감정 아크:
  R8~R9: "이걸 어떻게 이겨?" (절망)
  R10~R11: ★3 완성 → "사기" (쾌감의 정점)
  R14~R15: "마지막 벽" (긴장)

### 3. 빌드 간 비대칭 밸런스

```
정예강화 클리어율 28% > 물량배양 22.5%

원인:
- 정예기사의 높은 base HP (5) → 전투 생존성
- 적은 유닛 수 → 같은 카드 찾기 쉬움 → ★ 업그레이드 빠름
- 대장장이의 추가 ATK가 개당 임팩트 큼 (유닛 적으므로)

판단: 이 차이가 의도된 것인가?
→ 빌드별 특성 차이(물량 vs 정예)는 좋은 설계.
→ 하지만 물량배양이 "물량으로 압도"하는 판타지를 못 살림.
→ HP 성장 추가로 물량빌드 R12 승률 보완 필요.
```

## 반복 테스트 방법론

### 1단계: 파라미터 변경

`sim/simulate.py` 상단 CONFIG 섹션에서 조정:
- `ENEMIES` 테이블: 적 스케일링
- `HP_SCALE`: 전투 길이
- `UNIT_CAP`: 물량 상한
- `STARTING_GOLD`: 초반 경제
- `CARD_COST`, `REROLL_COST`: 경제 압박

### 2단계: 시뮬레이션 실행

```bash
# 빠른 체크 (100회)
python3 sim/simulate.py -n 100 --seed 42

# 정밀 테스트 (500회)
python3 sim/simulate.py -n 500

# 두 전략 비교
python3 sim/simulate.py -s 물량배양 -n 200 --seed 42
python3 sim/simulate.py -s 정예강화 -n 200 --seed 42

# 특정 시드 분석 (문제 런 재현)
python3 sim/simulate.py -n 1 -v --seed 7
```

### 3단계: Balance Alerts 확인

시뮬레이터가 자동 진단:
- 초반(R1-3) 승률 < 90% → 적 너무 강거나 시작 골드 부족
- 보스(R5,10,15) 승률 40-70% 범위 밖 → 적 스탯 조정
- 후반(R12+) 승률 > 70% → ★3 스파이크 과도
- 클리어율 20-40% 범위 밖 → 전체 난이도 곡선 재조정

### 4단계: 문제 진단 → 조정 → 반복

```
Iteration 1: 기본값 → R1 승률 62% (너무 낮음), R12 승률 90% (너무 높음)
  조정: 시작골드 10→13, R1적 약화, R8+적 강화

Iteration 2: → R3 64%✓, R12 87% (아직 높음)
  조정: R10-R12 적 HP 대폭 증가

Iteration 3: → 물량 클리어 22.5%✓, 정예 28%✓
  남은 이슈: 물량빌드 HP 부재, R10-11 ★3 스파이크
```

### 핵심 밸런스 레버 요약

| 레버 | 올리면 | 내리면 |
|------|--------|--------|
| `STARTING_GOLD` | 초반 안정 ↑, 빌드 다양성 ↑ | 초반 긴장감 ↑, 운 의존도 ↑ |
| 적 HP/유닛 | 전투 시간 ↑, 물량 불리 | 전투 시간 ↓, 물량 유리 |
| 적 유닛 수 | 물량 유리 (타겟 분산) | 정예 유리 (집중화력) |
| `UNIT_CAP` | 물량빌드 천장 ↑ | ★3 폭발 제한 |
| `HP_SCALE` | 전투 길이 ↑, 스탯차 압축 | 즉사 잦아짐, 선공 유리 |
| 카드 발동상한 | 체인 길이 ↑, ★ 가치 ↑ | 성장 속도 ↓ |
