# 업그레이드 아이템 풀 (26종)

> 참조: upgrade.md (시스템 구조), card_db.gd (카드 데이터)
> 모든 업그레이드는 **범용** (태그 한정 없음). 어떤 카드에든 부착 가능.
>
> **설계 원칙**:
> - 강화(체인) = "기존 능력이 더 세짐" (ATK%, HP%, 유닛+)
> - 업그레이드 = "없던 능력이 생김" (DEF, 관통, 리젠, 가시 등)
> - 커먼 기본 강화(C1,C2)만 강화와 일부 겹침 (안전한 선택지 역할)
> - 업그레이드는 Layer 2 (upgrade_mult) 적용. 강화(Layer 1 growth%)와 곱연산.
>
> **파워 스케일**:
> - 커먼 1개 = 전투력 +5~8%
> - 레어 1개 = 전투력 +12~18% (또는 질적 전환)
> - 에픽 1개 = 전투력 +25~40% (또는 게임 규칙 변경)
> - 에픽은 높은 난이도(7~8) 돌파의 핵심 수단.
>
> **AS(공격속도) 표기**: 높을수록 빠름. 내부는 attack_interval(낮을수록 빠름) 유지.
> "공격속도 +15%" = attack_interval × 0.85

## 5축 정의

```
축 1. 기본 강화     ATK%, HP% (소수. 강화와 겹치지만 안전한 선택지)
축 2. 새 스탯 부여   Range+, MS+, 공격속도+ (강화에 없는 축)
축 3. 공격 특수      관통, 스플래시, 연쇄, 크리티컬
축 4. 방어/생존      DEF, 가시, 방어막, 라이프스틸, 리젠, 회피
축 5. 조건부 트리거   광폭화, 처형, 사망 시 폭발/소환
```

## 커먼 (11종, 4 테라진)

"숫자가 올라가거나, 간단한 새 속성이 추가된다."

```
[C1] 강화 합금
축: ① 기본
효과: ATK +15%
  안전한 범용 강화. 어떤 카드에든 무난.

[C2] 생체 보강제
축: ① 기본
효과: HP +15%
  생존력 강화. 저HP 유닛(20~40)에 효과적.

[C3] 경량 장갑
축: ④ 방어
효과: DEF +1
  새 스탯 부여. 약한 공격(ATK 3~4)을 크게 감소.
  DEF 계산: 피격 데미지 = max(1, ATK - DEF)

[C4] 사거리 확장기
축: ② 새스탯
효과: Range +1
  근접(Range 1) → 2칸 원거리. 원거리 유닛은 더 안전한 위치에서 공격.

[C5] 추진 부스터
축: ② 새스탯
효과: MS +15
  빠른 접근. 원거리 적에게 먼저 도달.
  기본 MS 범위: 40~90. +15는 약 20~35% 상승.

[C6] 가속 장치
축: ② 새스탯
효과: 공격속도 +15% (attack_interval × 0.85)
  DPS 직접 상승. 모든 유닛에 유효.

[C7] 가시 외피
축: ④ 방어
효과: 피격 시 공격자에게 자신 ATK의 20% 반사 데미지
  물량 적(다수 약한 공격) 상대에 효과적.
  고ATK 유닛에 부착 시 반사량 증가.

[C8] 응급 수리
축: ④ 생존
효과: 전투 시작 시 HP 10% 회복
  연전 보정. 이전 전투 데미지 일부 복구.
  (현재 시스템에서 유닛 HP는 전투 간 리셋되므로,
   실질 효과는 "전투 시작 시 HP 110%로 시작"과 동일)

[C9] 관통탄
축: ③ 공격
효과: DEF 50% 무시
  적이 DEF를 가질 때(업그레이드/보스 특성) 유효.
  DEF 0인 적에게는 효과 없음.

[C10] 집중 사격
축: ③ 공격
효과: 동일 대상 연속 공격 시 ATK +10% 누적 (대상 변경 시 리셋)
  보스/탱크 킬러. 느리지만 강한 유닛(AS 1.5+)에 최적.
  5연속 공격 시: +50% ATK.

[C11] 보호막 발생기
축: ④ 방어
효과: 전투 시작 시 최대 HP의 15% 방어막 (초과 HP)
  첫 교전 생존. 방어막은 전투 중 회복 불가.
```

## 레어 (9종, 8 테라진)

"새 행동이 추가된다. 카드에 맞는 레어를 찾는 재미."

```
[R1] 흡혈 송곳니
축: ④ 생존
효과: 공격 데미지의 15% HP 회복 (라이프스틸)
  고ATK 유닛에 극강. ATK 15 유닛이면 매 공격 +2.25 HP.
  소수 정예(드루이드 빌드)에 최적.

[R2] 산탄 개조
축: ③ 공격
효과: 공격 시 주변 1칸 내 적에게 50% 스플래시 데미지
  물량전 특화. 밀집한 적을 동시에 타격.
  원거리 유닛에 부착 시 안전하게 범위 공격.

[R3] 위상 변환기
축: ④ 방어
효과: 첫 피격 완전 무효 (전투당 1회)
  핵심 유닛(에이스) 보호. 고ATK 적의 선제 공격 방어.
  소수 유닛 카드에서 가치 극대화.

[R4] 전술 후퇴
축: ④ 생존
효과: HP 25% 이하 시 MS ×2 + 5초 피격 무효, 이후 복귀
  "죽지 않는" 유닛. 전선에서 이탈→재진입.
  효과적으로 유닛 생존 시간 연장.

[R5] 연쇄 방전
축: ③ 공격
효과: 적 처치 시 주변 적에게 ATK 30% 데미지 (연쇄 1회)
  물량 적 소탕. 처치 연쇄로 도미노 킬.
  고ATK 유닛에 부착 시 연쇄 데미지도 증가.

[R6] 재생 프로토콜
축: ④ 생존
효과: 3초마다 최대 HP의 5% 회복
  장기전 유리. 30초 전투 시 총 HP 50% 회복.
  탱커(고HP) 유닛에 최적.

[R7] 중력 앵커
축: ② 새스탯
효과: 사거리 내 적 MS -30% (감속 오라)
  원거리 유닛의 카이팅 보조. 적이 접근하기 어려워짐.
  여러 유닛이 같은 적을 감속하면 중첩 (최대 -60%).

[R8] 정밀 조준
축: ③ 공격
효과: 크리티컬 15% 확률, 데미지 ×2.5
  기대값: ATK ×1.225 (+22.5%). 하지만 체감은 "가끔 폭발".
  소수 정예(고ATK, 느린 AS)에 최적 — 한 방이 크니까.

[R9] 중장갑
축: ④ 방어
효과: DEF +2, HP +20%
  복합 방어 강화. 단순하지만 강력한 탱킹.
  DEF +2는 커먼 C3의 2배. HP +20%는 후반에도 스케일링.
```

## 에픽 (6종, 보스 보상 / ★3 합성 전용)

"전투 양상이 바뀐다. 높은 난이도 돌파의 핵심 수단."

```
[E1] 광폭화
축: ⑤ 조건부
효과: HP 30% 이하 시 ATK ×2, 공격속도 ×2. 회복 불가.
  "죽기 직전에 가장 강하다." DPS 4배 폭발.
  R7+ 전술 후퇴와 조합하면 → HP 25%에서 후퇴+무적 → 복귀 후 광폭 유지.
  난이도 7~8에서 보스 녹이는 핵심.

[E2] 연쇄 폭발
축: ③ 공격
효과: 스플래시 70% + 처치 시 ATK 50% AOE 폭발 (반경 2칸)
  R2 산탄(50%)의 상위 호환 + 처치 폭발.
  물량전에서 학살. 밀집한 적 100기를 연쇄 폭발로 전멸.
  스팀펑크(물량) + 포식종(물량전) 상대로 결정적.

[E3] 불멸의 핵
축: ④ 생존
효과: 치명타 시 HP 1로 생존 + 3초 무적 (전투당 1회). 무적 해제 후 HP 50% 회복.
  에이스 유닛의 생존 보장. 어떤 공격이든 1번은 살아남음.
  드루이드(소수 정예, 극 고스탯)에 최적 — 세계수 버프 유닛이 절대 안 죽음.

[E4] 영혼 수확
축: ⑤ 조건부
효과: 적 처치 시 ATK +5% 영구 누적 (이번 전투). 10킬 달성 시 공격속도 ×1.5 추가.
  킬할수록 강해지는 눈덩이 효과.
  20킬 시: ATK +100%, AS ×1.5. 물량전 후반에 무쌍.
  군대(정예 소수) 빌드에 부착 시 정밀 처치 → 극강 스케일링.

[E5] 분열 증식
축: ⑤ 조건부
효과: 유닛 사망 시 같은 유닛 2기 소환 (HP 50%, ATK 50%). 소환 유닛은 분열 불가.
  "죽을수록 늘어난다." 10기 사망 → 20기 추가 = 총 30기.
  포식종(물량 빌드)에 최적. 부화로 키운 유닛이 사망해도 수가 줄지 않음.
  난이도 8에서 "이길 수 없는 전투"를 물량으로 뒤집는 카드.

[E6] 차원 붕괴
축: ③ 공격
효과: 공격 시 대상 현재 HP의 8% 추가 데미지 (고정, DEF 무시)
  고HP 적 녹이기. 보스(HP 2000+)에 극강.
  HP 1000 적 첫 공격: +80 데미지. 일반 ATK(10~20) 대비 4~8배.
  느리지만 강한 유닛(원거리, AS 1.5)에 최적.
```

## 런당 업그레이드 경제 예상

```
테라진 수입: ~24t (9승 6패)
★2 합성 보너스: 무료 레어 ×1~2장
★3 합성 보너스: 무료 에픽 ×0~1장
보스 보상: 에픽 ×1~2장 (R4/R8/R12 중 에픽 선택 시)

일반 런 (난이도 1~4):
  커먼 3~4장 (12~16t) + 레어 1장 (8t) + ★2 무료 레어 1장 = 5~6 업그레이드
  → 카드 1~2장에 집중 부착

고난도 런 (난이도 5~8):
  커먼 2장 (8t) + 레어 1장 (8t) + ★2 레어 2장 + 보스 에픽 1~2장 = 6~7 업그레이드
  → 에이스 카드에 에픽+레어 집중 = 그 카드 하나가 전쟁을 바꿈
```
