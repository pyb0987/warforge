# 업그레이드 시스템 & 경제

## 업그레이드 개요

카드(부대)에 부착하는 강화 효과. 한번 붙이면 런이 끝날 때까지 지속. 제거 불가 (또는 제값 회수 불가).
★합성이 카드의 트리거/효과를 **질적 변환**한다면, 업그레이드는 **새로운 속성을 추가**한다.

## 슬롯 체계

```
카드 1장당 최대 5슬롯. 모든 등급 1슬롯.

→ 차이는 슬롯 수가 아니라 효과 파워.
→ 5슬롯 = 5개 업그레이드. "어떤 효과 조합이 내 빌드에 맞는가"가 핵심.
```

## 효과 구조

### 적용 범위: 범용 vs 태그 한정

```
범용 업그레이드:    "ATK +2"              → 카드 내 모든 유닛에 적용
태그 업그레이드:    "#기계 한정: ATK +3"    → 카드 내 해당 태그 유닛에만 적용

트레이드오프:
  범용 = 안전하지만 약함 (어떤 카드에든 무난)
  태그 한정 = 강하지만 카드 구성에 의존 (유닛 태그 비율이 중요)

참조 가능한 범용 태그 (테마 무관, 모든 유닛에 존재):
  구성:  #기계 / #생체
  크기:  #대형 / #소형
  사거리: #근접 / #원거리
  → 어떤 테마든 이 범용 태그를 가지므로 업그레이드가 테마 독립
```

### 효과 축 (5종)

```
축 1. 직접 스탯       ATK+, HP+, DEF+
축 2. 전투 메카닉     AS 개선, Range+, MS+
축 3. 공격 특수효과   스플래시, 관통, 라이프스틸
축 4. 방어 특수효과   첫 피격 무효, 가시, 전투 시작 방어막
축 5. 조건부 효과     "HP 50% 이하 시 ATK 2배", "사망 시 AOE"
```

### 등급별 설계 원칙

```
커먼 (10~12종):  "숫자가 올라간다"
  - 단순 스탯 증가. 어떤 카드에 붙여도 나쁘지 않음.
  - 범용 + 일부 태그 한정 (태그 한정이 약간 더 강함)
  - 예: ATK +2 / HP +15 / DEF +1 / #기계 한정: ATK +3 / #대형 한정: HP +30

레어 (8~10종):  "새 행동이 추가된다"
  - 전투 메카닉 변경 또는 조건부 효과. 카드 선택이 중요해짐.
  - 태그 한정 비율 높음 (카드에 맞는 레어를 찾는 재미)
  - 예: ATK +30% / 라이프스틸 15% / 관통 (DEF 무시) / #대형 한정: DEF +3, HP +30

에픽 (5~6종):  "전투 양상이 바뀐다"
  - 강력한 특수효과. 빌드를 정의하는 수준.
  - 보스 보상 + ★3 합성으로만 획득.
  - 예: 스플래시 50% + 범위 확대 / #소형 한정: 사망 시 1기 유닛 추가

총 23~28종. 런당 적절한 반복 노출 + 다양성 확보.
```

## ★합성 보너스 업그레이드

```
★2 합성 시: 레어 업그레이드 3택1 즉시 부착 (무료)
★3 합성 시: 보너스 없음 (★2 단계에서 얻은 rare ×3이 자동 누적으로 충분)

캐스케이드 머지 (★1×9 → ★2×3 → ★3):
  - 중간 ★2 단계마다 레어 업그레이드 보상 정상 지급
  - ★2→★3 단계에서는 보너스 없음 (위 규칙과 동일)
  - 예: ★1×3→★2 (레어 1장) → ★2×3→★3 (보너스 없음)

→ ★합성의 보상: 스탯 증가 + 질적 변환 + ★2 무료 업그레이드
→ "★2를 빨리 만들수록 무료 업글을 일찍 받는다" = 합성 동기 강화
→ ★3 ceremony: ATK ×2.9 스탯 스파이크 + 누적된 rare ×3 시각화로 충분
→ 에픽 수급처: 보스 보상 전용 (희소성 유지)

★합성 시 업그레이드 누적:
  - donor 2장의 업그레이드를 모두 survivor 에 흡수
  - 흡수 순서: 배치 순서상 앞쪽(보드 왼→오 → 벤치 왼→오) donor부터 흡수
  - 5슬롯 상한 초과 시 뒤쪽 donor의 업그레이드부터 버림 (truncate)
  - survivor 선정: "업그레이드 수 최대 → 동점 시 leftmost(보드→벤치)"

★합성 시 도너 stat 흡수 정책 (2026-04-26):
  - 합산: 유닛(cap 60), 업그레이드 어레이(5슬롯), growth_atk_pct/hp_pct,
          tag_growth_*, upgrade_def/range/move_speed, shield_hp_pct,
          theme_state 그룹A (trees, manufacture_counter, attack_stack_pct,
          range_bonus)
  - 곱셈: stacks[].upgrade_atk_mult/hp_mult, upgrade_as_mult
          (% 업그레이드, 보스 보상, 커맨더 RAIDER 입력원 통합)
  - max:  tenure, theme_state["rank"], unit_cap_bonus, upgrade_slot_bonus,
          stacks[].unique_atk_mult/hp_mult, unique_as_mult ([고유효과] layer)
  - OR:   is_omni_theme, theme_state 그룹B (pending_epic_upgrade,
          high_rank_applied)
  - 0/false 리셋: activations_used, threshold_fired (evolve_star)
  - 기타 theme_state (그룹C/D, 라운드/전투 한정): survivor 유지

[고유효과] 분리 (2026-04-26):
  - 세계수(dr_world)의 multiply_stats는 매 RS 곱셈 누적이라 ★ 합성 시
    도너 누적분이 모두 곱해져 폭발할 위험. 별도 unique_*_mult 필드에
    누적하고 합성 시 max 정책 적용 (폭발 방지).
  - effective ATK = base × (1+growth) × upgrade_mult × unique_mult × temp
  - 향후 다른 [고유효과] 카드도 같은 layer 사용
```

## ★합성과의 역할 분리

| 구분 | ★합성 | 업그레이드 |
|------|-------|-----------|
| 성격 | 트리거/효과 질적 변환 | 전투/스탯에 새 속성 추가 |
| 예시 | "유닛추가 1기→2기" "범위: 인접→전체" | "사거리 +1" "피해 25% 회복" |
| 느낌 | "카드가 진화했다" | "카드에 무기를 달았다" |
| 획득 | 같은 카드 3장 합성 | 보스(에픽) / 테라진 상점(커먼·레어) |

## 경제 시스템 (확정)

### 수입
```
기본 수입:    R1~R5: 5골드 / R6~R10: 6골드 / R11~R15: 7골드
이자:        보유 골드 5당 +1골드, 최대 +2 (잔고 10 유지 시)
전투 보상:    승리 +1골드, 패배 0골드
시작 골드:    0골드

총 예산 (기본+이자): ~108골드. 전투 보상 포함 시 ~118골드.
판매 환급 포함 실질 예산: ~122골드 (BG와 동등 수준).
```

### 지출
```
카드 가격:    T1=2 / T2=3 / T3=4 / T4=5 / T5=6 골드
판매:        ★1 = 구매가 전액 환급, ★2/★3 = 총 투자액 환급 -1골드 (합성 비용 회수 방지)
             예) T1(cost=2): ★1→2g, ★2→2×3-1=5g, ★3→2×9-1=17g
리롤:        1골드
레벨업:      아래 상점 레벨업 참조
벤치:        8칸 (필드 외 카드 보관)

판매 환급 설계 근거:
  - ★1 전액 환급: 골드 부족 + 이자 + 벤치 8칸이 "신중한 구매"를 충분히 강제,
    교체/탐색/★2 방향전환의 자유도 유지
  - ★2/★3 -1골드: 합성 후에도 환급이 완전하면 합성 비용이 0에 수렴
    → 합성 리스크 부재 → 플레이 결정의 무게감 감소 방지

무료 리롤 (글로벌 원칙):
  - "무료 리롤"은 모두 **이번 라운드 한정**. 다음 라운드 시작 시 소멸.
  - 저장 필드: game_state.pending_free_rerolls (라운드 시작 시 0 리셋)
  - 소비 우선순위: (1) 🎲 도박꾼 확률 성공(즉시) → (2) pending → (3) 골드
  - 공급원 예: 폐품 상회(중립 T2) — 인접 최약 유닛 제거 시 +1~+2
```

### 상점 시스템
```
상점 표시:    6장
카드 종류:    55장 (중립15 / 테마별10)
상점 단계:    6단계 (Lv1~Lv6)
```

### 카드 풀 고갈 메커니즘 (OBS-049)
```
복사본 수 (티어별, 카드 1종당):
  T1=22  T2=18  T3=15  T4=13  T5=11

규칙:
  - 상점에 카드가 등장하면 풀에서 1장 소모 (draw)
  - 구매하지 않은 카드는 리롤/라운드 전환 시 즉시 풀에 반환
  - 판매 시 풀에 복귀: ★1→1장, ★2→3장, ★3→9장
  - 합성에 소모된 카드는 풀에서 영구 소모 (★2 보유 = 3장 빠진 상태)
  - 특정 티어 풀 완전 고갈 시 해당 상점 슬롯은 빈 칸 표시

★3 도달 난이도 (T5 예시, 4종 × 11장 = 44장):
  - 필요 9장. 풀의 82% 소모해야 함
  - 구매할수록 해당 카드 등장 확률 감소
```

### 상점 레벨업 (자동 할인 방식)
```
베이스 비용:  Lv2: 5 / Lv3: 7 / Lv4: 8 / Lv5: 11 / Lv6: 13 (5회 레벨업)
자동 할인:   매 라운드 시작 시 현재 레벨업 비용 -1골드 (최소 0골드)
리셋:       레벨업 구매 시 다음 단계 비용이 베이스로 리셋
연속 레벨업:  한 라운드에 2단계 가능 (골드 충분 시)
합계:       베이스 44골드. 자연 타이밍 시 ~31~32골드 (예산의 26~27%).
```

### 벤치 시스템
```
크기: 8칸 (프로토타입 후 7~10 범위에서 조정 가능)
규칙:
  - 필드 ↔ 벤치 자유 이동 (빌드 페이즈 중)
  - 벤치 카드: 전투/성장 체인 미참여
  - 벤치 카드: 판매 가능, 업그레이드 부착 불가
  - 자동 합성: 같은 카드 3장(필드+벤치) 도달 시 즉시 ★ 승급
```

### ★3 합성 경제 (확정 방향)
```
위치: ★3 = 드문 보너스 (클리어 필수가 아닌 "사기" 순간)
공식: 3×3=9 유지 (난이도가 희귀성을 만듦)

★3 없는 런: ★2 + 업그레이드로 R15 클리어 가능 (빡빡하지만 가능)
★3 있는 런: R13~R15를 압도적으로 돌파 (사기의 쾌감)
```

### 테라진 상점 (업그레이드 전용)
```
테라진 수입:
  승리: +2 테라진 / 패배: +1 테라진
  보통 런(9승 6패): ~24 테라진

상점 표시:
  Lv2부터 매 라운드 업그레이드 2개 표시 (카드 6장과 별도)
  각 슬롯 독립 확률: 커먼 70% / 레어 30%

가격:
  커먼: 4 테라진 (1슬롯)
  레어: 8 테라진 (1슬롯)
  에픽: 보스 전용

리롤: 1 테라진으로 업그레이드 2장만 갱신 (카드 리롤과 완전 분리)
부착: 구매 즉시 카드에 영구 부착 (인벤토리 없음)
★합성 시: donor 업그레이드 모두 survivor 에 흡수 (5슬롯 상한)
등급 분포: 상점 레벨 무관 (항상 70/30)
```

### 2화폐 경제 구조
```
┌─ 골드 경제 ──────────────────────────┐
│  수입: 기본(5/6/7) + 이자 + 승리(+1g)│

┌─ 테라진 경제 ────────────────────────┐
│  수입: 승리(+2t) / 패배(+1t)         │

└──────────────────────────────────────┘

┌─ 보스 보상 (R4★/R8★/R12★) ──────────┐
│  골드·테라진 밖의 직접 보상            │
│  에픽 업그레이드 / ★승급 / 기타       │
└──────────────────────────────────────┘

접점: 없음 (골드와 테라진은 완전 독립)
```

### 성장 경로 전체 구조
```
━━ 런 외부 (시작 전 선택) ━━
  🎭 커맨더 선택        ← 게임 규칙 변경, 최적 전략 결정
  💎 부적 선택          ← 커맨더와 곱셈적 시너지
  🏔️ 난이도 선택        ← 도전 수준

━━ 런 내부 (4층 성장) ━━
  매 라운드 (자동):     🔧유닛추가 + 📈강화  ← 체인이 해줌, 양적 성장
  카드 수집 (투자):     ⬆★합성            ← 같은 카드 3장, 질적 변환
  테라진 투자 (선택):   🔧업그레이드        ← 별도 화폐, 카드에 영구 부착

━━ 런 간 (메타 진행) ━━
  해금:               커맨더/부적/난이도 해금 ← 업적/클리어 보상
```
