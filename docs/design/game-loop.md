# 핵심 게임 루프 & 런 구조

## 핵심 게임 루프

```
[상점 페이즈] → [빌드/배치 페이즈] → [성장 체인 발동] → [자동 전투] → [정산/보상] → 반복
      │                │                    │                │              │
      │                │                    │                │              └─ 패배 시 적 생존 유닛 수 = 데미지
      │                │                    │                └─ 리얼타임 전투 (단순 전투 체인 보조)
      │                │                    └─ ★핵심★ 유닛추가/강화 체인이 군대를 성장시킴
      │                └─ 부대 카드를 필드에 배치, 순서/위치 결정
      └─ 부대 카드 구매/판매/업그레이드
```

## 1라운드 흐름 (예상 2~3분)

1. **상점 페이즈**: 카드 6장 + 업그레이드 2장 표시. 리롤 1골드(카드)/1테라진(업글). 판매 전액 환급. 상점 레벨업 가능.
2. **빌드 페이즈**: 보유 부대를 필드에 배치 (시작 6칸, 최대 8). 배치 순서 = 체인 우선순위. 벤치 8칸.
3. **성장 체인 페이즈**: ★핵심★ 배치된 카드의 성장 트리거가 왼→오 순서로 발동 → 유닛추가/강화 연쇄.
4. **전투 페이즈**: 성장한 군대가 리얼타임 자동 전투. 전투는 절대 멈추지 않음. 단순 전투 체인 보조.
5. **정산 페이즈**: 승리 → 다음 라운드. 패배 → 데미지 = ceil(적 생존 유닛 × 라운드 배수). 배수 R1=0.2 → R15=1.2 선형.
6. **보상 페이즈**: 기본 골드 + 이자 + 전투 보상. 보스 라운드 시 4택1 보상.

## 런 구조 (15라운드, 총 28~35분)

```
플레이어 HP: 30 (패배 시만 감소, 데미지 = ceil(적 생존 유닛 × 라운드 배수); R1=0.2 → R15=1.2)

R1~R3: 약한 적. 보드 채우기. 필드 6칸.     적: 약함
R4: ★보스★ → "기반" 보상                │ 성장 체인 시작. 빌드 방향 설정.
R5~R7: 빌드 핵심 구축. ★2 시작.          │ 성장 체인 본격화.
R8: ★보스★ → "전환" 보상                ↓ ★2 전환기. 파워 스파이크.
R9~R11: 빌드 최적화. 업그레이드 집중.     적: "불합리하게" 강함
R12: ★보스★ → "완성" 보상               ↓ ★2 완성. 최종 빌드 마무리.
R13~R14: 최종 도전.                      적: 완성 빌드로 빡빡하게 돌파
R15: ★최종 보스★ (보상 없음)             적: ★2+업글 풀빌드. ★3 있으면 압도.
  → 최종 라운드 생존 = 런 클리어!
  → 플레이어 HP 0 = 런 실패
```

## 보스 보상 (3회, 확정 구조)

보스별 전용 풀에서 랜덤 4개 표시, 1개 선택. 스킵 불가.

```
구조:
  풀 크기:    보스별 9개 (전용 풀, 보스 간 분리)
  표시:      4개 (풀에서 랜덤)
  선택:      1개
  유형 비율:  ⚡즉시행동 2 / 🔄영구규칙 4 / 💥직접강화 2 / 📐구조 1
  조합 수:   C(9,4) = 126가지/보스 → 매 런 다른 경험

  R4★ "기반": 빌드 방향 설정급 보상 (⚡/🔄/💥/📐 혼합)
  R8★ "전환": 파워 스파이크급 보상 (R4보다 강력)
  R12★ "완성": "사기" 완성급 보상 (최강)
```

### 유형별 설계 원칙

```
⚡ 즉시행동 (2/9) — "1회성 행동 자체가 보상"
  정의: 선택 즉시 1회 발동. 카드에 흔적 없음. 행동이 곧 보상.
  설계 축: 성장 가속 (★승급) / 경제 주입 (골드·테라진) / 일괄 강화
  원칙:
    - 빌드 무관하게 누구에게나 즉시 가치
    - 🔄보다 즉시 파워는 높지만 런 전체 가치는 낮음
    - "지금 vs 나중" 긴장의 "지금" 쪽
  R4: 소규모 (★승급 1회, 테라진 8)
  R8: 중규모 (★승급 + 업그레이드, 전체 강화)
  R12: 대규모 (★승급 2회, 전체 대폭 강화)

🔄 영구규칙 (4/9) — "사기의 핵심. 규칙을 바꾼다"
  정의: 런 종료까지 게임 규칙 영구 변경. 숫자가 아니라 규칙.
  설계 축: 체인 규칙 / 경제 규칙 / 전투 규칙 / 상점 규칙
  원칙:
    - "ATK+5" ❌ → "강화 시 추가 유닛" ✅ (숫자가 아닌 규칙 변경)
    - 선택 후 보드를 재배치하고 싶어져야 함
    - 같은 보스의 🔄 중 최소 2개는 서로 다른 빌드 축 강화
    - R4 규칙 + R8 규칙이 조합되면 시너지 발생 (곱셈적)
  R4: 방향 제시 ("이 규칙이면 물량이 유리")
  R8: 곱셈적 강화 ("R4 규칙 + 이거 = 사기 시작")
  R12: 단독 사기 ("이것 하나로 게임이 바뀐다")

💥 직접강화 (2/9) — "카드에 영구 각인"
  정의: 특정 카드/유닛에 영구적 수치 변경. 카드에 흔적이 남음.
  설계 축: 카드 지정 (ATK ×2) / 태그 지정 (#대형 HP+50) / 에픽 부착
  원칙:
    - 플레이어가 대상을 선택하는 결정 포함 ("어떤 카드에?")
    - 결과가 즉시 눈에 보임 (스탯/유닛 수 변화)
    - ⚡과의 차이: ⚡은 행동(★승급, 골드), 💥은 수치 각인(ATK×2, 유닛 2배)
  R4: 중간 (고정값 증가)
  R8: 강력 (배율 또는 에픽 업그레이드)
  R12: 파격 (유닛 수 2배, 에픽 2개)

📐 구조 (1/9) — "판을 넓힌다"
  정의: 보드/경제의 구조적 제약 변경. 가장 희소.
  설계 축: 필드 +1 / 벤치 +2 / 슬롯 +1 / 이자상한 +1 / 상점 +1장
  원칙:
    - 보스당 1개 → 4개 표시 중 등장 확률 44%. 안 나올 수 있음.
    - 나오면 항상 강력한 선택지 (빌드 무관 범용 이득)
    - 다른 유형과 비교해도 매력적이지만 "사기" 재미(🔄)와는 다른 축
  R4: 기본 (벤치 +2, 상점 +1)
  R8: 중급 (필드 +1, 슬롯 +1)
  R12: 최강 (필드 +1 + 이자상한 +1 복합)
```

### 유형 간 선택 긴장

```
⚡ 즉시:  "지금 당장 강해진다"     vs  "런 후반엔 의미 감소"
🔄 영구:  "런 전체를 바꾼다"       vs  "지금 당장은 약할 수 있음"
💥 직접:  "핵심 카드가 확 강해짐"   vs  "다른 카드는 그대로"
📐 구조:  "모든 게 넓어진다"       vs  "구체적 파워 증가는 없음"

R4 핵심 긴장: ⚡즉시 ★승급 (지금 ★2로 R5~R7 안정) vs 🔄영구규칙 (R8 이후 폭발)

설계 검증:
  🔄 등장률 96% → 핵심 재미가 거의 항상 선택지에 존재 ✓
  📐 등장률 44% → 희소하지만 안 나와도 치명적이지 않음 ✓
  4개 선택지 동등 파워, 다른 방향 → 매번 의미 있는 결정 ✓
  개별 보상 내용은 카드 풀 완성 후 밸런스 조정.
```

## 프로토타입 후 조정 항목

```
기본 수입:    5/6/7 → 6/7/8 (판매 환급 포함해도 부족 시)
시작 골드:    0 → 2~3골드 (R1 구매력 부족 시)
전투 보상:    +1골드 → 0골드 (레벨업이 너무 빨리 돌아갈 시)
벤치 크기:    8 → 7~10 (보관 압박 과/부족 시)
```
