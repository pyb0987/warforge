## 비주얼 컨셉

```
신병:           경량 전투복+풀페이스 헬멧. 바이저가 선명한 주황빛.
               깔끔한 군장. 소형 에너지 소총. 빠르게 전진하는 경보병 무리.
               "대량 배치된 정규군 1선" 이미지.
               색감: 깨끗한 짙은 남색 전투복, 흰색 마킹, 바이저 주황 발광.

강화 보병:      광택 나는 풀 파워아머. 넓은 어깨 장갑판에 부대 엠블럼.
               팔에 내장형 에너지 블레이드. 관절부 밀봉된 매끄러운 라인.
               "교과서적인 미래 보병" — 정돈되고 위협적.
               색감: 메탈릭 짙은 남색, 어깨 장갑 흰색 엠블럼, 블레이드 밝은 청색.

에너지 방패병:  중장갑+대형 에너지 실드 발생기. 실드가 안정적인 밝은 푸른 면.
               실드 표면에 육각형 패턴. 뒤의 아군을 보호하는 전선의 벽.
               발생기가 깔끔하게 팔에 통합됨.
               색감: 중장갑 짙은 남색, 실드 밝은 푸른 투명+육각 패턴, 발생기 은백색.

정찰 드론:      소형 비행 드론. 매끄러운 유선형 외장. 로터 4개.
               하단에 센서 포드+소형 무기. 근거리 교란(Range 2).
               군 로고가 새겨진 깔끔한 무인기.
               색감: 매끄러운 짙은 남색, 센서 밝은 녹색 렌즈, 로터 블러.

강습 바이커:    유선형 호버바이크. 지면 위 부유, 하단에 푸른 추진광.
               탑승자는 경량 전투복+공기역학 헬멧. 빠른 측면 돌격.
               "깔끔한 레이싱 바이크를 무장시킨" 느낌.
               색감: 바이크 광택 짙은 남색+은색 라인, 추진부 밝은 청색 발광.

플라즈마 사수:  중형 플라즈마 라이플. 무릎 꿇고 조준 자세.
               라이플이 세련되고 정밀함. 냉각핀이 깔끔하게 배치.
               발사 시 포구에서 밝은 푸른 플라즈마 탄. 정확한 한 발.
               색감: 파워아머 짙은 남색, 라이플 은색+검정, 플라즈마 밝은 청백색.

저격 드론:      정찰보다 큰 무장 드론. 하단에 대형 레일건.
               외장이 매끄럽고 정밀. 스텔스 코팅 느낌.
               발사 시 레일건 충전 → 날카로운 흰 섬광. 정밀 타격.
               색감: 무광 검정+짙은 남색, 레일건 은색, 발사 시 선명한 흰 섬광.

궤도 포대:      무한궤도 위 대형 에너지 포. 자주포 형태.
               차체가 깔끔하고 각진 미래형 디자인. 포신이 길고 세련됨.
               발사 시 포신 주변에 에너지 링 이펙트+밝은 포구 섬광(Range 6).
               게임 내 최고 ATK(12). "정밀 궤도 타격."
               색감: 짙은 남색 차체, 포신 은백색, 발사 시 밝은 주황+청백 에너지.
               → 군대의 가장 인상적인 화력 유닛.

전술 지휘관:    장교용 프리미엄 파워아머. 다른 병사보다 세련됨.
               등에 소형화된 통신 모듈+홀로그램 투영기.
               한 손에 밝은 홀로그램 전장 맵. 침착한 지휘 자세.
               색감: 검정+짙은 남색 아머, 금색 견장/라인, 홀로그램 밝은 청녹.
               → "유능한 사령관" 이미지.

중장갑 워커:    이족 보행 전투 메카. 인간 3배 크기.
               양팔에 중화기(미사일 포드+개틀링). 깔끔한 장갑 패널.
               조종석 바이저가 선명한 주황빛. 관절부 정밀 유압 기구.
               근거리 화기 난사(Range 2). 느리지만 압도적.
               색감: 짙은 남색+은백색 장갑, 화기 검정, 조종석 주황, 관절 청색 발광.
               → 군대 최종 병기. "이것이 우리 최강 전력."
```

## 톤 키워드

```
전체 톤: "전쟁에 지친 디스토피아" ❌ → "첨단 정예군의 위엄" ✅

공통 요소:
  - 깔끔한 장갑 표면, 광택, 정돈된 마킹/엠블럼
  - 에너지 무기의 밝고 선명한 발광 (청색/청백/주황)
  - 통일된 짙은 남색 기조 + 은백색 포인트
  - 육각형 실드 패턴, 유선형 드론, 정밀한 메카
  - "기술적으로 우월한 정예군" — 보는 것만으로 강해 보임

4테마 톤 대비:
  스팀펑크: 어둡고 묵직, 기름/녹/증기 (산업적 우울)
  드루이드: 고대의 자연, 이끼/껍질/발광 (원시적 신비)
  포식종:  에일리언, 키틴/산성/점액 (불쾌한 위압)
  군대:    첨단 정예, 광택/에너지/정돈 (세련된 위엄) ← 유일하게 "멋진" 테마
```

## 실루엣 차별화

```
신병:          달리는 경장 보병 (인간형, 소형)
강화 보병:     광택 파워아머 (인간형, 중형)
에너지 방패병:  인간형+육각 에너지 실드 (인간형+실드)
정찰 드론:     유선형 쿼드콥터 (비행체)
강습 바이커:   유선형 호버바이크+탑승자 (차량형)
플라즈마 사수:  무릎 꿇은 사수+세련된 라이플 (인간형, 조준 자세)
저격 드론:     스텔스 무장 드론+하단 포신 (비행체, 정찰보다 큼)
궤도 포대:     미래형 자주포 (차량형, 포신 위를 향함)
전술 지휘관:   장교 아머+홀로그램 (인간형, 장교)
중장갑 워커:   이족 메카+양팔 화기 (메카, 대형)

→ 인간형 5 / 비행체 2 / 차량형 2 / 메카 1
→ 10종 모두 실루엣 구분 ✓
```

## 특징: #생체 + #기계 혼합

```
#생체 6종: 신병, 강화보병, 방패병, 바이커, 사수, 지휘관
#기계 4종: 정찰드론, 저격드론, 궤도포대, 워커

→ 유일하게 양쪽 태그를 모두 보유하는 테마
→ 크로스 테마 잠재력이 가장 높음
```
