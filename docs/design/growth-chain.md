# 성장 체인 & 이벤트 시스템

## 성장 체인 시스템 (핵심 메카닉)

매 라운드 빌드 페이즈 종료 후, 배치된 카드의 성장 트리거가 **배치 순서(왼→오)**로 발동.
트리거 효과가 새 이벤트를 생성하면 다시 왼→오로 스캔 → **성장 체인**.
이 체인이 군대를 키우는 핵심 메카닉이다.

### 성장 곡선

```
★1 초반 (R1~R5):  성장 체인 미약. 2~4체인. 라운드당 소폭 성장.
★2 중반 (R6~R10): 성장 체인 본격화. 8~12체인. "조합이 맞물린다" 순간.
★2+업글 (R11~R15): 성장 체인 강화. 12~18체인. 업그레이드로 전투력 보강.
★3 (드문 보너스):   성장 체인 폭발. 20+체인. 완전한 "사기". 압도적 돌파.
```

### 시각 피드백 원칙

```
★1: 기본 색상, 이펙트 없음
★2: 색상 강화 + 은은한 반짝임. "이 카드 좀 키웠다"
★3: 완전히 다른 색감 + 화려한 이펙트. 보는 것만으로 만족.
     성장 체인 발동 시 전용 연출 (발라트로의 스코어링 연출처럼)

성장 체인 발동 시:
  - 체인 카운터 표시 ("3체인!", "10체인!", "20체인!!")
  - 체인 길어질수록 이펙트 강화 (소리, 화면 흔들림, 색상 강조)
  - 유닛 수/스탯 변화가 실시간으로 카드에 반영
```

## 이벤트 타입 (2층 구조)

### 층 1: 결과 범주 (6종) — 테마 무관

**성장 이벤트 (빌드 페이즈 — 주력)**
| 이벤트 | 설명 | 발생 원인 |
|--------|------|-----------|
| 🔧 유닛 추가 | 카드의 유닛 수가 증가 | 트리거 효과 (키워드별 독립) |
| 📈 강화 | 유닛 스탯이 영구 상승 | 트리거 효과 |
| ⬆ 합성 | 카드가 ★ 업그레이드 | 같은 카드 3장 합성 |

**전투 이벤트 (전투 중 — 보조)**
| 이벤트 | 설명 | 발생 원인 |
|--------|------|-----------|
| ⚔ 공격 | 유닛이 공격 실행 | 리얼타임 전투 자연 발생 (트리거 조건만, 생성 불가) |
| 💀 사망 | 유닛이 전사 | 체력 0 |
| 🛡 버프 | 유닛이 일시적 전투 강화 | 트리거 효과 |

### 층 2: 키워드 이벤트 (테마별)

```
카드 효과가 특정 "키워드"를 생성하고, 다른 카드가 그 키워드에 반응.
키워드는 테마/세력별로 고유하며, 같은 결과 범주에 속해도 독립 트리거.

구조:
  층 1 (결과 범주)        층 2 (키워드, 테마별)
  ─────────────────    ──────────────────────────
  🔧 유닛 추가     ←── 제조(스팀) / 번식(드루) / 부화(충군) / 징집(군대)
  📈 강화          ←── 개량(스팀) / 성장(드루) / 변태(충군) / 훈련(군대)
  ⚡ 충전 (범용)    ←── 대상 카드의 발동 상한 +1 (이번 라운드)

원칙:
  - 키워드별 독립: "제조 시" ≠ "번식 시" (유희왕 원칙)
  - 결과 범주 반응: "유닛 추가 시" = 제조든 번식이든 모두 반응
```

### 인접 위치 조건

```
필드 배치: [A] [B] [C] [D] [E]

- "인접 카드가 ~했을 때": B의 인접 = A, C
- "왼쪽 카드가 ~했을 때": C의 왼쪽 = B
- "같은 태그의 인접 카드가 ~했을 때": 위치 + 태그 복합 조건

→ 배치 순서 = 성장 체인 전략의 핵심
→ 왼쪽 카드의 효과가 먼저 발동하므로, 체인 시작점을 왼쪽에 배치
```

### 트리거 발동 시점

```
┌─ 빌드 페이즈 (주력):
│   ├─ 라운드 시작 시: 유닛추가/강화 체인 (매 라운드 자동 성장)
│   ├─ 카드 구매 시
│   ├─ 카드 판매 시
│   ├─ 합성(★ 상승) 시
│   └─ 성장 이벤트 연쇄: 유닛추가→유닛추가, 📈강화→유닛추가 등
│
├─ 전투 전후:
│   ├─ 전투 시작 직전: 준비 트리거
│   ├─ 전투 승리 후: 보상 트리거
│   └─ 전투 패배 후: 회복/성장 트리거
│
└─ 전투 중 (보조):
    ├─ 유닛 사망 시 (💀) → 단순 🛡버프
    └─ 특정 조건 시 (N번째 공격 등)
```

## 발동 모델

```
기본 원칙:
  카드는 트리거 조건에 맞는 이벤트 1개당 1회 발동.
  "발동 N회" = 라운드당 최대 N회 반응 가능한 상한 (cap).
  한 번에 여러 번 발동하는 것이 아님.

라운드 시작 카드:
  라운드 시작 이벤트는 1번만 발생 → 1회 발동.
  ★ 진화 = 발동 횟수 증가가 아닌 효과 자체가 강해짐.

이벤트 반응 카드:
  매칭 이벤트 1개당 1회 발동, 상한까지.
  ⚡충전으로 상한 +1 → 이번 라운드 1회 더 반응 가능.

추가 발동을 얻는 방법:
  ⚡충전: 대상 카드 발동 상한 +1
  특정 효과: "인접 카드를 1회 추가 발동" (명시적 재발동)
```

## 이벤트 흐름 규칙

1. 특정 시점에 이벤트 발생 (라운드 시작, 구매, 합성, 전투 중 사망 등)
2. **배치 순서(왼→오)**로 해당 이벤트의 트리거 조건을 가진 카드 확인
3. 조건 충족 + 발동 횟수 남음 → 효과 1회 발동 (상한 -1) → 새 이벤트 생성 가능
4. 새 이벤트가 다시 2번으로 → **체인**
5. 더 이상 트리거할 카드 없으면 체인 종료
