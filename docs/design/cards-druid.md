# 드루이드 카드 풀 (10장)

> 효과/수치 상세: `data/cards/druid.yaml` 참조 (단일 진실 소스).
> 이 문서는 설계 의도와 빌드 전략을 기술합니다.

> 유닛 풀: units-druid.md 참조.
> 핵심 판타지: "고대의 숲이 깨어남" — 소수 정예 유닛이 나무와 함께 성장.
> 고유 엔진: 나무 토큰 + 세계수 누적 곱연산.

## 나무 시스템

```
🌳 = 카드별 내부 카운터. 비전투. "🌳 N" 표시.
숲의 깊이 = 전체 드루이드 카드의 🌳 합계. "숲의 깊이: 🌳 N" 표시.

성장(N) = 이 카드의 🌳 × N% ATK+HP 증강 (카드 성장 modifier 누적)
번식 = 인접 카드의 최강 유닛(CP)을 그 카드에 복제

나무 생성:  요람(+1자기+1인접/R), 맥동(+1/전투), 뿌리깊은 자(+1/R), 세계수뿌리(+1/R), 가지치기(+1자기/R)
나무 변환:  가지치기(유닛→🌳, 대상 카드에 귀속)
나무 분배:  요람(인접+1), 세계수뿌리(🌳4이상→전체+1), 세계수(전체+1)
나무 흡수:  근원(인접에서 재분배, 자체+1)
나무 비생성: 은혜, 포자, 태고 (세계수/요람/세계수뿌리로 공급)
판매 규칙:  드루이드 카드 판매 시 그 카드의 🌳을 다른 드루이드 카드에 균등 분배

카드 🌳 조건부 효과 = 지속 조건 (🌳이 조건 미만이면 중단, 이상이면 재작동).

★합성 시 🌳:  3장의 🌳을 합산하여 승계. (예: 🌳4+🌳3+🌳3 = ★2 카드 🌳10)
이벤트 내 순서: 🌳 증가 → 효과 적용 (동일 트리거 내에서 항상 이 순서).
판매 시 대상 0장: 다른 드루이드 카드가 없으면 🌳 소멸.
흡수 하한:     인접 카드 🌳이 0이면 흡수량 0 (자기 🌳+1은 정상 적용).
"전체 드루이드 카드": 자기 자신 포함 (세계수뿌리, 세계수 모두 동일).
```

## 개요

| # | 이름 | 티어 | 🌳 | 🌳 비례 효과 | 소수정예 |
|---|------|------|-----|-------------|---------|
| 1 | 숲의 요람 | T1 | 자기+인접/R | 나무 생성+배분 전문 | — |
| 2 | 생명의 맥동 | T1 | 전투마다 | 🌳 비례 방어막 | 소수 시 계수 보너스 |
| 3 | 오래된 근원 | T2 | 자기+흡수/R | 🌳 비례 전체 드루이드 성장 | 소수 시 계수 보너스 |
| 4 | 숲의 은혜 | T2 | 비생성 | 🌳 비례 골드, 🌳 임계→테라진 | — |
| 5 | 가지치기 | T2 | +1/R | 유닛 최다 카드의 최약 유닛→🌳 변환 | 소수최강 강제 |
| 6 | 뿌리깊은 자 | T3 | +1/R | 🌳 비례 성장 + 🌳 임계→ATK 곱연산 | 소수 시 성장 보너스 |
| 7 | 포자 구름 | T3 | 비생성 | 🌳 비례 적 AS 감소, 상한 존재 | — |
| 8 | 태고의 분노 | T4 | 비생성 | 🌳 비례 이 카드 ATK 대폭 강화 | ≤5/6/7기 (★별 상한 증가) |
| 9 | 세계수의 뿌리 | T4 | +1/R | 🌳 임계→전체 드루이드 카드 🌳 배분 | — |
| 10 | 세계수 | T5 | 자기+전체/R | 누적 곱연산(ATK/HP/AS, 상한 없음) | ≤20/40/200 |

## 서브 전략: 세계수형 vs 정원형

```
세계수형 (소수 정예 — 수동 축적 → 후반 폭발):
  요람(T1) → 맥동(T1) → 은혜(T2) → 뿌리깊은 자(T3) → 포자(T3) → 세계수뿌리(T4) → 태고(T4) → 세계수(T5)
  ★1은 유닛 ≤20 유지. ★2는 ≤40, ★3은 사실상 제한 없음(≤200).
  핵심: deep(자기 집중 성장) + world(곱연산).

정원형 (소수 정예 — 능동 관리 → 중반 안정 성장):
  요람(T1) → 맥동(T1) → 은혜(T2) → 가지치기(T2) → 근원(T2) → 포자(T3) → 태고(T4) → +중립/다른테마
  가지치기로 약한 유닛 자동 제거 → 🌳 → 근원이 흡수하여 전체 드루이드 성장.
  deep/wt_root/world 없이도 기능. 세계수 불필요.

공유 카드: 요람, 맥동, 은혜, 포자, 태고(용병)
분기 시점: T2 (가지치기+근원 vs 뿌리깊은 자+세계수뿌리 방향)
배타 관계: 가지치기+근원 ↔ deep+wt_root (세계수형 anti_cards)
```

## ★2 합성 유인 검증

```
요람:   ★1×2 동등 효과 + 슬롯 절약.               ✓ 동등+슬롯
맥동:   범위 유지, 방어막 기본값+계수 강화.           ✓ 수치
근원:   흡수량 증가, 성장 계수 강화, low_unit 완화.   ✓ 수치
은혜:   기본 골드 증가, 테라진 임계 완화, 패배 페널티 제거. ✓ 수치
가지치기: 변환 수 유지, ★2에서 enhance 추가 획득.      ✓ 신규 효과
뿌리:   성장 계수 강화, 곱연산 유지, 임계 완화.        ✓ 수치
포자:   AS 감소 기본값 증가.                         ✓ 수치
태고:   ATK 강화 기본값 대폭 증가.                   ✓ 수치
뿌리T:  배분 임계 완화 (더 일찍 시작).               ✓ 수치
세계수:  ATK 곱연산 기본값 증가, 깊이 증가 주기 단축.  ✓ 수치
```

## 나무 흐름도

```
🌳 생성:     요람(자기+인접/R), 맥동(전투마다), 뿌리깊은 자(+/R), 세계수뿌리(+/R), 대지축복(+/R)
🌳 분배:     요람(인접), 세계수뿌리(임계→전체 배분), 세계수(전체 배분)
🌳 흡수:     근원(인접에서 자기로, +자체)
🌳 판매:     드루이드 카드 판매 시 🌳 균등 분배
🌳→성장:    뿌리깊은 자(🌳 비례 자기 성장), 근원(🌳 비례 전체 드루이드 성장)
🌳→방어막:   맥동(🌳 비례 방어막)
🌳→경제:    은혜(🌳 비례 골드, 임계→테라진)
🌳→디버프:   포자(🌳 비례 AS 감소)
🌳→ATK:    태고(🌳 비례 ATK 대폭 강화)
유닛→🌳:    가지치기(유닛 최다 카드의 최약 유닛 제거 → 해당 카드 🌳)
🌳→곱연산:   뿌리깊은 자(임계→ATK 곱연산), 세계수(ATK/HP/AS 깊이 누적 곱연산, 상한 없음)
```

## 소수정예

```
세계수:       전체 유닛 ★1≤20, ★2≤40, ★3≤200 (사실상 무제한)
태고의 분노:   이 카드 유닛 ★1≤5, ★2≤6, ★3≤7기 (ATK 보너스 조건)
뿌리깊은 자:   이 카드 유닛 소수 시 성장 계수 보너스 (임계는 YAML 참조)
맥동:         이 카드 유닛 소수 시 방어막 계수 보너스 (임계는 YAML 참조)
```

## 타임라인 — 세계수형 (현실적 카드 획득 기준)

```
R1:    [요람][맥동] 배치. 맥동 요람 인접으로 빠른 🌳 축적. 방어막 점점 강화 (수치는 YAML 참조).
R3-4:  은혜(T2) 합류. 전투 종료 시 골드 (패배 시 절반). 초반 경제 피드백.
R5:    맥동 🌳 임계 도달 → 방어막 강화. 뿌리깊은 자(T3) 합류.
R7:    세계수뿌리(T4) 합류. 몇 라운드 내 🌳 임계 → 전체 배분 시작.
R8:    (선택) 맥동 판매 → 🌳 분배 → 비생성 카드 활성화.
R9-10: 세계수(T5) 합류 → ATK 곱연산 시작 + 전체 🌳 배분.
       숲의 깊이 축적에 따라 곱연산 계수 단계 상승 (수치는 YAML 참조).
R11:   뿌리깊은 자 🌳 임계 → ATK 곱연산 적용 (이 카드만).
R13-14: 숲의 깊이 증가 → 세계수 곱연산 계수 단계 상승 반복.
        ★2/★3 세계수는 계수 기본값+주기 단축으로 더 빠른 스텝업.
        (태고: 이 카드만 추가 ATK. 세계수: ATK/HP/AS 모두 곱연산.)
```

## 타임라인 — 정원형

```
R1:    [요람][맥동] 배치. 맥동 빠른 🌳 축적. 방어막.
R3-4:  은혜(T2), 가지치기(T2), 근원(T2) 중 선택 합류.
       가지치기: 유닛 최다 카드에서 최약 2기→🌳. 약한 유닛 자동 정리.
       근원: 요람 인접 → 흡수로 🌳 축적 → 전체 드루이드 성장.
R5-6:  가지치기가 매 라운드 유닛 정리 → 맥동 low_unit 보너스 자연 활성화.
       근원: 흡수된 🌳으로 전체 드루이드 ATK+HP 누적 성장.
R7-8:  포자(T3) 또는 태고(T4) 합류.
       근원 🌳 축적 → ★3 tree_bonus 임계 도달 시 추가 성장.
       세계수형과 달리 deep/wt_root 없이도 안정적 중반 전력.
R10+:  태고의 분노(T4): ≤5/6/7기 유닛캡. 가지치기가 유닛 수 자동 관리.
       가지치기 ★3: 3기 변환 + 5% enhance → 정리+강화 동시.
```
