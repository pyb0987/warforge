# 중립 카드 풀 (14장)

> 유닛 풀: units-neutral.md 참조. 어떤 테마 유닛이든 사용 가능.
> Layer 1 결과 범주 직접 사용 (테마 무관). 굿스터프 덱의 코어.
> 정체성: "방랑자·고대유물·자연의 힘" — 어느 테마에도 속하지 않는 보편적 존재들.
>
> **중립 메커니즘**:
> - 유닛 추가 = 대상 카드 composition 내 무작위 유닛 1기 추가. Layer 1 UNIT_ADDED 방출 (Layer 2 없음).
> - 강화(ATK +N%) = 대상 카드의 성장 누적 modifier에 +N% 추가 (영구). Layer 1 ENHANCED 방출.
> → 테마 카드의 Layer 2 키워드(제조/개량/번식 등)와 독립. 중립 효과는 Layer 2를 방출하지 않음.
>
> **엔진**: 시작기(이벤트 생성) → 브릿지(범주 교차) → 증폭기(같은 범주 확산) → 체인 연장.
> **굿스터프**: 중립 5~6 + 테마 용병 2~3으로 독립 아키타입.
> **크로스 테마**: 테마 덱에 중립 1~2장 삽입하여 Layer 1 반응으로 체인 확장.

## 빌드 패턴

```
굿스터프 (중립 5~6 + 용병 2~3):
  시작기 → 브릿지 → 증폭기 → (용병의 Layer 1 이벤트도 합류)
  중립 엔진이 주력. 용병은 Layer 2 시너지 없이 Layer 1로만 기여.
  차원 행상인(T4)이 테마 다양성을 경제+물량으로 전환.

하이브리드 (테마 5~6 + 중립 2~3):
  테마 Layer 2 체인 + 중립 브릿지가 Layer 1로 추가 반응 → 체인 +2~4 연장.
  경제/전투 유틸리티로 테마 약점 보완.
```

## 상세

```
[대지의 울림] T1 — 체인 시작기 (UNIT_ADDED)  ← 신규
구성: 고철 수집가 ×2, 기계 독수리 ×1       (3유닛, ATK 9, HP 65)
카드 태그: #중립 #생산
트리거: 라운드 시작
효과: 오른쪽 인접 카드에 유닛 1기 추가
★2: 유닛 2기 추가
★3: 양쪽 인접 카드에 유닛 2기 추가

  매 라운드 UNIT_ADDED를 무조건 생성하는 엔진 점화기.
  배치 왼쪽 끝에 놓고 오른쪽으로 체인을 시작하는 구조.
  ★3에서 양쪽 범위 확장 → 배치 유연성 확보.

[야생의 맥동] T1 — 체인 시작기 (ENHANCED)  ← 신규
구성: 방랑 궁수 ×1, 야생 골렘 ×1            (2유닛, ATK 8, HP 100)
카드 태그: #중립 #강화
트리거: 라운드 시작
효과: 자신 카드 강화(ATK +3%, 영구 누적)
★2: 강화(ATK +5%)
★3: 인접 카드도 강화(ATK +3%)

  매 라운드 ENHANCED를 무조건 생성. 자신이 점점 강해지는 축적형.
  돌연변이 적응(T3)과 연결 시 ENHANCED → UNIT_ADDED 체인 파생.
  ★3에서 인접도 강화 → 주변 카드를 키우는 오라로 전환.

[방랑 상인] T1 — 경제 (패배 보상)  ← 수치 개편
구성: 방랑 궁수 ×1, 고철 수집가 ×1         (2유닛, ATK 7, HP 55)
카드 태그: #중립 #경제
트리거: 전투 패배 시
효과: 3골드 획득
발동: 최대 1/R
★2: 4골드 + 1 테라진 획득
★3: 승패 무관 발동, 3골드 + 자신 유닛 1기 추가

  초반 패배 시 경제 안전망. 3골드는 T1 카드 1장 구매 가능 수준.
  ★2에서 테라진까지 주므로 합성 경제 보조.
  ★3에서 승패 무관 + 매 전투 유닛 추가 → 경제 카드가 물량 카드로 질적 전환.
  유닛 추가는 UNIT_ADDED emit → 전투 종료 후 체인에도 참여.

[유적의 공명] T2 — 체인 시작기 (듀얼)  ← 신규
구성: 고대 파수꾼 ×1, 고철 수집가 ×1       (2유닛, ATK 12, HP 60)
카드 태그: #중립 #고대
트리거: 라운드 시작
효과: 자신 유닛 1기 추가, 자신 카드 강화(ATK +2%, 영구 누적)
★2: 유닛 1기 + 강화(ATK +4%)
★3: 유닛 2기 + 강화(ATK +4%), 유닛 8기 이상이면 인접 카드도 강화(ATK +2%)

  UNIT_ADDED + ENHANCED를 동시에 생성하는 듀얼 시작기.
  T1 시작기 2장 합친 효과이나 티어가 높아 초반 접근성은 낮음.
  ★3에서 유닛 축적 시 주변까지 강화 → 중후반 핵심 엔진.

[떠돌이 무리] T2 — 브릿지 (UNIT_ADDED → ENHANCED)  ← 개편
구성: 떠돌이 용병 ×1, 고철 수집가 ×2       (3유닛, ATK 9, HP 95)
카드 태그: #중립 #범용
트리거: 다른 카드의 유닛 추가 효과 발동 시 (Layer 1 — UNIT_ADDED)
효과: 유닛이 추가된 카드 강화(ATK +3%, 영구 누적)
발동: 최대 2/R
★2: 강화(ATK +5%)
★3: 최대 3/R, 강화 시 자신 유닛 1기 추가

  새 유닛이 온 카드를 즉시 강화. 시작기의 효과를 증폭하는 핵심 연결고리.
  용병 카드가 유닛 추가 효과를 쓰면 용병 자체가 강화됨 → 용병 가치 증대.
  ★3에서 자기 유닛도 추가 → UNIT_ADDED 자체 생산. 자급 브릿지.

[마력 결정] T2 — 증폭기 (UNIT_ADDED 확산)  ← 재설계
구성: 마력 정령 ×1, 고철 수집가 ×1           (2유닛, ATK 8, HP 45)
카드 태그: #중립 #마력
트리거: 다른 카드의 유닛 추가 효과 발동 시 (Layer 1 — UNIT_ADDED)
효과: 유닛이 추가된 카드의 양쪽 인접 카드에도 유닛 1기씩 추가
발동: 최대 2/R
★2: 인접 카드에 유닛 2기씩 추가
★3: 최대 3/R, 확산 대상에도 유닛 1기 추가 (3방향 확산)

  > 재설계: CHARGE(간접 증폭) → UNIT_ADDED 확산(직접 이벤트 생성).
  > max 상향: 1/R → 2/R (★3: 3/R). 체인 길이 확보를 위한 증폭기 상한 완화.

  하나의 UNIT_ADDED가 최대 3개로 확산. 브릿지의 반응 기회를 증가.
  2/R로 Wave 2까지 반응 가능 → 체인 조기 소진 방지.
  ★3에서 3/R + 3방향 확산 → 물량 대폭발.

[야생의 힘] T2 — 전투 보조
구성: 광란의 야수 ×1, 키메라 ×1             (2유닛, ATK 15, HP 85)

[고대의 잔해] T2 — 인내 (중기)  ← 일부 개편
구성: 야생 골렘 ×1, 마력 정령 ×1             (2유닛, ATK 9, HP 90)
카드 태그: #중립 #시간
트리거: 라운드 시작 (조건: 필드에 2라운드+ 체류)

★2: 3골드, 유닛 2기 추가
★3: 양쪽 인접 카드에 유닛 2기 추가, 4라운드+ 체류 시 매 라운드 인접 카드에도 유닛 1기씩 추가

  > ★3 개편: 자기 강화 → 인접 유닛 생산. 인내 보상이 "물량 확산"으로 전환.

  체류할수록 가치가 올라가는 투자형 카드.
  유닛 추가 효과는 UNIT_ADDED를 방출 → 중립 체인에 합류.
  ★3에서 장기 체류 시 인접 카드에도 유닛을 계속 공급 → 오래 놔둘수록 주변이 부유해짐.

[돌연변이 적응] T3 — 브릿지 (ENHANCED → UNIT_ADDED)  ← 신규
구성: 광란의 야수 ×1, 돌연변이 거수 ×1       (2유닛, ATK 14, HP 135)

발동: 최대 2/R
★2: 유닛 2기 추가

  > ★3 개편: 자기 강화 → 인접 유닛 확산. 유닛 축적 보상이 물량 공유로 전환.

  강화 이벤트를 유닛 추가로 전환하는 역방향 브릿지.
  떠돌이 무리(T2)와 쌍으로 UNIT↔ENHANCED 순환 루프 형성.
  ★3에서 유닛 축적 시 인접에도 유닛 → 자기 성장이 주변으로 번지는 쾌감.

[고대 촉매] T3 — 증폭기 (ENHANCED 확산)  ← 재설계
구성: 고대 파수꾼 ×1, 마력 정령 ×1           (2유닛, ATK 16, HP 55)
카드 태그: #중립 #고대

  마력 결정이 UNIT을 퍼뜨리면 이것은 ENHANCED를 퍼뜨림 → 양 채널 증폭.
  2/R로 Wave 2까지 반응 가능 → 체인 조기 소진 방지.

[정령의 축복] T3 — 경제 (합성 보상)  ← 개편

  굿스터프 전용 경제 엔진. 용병이 많을수록 수익과 물량 증가.
  ★3에서 "용병을 모을수록 경제+물량이 좋아진다" — 굿스터프의 정체성을 대변.
  예시: 중립5+용병3(3테마) = 4종×3골드(12골드) + 4테라진 + 용병3장에 각 유닛1기.
  용병 유닛 추가는 UNIT_ADDED emit → 체인에 합류.

[고대의 각성] T4 — 인내 (장기)  ← 대폭 개편
구성: 고대 파수꾼 ×1, 야생 골렘 ×1           (2유닛, ATK 13, HP 105)
카드 태그: #중립 #고대
트리거: 이 카드가 필드에 4라운드+ 체류 시 1회 발동 (★ 합성 시 카운터 리셋)
효과: 전체 아군 카드에 유닛 2기씩 추가, 전체 아군 강화(ATK +10%, 영구 누적), 전체 아군 유닛에 🛡방어막(기본 HP의 20%)
★2: 유닛 3기, 강화(ATK +15%), 방어막 30%
★3: +발동 후 매 라운드, 전체 아군 카드에 유닛 1기 추가 + 🛡방어막(기본 HP의 10%)

  > ★3 조정: 강하지만 자동 승리 아닌 수준. 강화(ATK%) 제거, 유닛 2→1기, 방어막 15→10%.

  4라운드 인내의 대보상. 전체 아군에 유닛+방어막 동시 적용.
  발동 시 모든 아군 카드에서 UNIT_ADDED 방출 → 체인에 합류.
  ★ 합성 시 카운터 리셋 → 한 런에서 2번 발동 가능 (★1 후 ★2로).
  ★3에서 매 라운드 전체 16유닛+40%ATK+방어막 → 후반 굿스터프의 캡스톤.
  R11 발동 후 R15까지: +64유닛, +20%ATK, 방어막 4겹. 도달하면 게임을 지배.
  R7 획득→R11 발동, R9 획득→R13 발동. R10까지 찾으면 유효.
```

## 체인 흐름

```
Layer 1 이벤트 소스 (라운드 시작):
  대지의 울림 (T1) → 인접 유닛 추가          → UNIT_ADDED
  야생의 맥동 (T1) → 자신 강화              → ENHANCED
  유적의 공명 (T2) → 유닛 추가 + 강화        → UNIT_ADDED + ENHANCED
  고대의 잔해 (T2) → 인접 유닛 추가 (2R+)    → UNIT_ADDED

UNIT_ADDED 반응:
  떠돌이 무리 (T2) → 해당 카드 강화           → ENHANCED    [브릿지: 범주 교차]
  마력 결정 (T2)   → 인접 카드에도 유닛 추가   → UNIT_ADDED  [증폭기: 같은 범주 확산]

ENHANCED 반응:
  돌연변이 적응 (T3) → 자신 유닛 추가         → UNIT_ADDED  [브릿지: 범주 교차]
  고대 촉매 (T3)    → 인접 카드에도 강화      → ENHANCED    [증폭기: 같은 범주 확산]

브릿지 vs 증폭기:
  브릿지 = 범주 교차 (UNIT→ENHANCED, ENHANCED→UNIT) → 다른 채널 활성화
  증폭기 = 같은 범주 확산 (UNIT→더 많은 UNIT, ENHANCED→더 많은 ENHANCED) → 같은 채널 증폭

순환 루프 (★1 기준):
  시작기(UA) → 떠돌이(EN) + 마력결정(UA×2 확산) → 돌연변이(UA) + 고대촉매(EN×2 확산) → ...
  브릿지 각 max 2/R, 증폭기 각 max 2/R → 총 반응 8회 + 시작기 3~4회 = 체인 약 12
  ★2에서 증폭기 수치↑, ★3에서 max 3/R → 체인 15~18

전투 전환:
  야생의 힘 (T2): 전투 ATK 버프
  키메라의 울부짖음 (T3): 패배 → 영구 ATK/HP 성장, ★3 인접 유닛+HP재생

경제:
  방랑 상인 (T1): 패배 → 골드 (초반 안전망, ★3에서 물량 전환)
  고대의 잔해 (T2): 체류 → 골드+유닛 (중기 투자, ★3 인접 유닛 생산)
  정령의 축복 (T3): 합성 → 테라진+유닛 (합성 가치 물량 증폭)
  차원 행상인 (T4): 테마 다양성 → 골드+테라진+유닛+강화 (★3에서 굿스터프 캡스톤)

인내:
  고대의 잔해 (T2): 2R 체류 → 중기 보상
  고대의 각성 (T4): 4R 체류 → 대폭발 (★3에서 매 라운드 전체 엔진, 판도 전환)
```
