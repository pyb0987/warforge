# 중립 카드 풀 (15장)

> 수치 상세: `data/cards/neutral.yaml` 참조 (단일 진실 소스).
> 이 문서는 설계 의도와 빌드 전략을 기술합니다.

> 유닛 풀: units-neutral.md 참조. 어떤 테마 유닛이든 사용 가능.
> Layer 1 결과 범주 직접 사용 (테마 무관). 굿스터프 덱의 코어.
> 정체성: "방랑자·고대유물·자연의 힘" — 어느 테마에도 속하지 않는 보편적 존재들.
>
> **중립 메커니즘**:
> - 유닛 추가 = 대상 카드 composition 내 무작위 유닛 1기 추가. Layer 1 UNIT_ADDED 방출 (Layer 2 없음).
> - 강화(ATK +N%) = 대상 카드의 성장 누적 modifier에 +N% 추가 (영구). Layer 1 ENHANCED 방출.
> → 테마 카드의 Layer 2 키워드(제조/개량/번식 등)와 독립. 중립 효과는 Layer 2를 방출하지 않음.
>
> **엔진**: 시작기(이벤트 생성) → 브릿지(범주 교차) → 증폭기(같은 범주 확산) → 체인 연장.
> **굿스터프**: 중립 5~6 + 테마 용병 2~3으로 독립 아키타입.
> **크로스 테마**: 테마 덱에 중립 1~2장 삽입하여 Layer 1 반응으로 체인 확장.

## 빌드 패턴

```
굿스터프 (중립 5~6 + 용병 2~3):
  시작기 → 브릿지 → 증폭기 → (용병의 Layer 1 이벤트도 합류)
  중립 엔진이 주력. 용병은 Layer 2 시너지 없이 Layer 1로만 기여.
  차원 행상인(T4)이 테마 다양성을 경제+물량으로 전환.

하이브리드 (테마 5~6 + 중립 2~3):
  테마 Layer 2 체인 + 중립 브릿지가 Layer 1로 추가 반응 → 체인 +2~4 연장.
  경제/전투 유틸리티로 테마 약점 보완.
```

## 상세

```
[대지의 울림] T1 — 체인 시작기 (UNIT_ADDED)

  매 라운드 UNIT_ADDED를 무조건 생성하는 엔진 점화기.
  배치 왼쪽 끝에 놓고 오른쪽으로 체인을 시작하는 구조.
  ★3에서 양쪽 범위 확장 → 배치 유연성 확보.

[야생의 맥동] T1 — 체인 시작기 (ENHANCED)

  매 라운드 ENHANCED를 무조건 생성. 이 카드가 점점 강해지는 축적형.
  돌연변이 적응(T3)과 연결 시 ENHANCED → UNIT_ADDED 체인 파생.
  ★3에서 인접도 강화 → 주변 카드를 키우는 오라로 전환.

[방랑 상인] T1 — 경제 (패배 보상)

  초반 패배 시 경제 안전망 (수치는 YAML 참조).
  ★2에서 테라진까지 주므로 합성 경제 보조.
  ★3에서 승패 무관 + 매 전투 유닛 추가 → 경제 카드가 물량 카드로 질적 전환.
  유닛 추가는 UNIT_ADDED emit → 전투 종료 후 체인에도 참여.

[유적의 공명] T2 — 체인 시작기 (듀얼)

  UNIT_ADDED + ENHANCED를 동시에 생성하는 듀얼 시작기.
  T1 시작기 2장 합친 효과이나 티어가 높아 초반 접근성은 낮음.
  ★3에서 유닛 축적 시 주변까지 강화 → 중후반 핵심 엔진.

[떠돌이 무리] T2 — 브릿지 (UNIT_ADDED → ENHANCED)

  새 유닛이 온 카드를 즉시 강화. 시작기의 효과를 증폭하는 핵심 연결고리.
  용병 카드가 유닛 추가 효과를 쓰면 용병 자체가 강화됨 → 용병 가치 증대.
  ★3에서 이 카드에도 유닛 추가 → UNIT_ADDED 자체 생산. 자급 브릿지.

[마력 결정] T2 — 증폭기 (UNIT_ADDED 확산)

  하나의 UNIT_ADDED가 최대 3개로 확산. 브릿지의 반응 기회를 증가.
  2/R로 Wave 2까지 반응 가능 → 체인 조기 소진 방지.
  ★3에서 3/R + 3방향 확산 → 물량 대폭발.

[야생의 힘] T2 — 전투 보조

[고대의 잔해] T2 — 인내 (중기)

  체류할수록 가치가 올라가는 투자형 카드.
  유닛 추가 효과는 UNIT_ADDED를 방출 → 중립 체인에 합류.
  ★3에서 장기 체류 시 인접 카드에도 유닛을 계속 공급 → 오래 놔둘수록 주변이 부유해짐.

[돌연변이 적응] T3 — 브릿지 (ENHANCED → UNIT_ADDED)

  강화 이벤트를 유닛 추가로 전환하는 역방향 브릿지.
  떠돌이 무리(T2)와 쌍으로 UNIT↔ENHANCED 순환 루프 형성.
  ★3에서 유닛 축적 시 인접에도 유닛 → 이 카드의 성장이 주변으로 번지는 쾌감.

[고대 촉매] T3 — 증폭기 (ENHANCED 확산)

  하나의 ENHANCED가 최대 여러 개로 확산. 돌연변이 적응의 반응 기회를 증가.
  마력 결정이 UNIT을 퍼뜨리면 이것은 ENHANCED를 퍼뜨림 → 양 채널 증폭.
  2/R로 Wave 2까지 반응 가능 → 체인 조기 소진 방지.
  ★3에서 이 카드도 강화 → ENHANCED 추가 emit, 이 카드가 확산한 것을 다시 증폭.

[정령의 축복] T3 — 경제 (합성 보상)

  합성할 때마다 테라진 + 유닛 추가. 합성의 가치를 물량으로 증폭.
  UNIT_ADDED를 방출하므로 떠돌이 무리(브릿지)와 마력 결정(증폭기)을 트리거.
  ★3에서 전체 유닛 버스트 → 합성 한 번이 전체 체인을 점화.

[키메라의 울부짖음] T3 — 전투 보조 (패배 성장)

  "죽이지 못하면 더 강해진다." 키메라 진화 컨셉.
  패배할 때마다 영구 ATK/HP 성장. 연패 후 역전하는 쾌감.
  ★3에서 승패 무관 + 인접 유닛 + 방어막 → 불사의 키메라가 점점 단단해짐.

[차원 행상인] T4 — 경제 (다양성 보상)

  굿스터프 전용 경제 엔진. 용병이 많을수록 수익과 물량 증가.
  ★3에서 "용병을 모을수록 경제+물량이 좋아진다" — 굿스터프의 정체성을 대변.
  용병 유닛 추가는 UNIT_ADDED emit → 체인에 합류.

[고대의 각성] T4 — 인내 (장기)

  4라운드 인내의 대보상. 필드 위 모든 카드에 유닛+방어막 동시 적용.
  발동 시 모든 아군 카드에서 UNIT_ADDED 방출 → 체인에 합류.
  ★ 합성 시 카운터 리셋 → 한 런에서 2번 발동 가능 (★1 후 ★2로).
  ★3에서 매 라운드 전체 유닛+방어막 → 후반 굿스터프의 캡스톤.
  R7 획득→R11 발동, R9 획득→R13 발동. R10까지 찾으면 유효.

[폐품 상회] T2 — 경제 (리롤 수급)

  > "이번 라운드 한정": 사용하지 않은 저축 리롤은 다음 라운드 시작 시 소멸.
  > 모든 무료 리롤은 이 규칙을 따른다 (글로벌 기획 원칙).
  > 리롤 소비 우선순위: (1) 🎲 도박꾼 확률 성공 → (2) pending_free_rerolls → (3) 골드.

  드루이드 ★2/★3 합성으로 누적된 유닛 상한 관리 도구.
  포식종 물량형 빌드에서 잉여 유충을 리롤로 환전하는 진화형 경제.
  군대/스팀펑크는 유닛 수가 적어 효과가 미미 → 4테마 정체성 보존.
  UNIT_ADDED/ENHANCED 이벤트 방출 없음 (조용한 제거).
```

## 체인 흐름

```
Layer 1 이벤트 소스 (라운드 시작):
  대지의 울림 (T1) → 인접 유닛 추가          → UNIT_ADDED
  야생의 맥동 (T1) → 자신 강화              → ENHANCED
  유적의 공명 (T2) → 유닛 추가 + 강화        → UNIT_ADDED + ENHANCED
  고대의 잔해 (T2) → 인접 유닛 추가 (2R+)    → UNIT_ADDED

UNIT_ADDED 반응:
  떠돌이 무리 (T2) → 해당 카드 강화           → ENHANCED    [브릿지: 범주 교차]
  마력 결정 (T2)   → 인접 카드에도 유닛 추가   → UNIT_ADDED  [증폭기: 같은 범주 확산]

ENHANCED 반응:
  돌연변이 적응 (T3) → 자신 유닛 추가         → UNIT_ADDED  [브릿지: 범주 교차]
  고대 촉매 (T3)    → 인접 카드에도 강화      → ENHANCED    [증폭기: 같은 범주 확산]

브릿지 vs 증폭기:
  브릿지 = 범주 교차 (UNIT→ENHANCED, ENHANCED→UNIT) → 다른 채널 활성화
  증폭기 = 같은 범주 확산 (UNIT→더 많은 UNIT, ENHANCED→더 많은 ENHANCED) → 같은 채널 증폭

순환 루프 (★1 기준):
  시작기(UA) → 떠돌이(EN) + 마력결정(UA 확산) → 돌연변이(UA) + 고대촉매(EN 확산) → ...
  브릿지/증폭기 각 최대 횟수 제한 → 체인 길이는 등급과 배치에 따라 다름 (수치는 YAML 참조)
  ★2에서 증폭기 수치 증가, ★3에서 최대 횟수 증가 → 체인 길이 확장

전투 전환:
  야생의 힘 (T2): 전투 ATK 버프
  키메라의 울부짖음 (T3): 패배 → 영구 ATK/HP 성장, ★3 인접 유닛+HP재생

경제:
  방랑 상인 (T1): 패배 → 골드 (초반 안전망, ★3에서 물량 전환)
  고대의 잔해 (T2): 체류 → 골드+유닛 (중기 투자, ★3 인접 유닛 생산)
  정령의 축복 (T3): 합성 → 테라진+유닛 (합성 가치 물량 증폭)
  차원 행상인 (T4): 테마 다양성 → 골드+테라진+유닛+강화 (★3에서 굿스터프 캡스톤)

인내:
  고대의 잔해 (T2): 2R 체류 → 중기 보상
  고대의 각성 (T4): 4R 체류 → 대폭발 (★3에서 매 라운드 전체 엔진, 판도 전환)
```
