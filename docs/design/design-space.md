# 카드 설계 공간 (Design Space)

> **공유 설계 공간**: 이 트리거/효과 풀은 카드뿐 아니라 업그레이드/부적/커맨더의 재료이기도 하다.
> - 카드 = 트리거 1개 + 효과 1~2개 + 대상 조합
> - 업그레이드 = 패시브 효과 또는 조건부 트리거 + 효과 (카드에 부착)
> - 부적 = 런 시작 시 선택하는 글로벌 패시브/트리거
> - 커맨더 = 런 전체 규칙을 변경하는 강력한 패시브
> "이게 되네?" 하는 발라트로식 창의적 효과를 지향.

## 트리거 조건 풀

**성장 페이즈 — 이벤트 반응**
```
⚡충전 시
유닛 추가 시 (Layer 1) / 제조·번식·부화·징집 시 (Layer 2)
강화 시 (Layer 1) / 개량·성장·변태·훈련 시 (Layer 2)
⬆합성 시
```

**성장 페이즈 — 시점**
```
라운드 시작 시
카드 구매 시
카드 판매 시
업그레이드 부착 시
리롤 시
```

**전투 — 이벤트 반응**
```
⚔공격 실행 시
💀유닛 사망 시 (아군/적군)
🛡버프 적용 시
```

**전투 — 시점**
```
전투 시작 시
전투 종료 시
전투 승리 시
전투 패배 시
```

**조건부 (상태 기반)**
```
보드 상태:
  자신 유닛이 N기 이하/이상/정확히 N일 때
  필드에 같은 카드 태그가 N장 이상일 때
  같은 카드를 2장 이상 필드에 포함시켰을 때
  빈 필드 칸이 N개 이상일 때
  필드가 가득 찼을 때 (8장)
  인접 카드가 발동했을 때
  양쪽 인접 카드가 같은 테마일 때

벤치/보관:
  벤치에 보관 중인 카드가 N장 이상/이하/정확히 N일 때
  벤치가 가득 찼을 때 / 비었을 때

경제/진행 상태:
  골드가 정확히 0일 때 / N 이상 보유 시
  보유 골드가 10 이상인 채로 전투 진입 시
  테라진 N 이상 보유 시
  연승/연패 N회 시
  현재 라운드가 N 이상일 때
  리롤할 때마다
  이번 런에서 판매한 카드 수가 N장 이상일 때

상점/진행:
  상점 레벨을 높였을 때
  상점에서 특정 티어 카드가 등장했을 때

카드 상태:
  이 카드가 ★N일 때 (★2/★3 전용 추가 효과)
  업그레이드가 N개 이상 부착 시
  자신 유닛 ATK 합이 N 이상일 때
  이 카드에 부착된 업그레이드가 특정 등급일 때

시간/누적:
  이 카드가 필드에 N라운드 이상 머물러 있을 때
  이 카드 위에서 체인이 N회 이상 발동했을 때 (런 누적)
  이 카드가 처치한 적 유닛 수가 N기 이상일 때 (런 누적)

일회성 임계점 (1회만 발동):
  이 카드의 유닛이 처음으로 N기를 넘었을 때
  ★2/★3가 된 순간
  이 카드의 ATK 합이 처음으로 N을 넘었을 때
```

## 강화 수치 체계

```
강화 효과의 수치 표현 3종:

기본 (대부분):  "개량(ATK +30%)" = 원래(기본) 스탯 × N% 만큼 추가
               → 기본 ATK 2 유닛: +0.6 / 기본 ATK 4 유닛: +1.2
               → 고티어 유닛이 자연스럽게 더 많이 성장
               → 밸런스 조정이 용이 (% 하나로 전체 스케일 제어)

곱연산 (명시):  "개량(ATK ×1.5, 곱연산)" = 현재(누적된) 스탯 × N배
               → Layer 1, Layer 2 위에 추가로 곱해짐 (Layer 3). 매우 강력.
               → ★3 효과, 보스 보상, 임계점 보상 등 드문 상황에만 사용.
               → 반드시 "곱연산" 표기.
               → 참고: 성장 누적(L1) × 업그레이드(L2)의 레이어 간 곱연산은 시스템 구조이며
                 여기서 말하는 "드문 곱연산"과 별개.

고정값 (드문):  "개량(ATK +1)" = 고정 수치 직접 추가
               → 기본 모델이 아님. 특수 카드/업그레이드에서만.
               → 저티어 유닛에 상대적으로 유리 → 의도적 사용처가 있을 때만.

설계 근거:
  - % 기반이면 R1에서 강화한 T1 유닛(base 1)이 R12의 T5 유닛(base 4)을
    절대 압도하지 못함 → 고티어 카드의 가치 유지
  - 곱연산은 "사기" 순간에만 → ★3, 보스 보상의 체감 파워 보장
```

## 효과의 지속 유형

```
효과가 대상에 적용되는 방식은 4종:

카드 귀속 누적 (강화 효과의 기본):
  "이 카드의 유닛 전체 개량(ATK +5%)" / "성장(ATK +3%)"
  → 카드의 성장 누적 modifier에 추가. 모든 유닛(기존+신규)에 동일 적용.
  → 카드 트리거/효과에서 발생하는 강화(개량/성장/변태/훈련)는 모두 카드 귀속.
  → 유닛이 카드를 떠나면 기본 스탯으로 복귀.
  → 상세: architecture.md "카드 레벨 스탯 modifier" 참조.

영구 규칙 (카드에 부착):
  "이 카드의 유닛은 ATK 2배"
  → 카드에 붙는 영구 규칙. 이후 추가된 유닛에도 적용.
  → 업그레이드, 보스 보상(🔄영구규칙)의 영역.

일시적 (전투 중):
  "이번 전투 동안 #무기 유닛 ATK +15%"
  → 🛡버프. 전투 종료 시 사라짐.

1회성 (비강화 카드 효과):
  "자신 유닛 전체에 방어막(기본 HP의 20%)"
  → 발동 시점에 존재하는 유닛에 즉시 적용. 이후 추가 유닛에는 미적용.
  → 방어막, 유닛 추가, 자원 획득 등 비강화 효과에 해당.

구분 원칙:
  카드 효과 (강화) → 카드 귀속 누적 (modifier 누적, 전 유닛 적용)
  카드 효과 (비강화) → 1회성 (발동 시 유닛에 적용, 끝)
  업그레이드 → 영구 규칙 (카드에 부착, 이후 유닛에도 계속 적용)
  보스 보상 🔄 → 영구 규칙 (런 전체 규칙 변경)
  전투 버프 → 일시적 (이번 전투만)
```

## 자기 반응 규칙

```
자기 효과로 생성된 이벤트에 자기가 반응하는 것은 가능하지만, 자기 루프를 방지하기 위해
카드 설계 시 트리거 조건에 자기를 제외하도록 설계한다.

  ❌ "제조 시 → 자신 유닛 1기 제조" → 자기 루프 발생
  ✅ "다른 카드가 제조 시 → 자신 유닛 1기 제조" → 자기 제외, 안전
  ✅ "인접 카드가 제조 시 → ..." → 위치 조건으로 자연스럽게 자기 제외
  ✅ "제조 시 → ⚡충전" → 제조와 다른 이벤트를 생성하므로 자기 루프 없음
```

## 성장의 주 축 분리

```
카드 효과의 강화 (개량/성장/변태/훈련):
  → 카드 귀속. 유닛 태그로 대상 지정. 카드의 성장 누적 modifier에 추가.
  → "아군 카드 내 #기어 유닛 전체 개량(ATK +5%)" → 대상 카드에 [#기어 ATK +5%] 누적
  → 드묾 (테마 10장 중 2~3장만). 소폭 (5~8%).
  → 모든 유닛(기존+신규)에 동일 적용. 유닛이 카드를 떠나면 소실.

업그레이드 (테라진 구매):
  → 카드 대상. 카드에 영구 부착. 5슬롯.
  → "이 카드 ATK +10%" / "#기계 한정: ATK +15%"
  → 주력 스탯 성장 경로. 분리/이동 가능.

두 소스의 관계:
  레이어 간 곱연산. 둘 다 투자할수록 서로의 가치가 올라감 (보완재).
  유닛 실제 스탯 = 기본 × (1 + 성장 누적%) × (1 + 업그레이드%) × 곱연산

역할:
  카드 효과 = "무엇이 일어나는가" (제조, 충전, 방어막, 창의적 효과 + 드문 소폭 강화)
  업그레이드 = "얼마나 강한가" (스탯 성장의 주 축)
  성장 누적 = "얼마나 오래/많이 강화받았는가" (카드 효과에 의한 부가 성장)
```

## 효과 풀

**성장 — 기본**
```
유닛 추가:     대상 카드에 유닛 N기 추가 (제조/번식/부화/징집)
유닛 강화:     대상 유닛 스탯 영구 변경 — 기본 % 사용 (개량/성장/변태/훈련)
⚡충전:       대상 카드 발동 상한 +1
추가 발동:     대상 카드를 1회 추가 발동시킨다
```

**성장 — 발라트로식 (규칙 변경/스케일링)**
```
강화 증폭:     "이 카드 위 유닛 수만큼 강화량 추가" (스케일링)
효과 복사:     "인접 카드의 효과를 복사하여 1회 발동"
조건부 폭발:   "유닛 30기 이상이면 전체 ATK ×2" (임계점)
연쇄 보너스:   "체인이 N링크 이상이면 마지막 효과 반복"
변환:          "유닛 N기를 제거 → 남은 유닛 전체 강화" (물량↔품질)
흡수:          "이 카드가 판매되면 유닛 스탯이 인접 카드에 분배"
역전:          "전투 패배 시 사망한 아군 유닛 수만큼 다음 라운드 유닛 추가"
누적:          "매 라운드 이 카드 유닛 ATK+1 (영구 누적, 체인 밖)"
이자형:        "이 카드 위 유닛 수 5당 추가 유닛 1기 (이자처럼)"
도박:          "50% 확률로 강화 2배, 50% 확률로 강화 없음"
기생:          "인접 카드에서 유닛 1기 빼앗아 자신에게 추가"
확률 증강:     "모든 확률형 효과의 확률을 N배 증강"
전용 유닛:     "카드 풀에 없는 전용 유닛을 얻는다" (카드 고유 특수 유닛)
스탯 교환:     "유닛의 ATK와 HP를 교환"
연금술:        "유닛 N기를 골드/테라진으로 변환"
공명:          "같은 카드를 2장 이상 필드에 놓으면 양쪽 모두 효과 강화"
```

**성장 — 대가형 (부정적 효과 + 강력한 보상)**
```
헌신:          "이 카드의 유닛은 강화 불가. 대신 매 라운드 인접 카드에 유닛 N기 추가"
징집령:        "골드 수입 -N. 대신 매 라운드 모든 카드에 유닛 1기"
비전투원:      "이 카드는 전투 불참. 대신 성장 체인 효과 모두 2배"
소모품:        "N라운드 후 이 카드 자동 파괴. 파괴 시 유닛 전부 인접 카드에 분배"
카운트다운:    "매 라운드 유닛 1기를 잃는다. 0기 도달 시 인접 카드 전체 ATK ×3"
시한폭탄:      "매 라운드 카운터 +1. N 도달 시 전체 아군 유닛 수 2배 (1회)"
```

**성장 — 보드 레이아웃 활용**
```
빈칸 보상:     "빈 필드 칸 1개당 전체 ATK +N"
풀보드 보상:   "필드가 가득 차면 전체 유닛 HP ×1.5"
위치값:        "배치 위치(1~8) 숫자만큼 발동 상한 추가"
```

**성장 — 복사/참조**
```
트리거 복사:   "인접 카드의 트리거 조건을 복사한다 (자신 효과로 발동)"
카멜레온:      "이 카드의 유닛은 인접 카드의 유닛 태그를 전부 갖는다"
미러:          "필드에서 가장 유닛이 많은 카드의 유닛 수를 복사 (전투 시)"
효과 복사:     "인접 카드의 업그레이드 효과를 복사 (부착 없이 동일 적용)"
```

**성장 — ★합성 변형**
```
속성합성:      "★합성에 2장만 필요. 대신 ★3 불가"
흡수합성:      "★합성 시 촉매 카드의 유닛이 전부 이 카드에 합류"
촉매 희생:     "이 카드를 ★합성의 촉매로 사용하면 대상 카드에 에픽 업그레이드"
```

**경제 — 전투 연동**
```
전투 수입:     "전투에서 적 유닛 처치 시 1골드 획득"
금고:          "보유 골드만큼 전투 시작 시 전체 ATK 증가 (소비 안 함)"
폐기 수익:     "이 카드를 판매하면 상점의 모든 카드가 1티어 상승"
```

**전투**
```
🛡방어막:      유닛에 일시적 방어 부여
버프:          ATK/HP/AS/Range/MS 일시 증가
디버프:        적 유닛 약화 (느려짐, ATK 감소)
관통 부여:     이번 전투 DEF 무시
폭발:         "아군 유닛 사망 시 ATK의 N% 주변 적 피해"
흡혈:         "전투 중 유닛이 적을 처치하면 HP 회복"
위협:         "전투 시작 시 유닛 수만큼 적 전체에 1 데미지"
불사:         "아군 유닛 첫 사망 시 HP 1로 생존 (유닛당 1회)"
```

**경제/메타**
```
골드/테라진 획득
리롤 무료 N회
상점 카드 1장 무료 획득
업그레이드 랜덤 부착 (인접/지정 카드에)
업그레이드 이동:  "대상 카드의 업그레이드를 다른 카드로 옮긴다"
카드 복사 (상점에 같은 카드 추가 등장)
할인:          "다음 구매 N골드 할인"
정찰:          "다음 라운드 적 구성 미리보기"
부적 획득:     "이번 런 동안 랜덤 부적 1개를 얻는다"
상점 조작:     "다음 상점에 특정 티어 카드 1장 보장"
```

**구조**
```
카드 태그 추가:    "특정 카드에 특정 태그를 추가한다"
유닛 태그 부여:    임시(이번 라운드) 또는 영구
발동 상한 영구 +1 (⚡충전과 다름 — 영구)
배치 변경:        "자신과 인접 카드의 위치를 교환" / "배치 순서를 재배열"
필드/벤치 조작:    "벤치 카드 1장을 즉시 필드에 배치"
부적 획득:        "이번 런 동안 랜덤 부적 1개를 얻는다"
```

## 대상 지정 풀

```
자신:         이 카드 / 이 카드의 유닛
인접:         왼쪽 / 오른쪽 / 양쪽 인접 카드
유닛 태그:    #유닛태그 유닛 전체 (아군 카드 내)
카드 태그:    #카드태그 카드 전체 (필드 위)
전체 아군:    필드 위 모든 아군 카드/유닛
랜덤:         랜덤 아군 카드 1장 / 랜덤 유닛 N기
이벤트 대상:  "강화된 카드", "추가된 유닛" 등 이벤트 원인 참조
조건부:       "ATK가 가장 높은 유닛", "유닛 수가 가장 적은 카드"
```

## 설계 원칙 (발라트로 교훈)

```
1. 읽으면 3초 안에 이해 — 효과는 한 문장. 복잡한 건 ★3에서.
2. 단독으로도 작동, 조합하면 "사기" — 혼자서도 가치 있지만 특정 조합에서 폭발.
3. 빌드 판단을 바꾸게 만든다 — "이 카드가 있으면 물량이 아니라 정예로 가야 해."
4. 스케일링은 보상 — "매 라운드 +1"은 느리지만 R15에서 최강. 인내의 보상.
5. 리스크/리워드 — 도박, 변환, 기생 등 위험을 감수하면 더 강한 효과.
```

## 시스템 간 효과 강도

```
같은 효과라도 부여 시스템에 따라 강도가 다르다:

  커맨더:    가장 강력 (런 전체 규칙 변경)
             "모든 확률형 효과의 확률 1.5배"
             "매 라운드 리롤 1회 무료"

  부적:      중간 (런 시작 시 글로벌 패시브)
             "체인 10+ 달성 시 마지막 효과 반복"
             "전투 패배 시 아군 유닛 수만큼 다음 라운드 유닛 추가"

  보스 보상:  1회 선택, 강력 (런 중 규칙 추가)
             "강화 시 추가 유닛 1기"
             "이번 런 동안 랜덤 부적 획득"

  카드 효과:  가장 세밀 (트리거 조건 필요, 발동 상한 있음)
             "⚡충전 시 → #기어 유닛 강화"
             "전투 패배 시 → 자신 유닛 2기 추가"

  업그레이드: 카드에 부착 (패시브 또는 조건부)
             "#기계 한정: ATK +3"
             "이 카드의 유닛 사망 시 인접 유닛 버프"

→ 같은 "유닛 추가" 효과:
   커맨더:  "매 라운드 시작 시 모든 카드에 유닛 1기"
   부적:    "전투 승리 시 랜덤 카드에 유닛 1기"
   카드:    "⚡충전 시 → 자신 유닛 1기 (발동 2회)"
   업글:    "이 카드에 매 라운드 유닛 1기 (패시브)"
```
