# 포식종 카드 풀 (10장)

> 수치 상세: `data/cards/predator.yaml` 참조 (단일 진실 소스).
> 이 문서는 설계 의도와 빌드 전략을 기술합니다.

> 유닛 풀: units-predator.md 참조.
> 핵심 판타지: "끝없이 밀려오는 벌레 떼" — 유충이 쏟아지고, 일부가 상위 포식자로 진화.
> 고유 엔진: 부화(물량 생산) + 변태(양→질 변환). 유닛=자원.

## 부화/변태 시스템

```
부화 = 칼날 유충 N기를 대상 카드에 추가 (Layer1=UNIT_ADDED, Layer2=HATCH)
변태 N기 = 이 카드의 최약 유닛 N기 소모 → 최강 유닛 1기 추가 (Layer1=ENHANCED, Layer2=METAMORPHOSIS)

부화 대상: 항상 칼날 유충 (카드 composition 무관)
변태 소모: 항상 카드 내 가장 약한 유닛 스택에서 차감
변태 생성: 항상 카드 composition 내 가장 강한 유닛 타입에 추가

변태 조건: 소모할 유닛이 충분할 때만 발동
변태 상한: 카드별 max_activations으로 제한 (라운드당)
변태 표기: "변태 N기" = N기 소모 → 1기 생성. 어떤 유닛인지는 카드 구성이 결정.

핵심 결정: 부화로 유충을 쌓아 물량으로 밀 것인가, 변태로 소모하여 상위 유닛을 키울 것인가.
```

## 개요

| # | 이름 | 티어 | 부화/변태 | 역할 | 체인 위치 |
|---|------|------|----------|------|----------|
| 1 | 유충 둥지 | T1 | 부화 2+인접1 | 체인 시작+공급 | RS starter |
| 2 | 독 양식장 | T1 | 부화 1 | 물량형 경제 (유닛수 비례 골드) | RS + POST |
| 3 | 탈피의 방 | T2 | 변태 3기 | 변태 엔진 | ON_EVENT (HATCH) |
| 4 | 군체 감각 | T2 | — | 유닛수 비례 전투 버프 | BS |
| 5 | 변태 수확 | T2 | 부화 1 | 진화형 경제 (변태→테라진) | ON_EVENT (META) |
| 6 | 여왕의 산란 | T3 | 부화 3+인접1 | 대규모 부화 | RS |
| 7 | 적응 갑각 | T3 | 변태→인접부화1 | 변태 반응 강화+중계 | ON_EVENT (META) |
| 8 | 기생 진화 | T4 | 전투→부화 | 전투 피드백 | POST |
| 9 | 포식자의 사냥 | T4 | 변태 2기 | 상위 변태 | ON_EVENT (META) |
| 10 | 군체 초월 | T5 | 부화 2기 | 사망 시 광폭화 | RS + 전투 |

## 상세

```
[유충 둥지] T1
구성: 칼날 유충 ×3, 갑충 일꾼 ×1
카드 태그: #포식종 #부화
★1: 라운드 시작: 이 카드에 부화. 오른쪽 인접 카드에 부화.
★2: 라운드 시작: 이 카드에 부화 수량 증가. 오른쪽 인접 카드에 부화 수량 증가.
★3: 라운드 시작: 이 카드에 부화. 양쪽 인접 카드에 부화 (범위 확대). 부화한 유닛 ATK 성장.

  포식종의 체인 시작. ★1부터 인접 공급.
  오른쪽에 탈피의 방 배치 → R1부터 변태 체인 가동.
  체인: [유충 둥지] 부화(자기+인접) → [탈피의 방] 변태 반응


[독 양식장] T1
구성: 독침 사수 ×1, 산성 거미 ×1
카드 태그: #포식종 #경제
★1: 라운드 시작: 이 카드에 부화.
    전투 종료: 이 카드 유닛 수 비례 골드 획득 (상한 존재). 전투 패배 시 골드 절반.
★2: 라운드 시작: 이 카드에 부화 유지.
    전투 종료: 유닛 수 비례 골드 (상한 증가). 패배 시에도 전액 지급.
★3: 라운드 시작: 이 카드에 부화 수량 증가.
    전투 종료: 유닛 수 비례 골드 (상한 추가 증가) + 테라진. 패배 시에도 전액 지급.

  물량형 경제 카드. T1이라 초반부터 채용 가능.
  부화/R → 유충 둥지 없이도 최소한의 부화 시작 가능 (둥지 필수성 해소).
  유닛 수 비례 수입 → 물량을 쌓을수록 보상 (수치는 YAML 참조).


[탈피의 방] T2
구성: 칼날 유충 ×2, 변태 수호충 ×1
카드 태그: #포식종 #변태
★1: [반응] 다른 카드의 부화 발동 시: 이 카드에서 변태 (소모 임계 존재)
★2: [반응] 다른 카드의 부화 발동 시: 이 카드에서 변태 (소모 임계 감소, 효율 향상) / 최대 횟수 증가
★3: [반응] 다른 카드의 부화 발동 시: 이 카드에서 변태 / 최대 횟수 유지. 변태 시 이 카드 ATK 영구 성장

  변태 엔진. 유충 둥지의 인접 부화로 R1부터 유충 공급 가능.
  R1: 둥지 인접 부화 → 탈피의 방에 유충 누적 → 변태 발동.
  ★1 CP 보존, ★2 소모 임계 감소로 CP 효율 향상 (수치는 YAML 참조).
  METAMORPHOSIS 이벤트 emit → 적응 갑각, 포식자의 사냥 반응.


[군체 감각] T2
구성: 산성 거미 ×2, 칼날 유충 ×1
카드 태그: #포식종 #전투
★1: 전투 시작: 필드 위 모든 포식종 유닛에 이 카드 유닛 수 비례 ATK 전투 버프
★2: 전투 시작: 유닛 수 비례 ATK 버프 강화. 이 카드 유닛 임계 이상이면 적 전체 AS 감소 추가
★3: 전투 시작: 버프 비례 계수 강화. 임계 이상이면 적 전체 AS+ATK 감소 (상한 존재)

  물량 보상 카드. 유충을 많이 모을수록 전투력 버프 (수치는 YAML 참조).
  T2라 초반 급하지 않고, 유닛이 쌓인 뒤 투입해도 됨.
  배치: 유충 둥지 인접으로 부화 받아 자기 유닛 축적 → 버프 극대화.


[변태 수확] T2  ← 진화형 경제
구성: 산성 거미 ×1, 독침 사수 ×1
카드 태그: #포식종 #경제
★1: [반응] 다른 카드의 변태 발동 시: 테라진 획득. 이 카드에 부화
★2: [반응] 다른 카드의 변태 발동 시: 테라진 획득. 이 카드에 부화 + 변태한 유닛 ATK 영구 성장 / 최대 횟수 증가
★3: [반응] 다른 카드의 변태 발동 시: 테라진 획득. 이 카드에 부화 수량 증가 + 필드 위 모든 포식종 카드 ATK 영구 성장 / 최대 횟수 유지

  진화형 경제 카드. 변태가 발동할수록 테라진 수입 → 업그레이드 자금.
  ★2에서 최대 횟수 증가 → 테라진 수입 증가 (수치는 YAML 참조).
  독 양식장(물량형 골드) vs 변태 수확(진화형 테라진) = 경제 카드 분기.
  부화도 함 → 유충 둥지 대안 보강. 탈피의 방 없이는 경제 미작동.


[여왕의 산란] T3
구성: 여왕충 ×1, 갑충 일꾼 ×2, 칼날 유충 ×2
카드 태그: #포식종 #부화
★1: 라운드 시작: 이 카드에 부화 대량. 오른쪽 인접 카드에 부화.
★2: 라운드 시작: 이 카드에 부화 수량 증가. 양쪽 인접 카드에 부화 (범위 확대+수량 증가).
★3: 라운드 시작: 이 카드에 부화 추가 증가. 양쪽 인접 카드에 부화 수량 증가. 이 카드 #여왕 유닛 HP 영구 성장

  대규모 부화. 유충 둥지의 상위 호환.
  인접 대량 배분으로 탈피의 방/군체 감각 등에 유충 공급.
  체인: [여왕] 부화 → [탈피의 방] 변태 반응 (인접 배치 시)


[적응 갑각] T3
구성: 뿔 돌진자 ×1, 돌격 전사충 ×1, 갑충 일꾼 ×1
카드 태그: #포식종 #강화
★1: [반응] 다른 카드의 변태 발동 시: #갑충 유닛이 있는 필드 위 모든 포식종 카드 ATK+HP 영구 성장. 오른쪽 인접 카드에 부화
★2: [반응] 다른 카드의 변태 발동 시: #갑충 카드 ATK+HP 성장 강화. 인접 부화 유지
★3: [반응] 다른 카드의 변태 발동 시: 조건 제거 → 필드 위 모든 포식종 카드 ATK+HP 성장. 양쪽 인접 부화. 변태 원본 카드에 방어막 / 최대 횟수 증가

  변태 반응 강화 + 부화 중계. 변태를 받아 강화하면서 인접에 유충을 공급.
  진화형 배치 [둥지]→[탈피]→[갑각]→[포식자]: 갑각이 포식자에 부화 공급.
  #갑충 = 일꾼, 전사충, 돌진자, 수호충 (4종) → 대부분의 포식종 유닛이 혜택.
  인접 부화 = HATCH 이벤트 emit → 탈피의 방이 다시 반응 가능 (추가 체인).


[기생 진화] T4
구성: 비행 포식자 ×2, 독침 사수 ×1
카드 태그: #포식종 #전투
★1: 전투 종료: 이번 전투에서 생존한 이 카드 유닛 수 비례 부화 (상한 존재). 전투 승리 시 추가로 변태
★2: 전투 종료: 생존 유닛 비례 부화 (상한 증가). 전투 승리 시 변태 + 생존 유닛 HP 회복
★3: 전투 종료: 생존 유닛 비례 부화. 승패 무관 변태 + HP 회복. 변태된 유닛에 방어막

  > 강화 과다 해소: ★2/★3의 ATK% 영구 성장 → HP 회복으로 대체.
  > "전투에서 살아남으면 회복하고 더 많이 낳는다" — 전투 생존 피드백.

  전투→생존 피드백 루프. 부화 상한으로 선형 증가 보장 (지수적 증가 방지) (수치는 YAML 참조).
  비행 포식자(빠른 AS) = 빠른 공격으로 적 처치 유리 → 생존율 높음.
  ★3에서는 패배해도 변태 + HP 회복 + 방어막 → 역전 가능성.


[포식자의 사냥] T4
구성: 거대 포식자 ×1, 뿔 돌진자 ×1
카드 태그: #포식종 #포식
★1: [반응] 다른 카드의 변태 발동 시: 이 카드에서 변태. 이 카드 유닛 ≤5기이면 ATK 전투 버프
★2: [반응] 다른 카드의 변태 발동 시: 이 카드에서 변태. 이 카드 유닛 ≤5기이면 ATK 버프 강화 / 최대 횟수 증가
★3: [반응] 다른 카드의 변태 발동 시: 이 카드에서 변태 (소모 감소). 이 카드 유닛 ≤5기이면 ATK 곱연산 + 적 처치 시 HP 회복

  먹이사슬 최상위. 변태 체인의 정점.
  탈피의 방의 변태 이벤트에 반응 → 이중 변태 체인.
  소수정예 조건(≤5기) = 포식자가 적을수록 더 강해지는 정점 보상.
  ★3에서 변태 1기(효율 극대화) + ATK ×2 + HP 회복 → 먹이사슬 왕.


[군체 초월] T5
구성: 거대 포식자 ×1, 여왕충 ×1, 변태 수호충 ×1
카드 태그: #포식종 #군체
★1: 라운드 시작: 이 카드에 부화. 필드 위 모든 포식종 카드에 부화 분배.
    [지속] 전투 중 포식종 유닛 사망 시 생존 포식종 유닛 전체 ATK 누적 버프(전투 한정). 이 카드 유닛이 적 처치 시 HP 회복.
★2: 라운드 시작: 이 카드에 부화 수량 증가. 필드 위 모든 포식종 카드에 부화 분배 수량 증가.
    [지속] 사망→생존자 ATK 누적 버프 강화. HP 회복 증가.
★3: 라운드 시작: 이 카드에 부화 + 전체 분배 유지. 이 카드에서 자동 변태 추가.
    [지속] 사망 버프 유지. HP 회복 유지. 변태 발생 시 이 카드 ATK 영구 성장 추가.

  > 강화 과다 해소: 매 라운드 전체 ATK +5% 영구 누적 → 전체 부화로 대체.
  > "캡스톤이 물량 공급 = 물량 테마 정체성 극대화. ATK 성장은 사망 광폭화에 집중."

  포식종 캡스톤. 2중 역할:
  1) 물량 공급 — 매 라운드 전체 부화로 유충 지속 공급. 물량형/진화형 모두 수혜.
     R10 합류 시 남은 라운드 동안 카드당 부화 누적 공급. 진화형은 즉시 변태 연료.
  2) 전투 동적 스케일링 — "약한 자가 쓰러지면 강한 자가 더 강해진다."
     물량형: 유충 대량 사망 → 생존자 광폭화 (유충이 연료)
     진화형: 소수 포식자가 유충 희생으로 ATK 폭증 (자연선택)
```

## ★2 합성 유인 검증

```
둥지:   ★1×2 동등 효과 + 슬롯 절약.                  ✓ 동등+슬롯
양식:   골드 상한 증가. 패배 페널티 제거.               ✓ 수치
탈피:   소모 임계 감소. 변태 효율 향상.                 ✓ 수치
감각:   버프 비율 강화.                               ✓ 수치
수확:   최대 횟수 증가. ATK 성장 추가.                  ✓ 수치+효과
여왕:   부화 수량 증가 + 인접 범위 확대 및 수량 증가.     ✓ 수치
갑각:   성장 퍼센트 증가 + 인접 부화 유지.              ✓ 수치
기생:   부화 수량 증가(상한 증가), HP 회복 추가.         ✓ 수치+효과
포식:   최대 횟수 증가. ATK 버프 강화.                  ✓ 수치
초월:   자기 부화+전체 분배 수량 증가, 사망 ATK 버프 강화. ✓ 수치+물량
```

## 체인 흐름도

```
부화 생성:   유충 둥지(자기+인접), 독 양식장(자기), 여왕의 산란(자기+인접),
             군체 초월(자기+전체/R), 기생 진화(전투 후), 변태 수확(변태 시)
             ↓ HATCH 이벤트
변태 반응:   탈피의 방(소모→변태), 포식자의 사냥(소모→변태)
             ↓ METAMORPHOSIS 이벤트
강화+중계:   적응 갑각(#갑충 카드 ATK+HP 성장 + 인접 부화)
             변태 수확(테라진 + 부화)
             ↓ HATCH 이벤트 (2차 체인)
             탈피의 방 추가 반응 가능

물량 지원:   군체 감각(유닛수 비례 ATK 전투버프)
경제:        독 양식장(물량형 골드) / 변태 수확(진화형 테라진)
피드백:      기생 진화(전투 생존→부화→변태)
캡스톤:      군체 초월(전투 중 사망→생존자 광폭화)
```

## 배치 권장

```
물량형:   [유충 둥지]→[독 양식장]→[군체 감각]→[여왕의 산란]→[기생 진화]→[군체 초월]
          둥지가 양식장에 인접 부화, 여왕이 기생진화에 인접 부화.
          군체 감각은 자체 유닛 축적 위해 부화 인접.

진화형:   [유충 둥지]→[탈피의 방]→[변태 수확]→[적응 갑각]→[포식자의 사냥]→[군체 초월]
          둥지→탈피(인접 부화→변태), 탈피 변태→수확(테라진)+갑각(강화)→포식자(이중 변태).
          변태 수확이 테라진 공급 → 업그레이드 자금.

하이브리드: [유충 둥지]→[탈피의 방]→[여왕의 산란]→[적응 갑각]→[기생 진화]→[군체 초월]
            둥지→탈피(인접 변태), 여왕(대량 부화+인접 공급).
```
