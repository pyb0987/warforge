# 포식종 카드 풀 (10장)

> 유닛 풀: units-predator.md 참조.
> 핵심 판타지: "끝없이 밀려오는 벌레 떼" — 유충이 쏟아지고, 일부가 상위 포식자로 진화.
> 고유 엔진: 부화(물량 생산) + 변태(양→질 변환). 유닛=자원.

## 부화/변태 시스템

```
부화 = 칼날 유충 N기를 대상 카드에 추가 (Layer1=UNIT_ADDED, Layer2=HATCH)
변태 N기 = 이 카드의 최약 유닛 N기 소모 → 최강 유닛 1기 추가 (Layer1=ENHANCED, Layer2=METAMORPHOSIS)

부화 대상: 항상 칼날 유충 (카드 composition 무관)
변태 소모: 항상 카드 내 가장 약한 유닛 스택에서 차감
변태 생성: 항상 카드 composition 내 가장 강한 유닛 타입에 추가

변태 조건: 소모할 유닛이 충분할 때만 발동
변태 상한: 카드별 max_activations으로 제한 (라운드당)
변태 표기: "변태 N기" = N기 소모 → 1기 생성. 어떤 유닛인지는 카드 구성이 결정.

핵심 결정: 부화로 유충을 쌓아 물량으로 밀 것인가, 변태로 소모하여 상위 유닛을 키울 것인가.
```

## 개요

| # | 이름 | 티어 | 부화/변태 | 역할 | 체인 위치 |
|---|------|------|----------|------|----------|
| 1 | 유충 둥지 | T1 | 부화 2+인접1 | 체인 시작+공급 | RS starter |
| 2 | 독 양식장 | T1 | 부화 1 | 물량형 경제 (floor(유닛수/5) 골드) | RS + POST |
| 3 | 탈피의 방 | T2 | 변태 3기 | 변태 엔진 | ON_EVENT (HATCH) |
| 4 | 군체 감각 | T2 | — | 유닛수 비례 전투 버프 | BS |
| 5 | 변태 수확 | T2 | 부화 1 | 진화형 경제 (변태→테라진) | ON_EVENT (META) |
| 6 | 여왕의 산란 | T3 | 부화 3+인접1 | 대규모 부화 | RS |
| 7 | 적응 갑각 | T3 | 변태→인접부화1 | 변태 반응 강화+중계 | ON_EVENT (META) |
| 8 | 기생 진화 | T4 | 전투→부화 | 전투 피드백 | POST |
| 9 | 포식자의 사냥 | T4 | 변태 2기 | 상위 변태 | ON_EVENT (META) |
| 10 | 군체 초월 | T5 | 부화 2기 | 사망 시 광폭화 | RS + 전투 |

## 상세

```
[유충 둥지] T1
구성: 칼날 유충 ×3, 갑충 일꾼 ×1               (4유닛, ATK 5, HP 50)
카드 태그: #포식종 #부화
트리거: 라운드 시작
효과: 이 카드에 부화 2기. 오른쪽 인접 카드에 부화 1기.
★2: 이 카드에 부화 4기. 오른쪽 인접에 부화 2기.
★3: 이 카드에 부화 4기. 양쪽 인접에 부화 2기씩. 부화한 유닛 ATK +10%(성장).

  포식종의 체인 시작. ★1부터 인접 공급.
  오른쪽에 탈피의 방 배치 → R1부터 변태 체인 가동.
  체인: [유충 둥지] 부화(자기+인접) → [탈피의 방] 변태 반응


[독 양식장] T1
구성: 독침 사수 ×1, 산성 거미 ×1               (2유닛, ATK 5, HP 27)
카드 태그: #포식종 #경제
트리거: 라운드 시작 + 전투 종료 시
효과: [라운드 시작] 이 카드에 부화 1기.
      [전투 종료] floor(이 카드 유닛 수 ÷ 5) 골드 (최대 3골드).
      전투 패배 시 골드 절반(버림).
★2: 부화 1기. floor(유닛수/5) 골드 (최대 5골드). 패배 시에도 전액.
★3: 부화 2기. floor(유닛수/5) 골드 (최대 7골드). 전액. 추가로 1 테라진.

  물량형 경제 카드. T1이라 초반부터 채용 가능.
  부화 1기/R → 유충 둥지 없이도 최소한의 부화 시작 가능 (둥지 필수성 해소).
  유닛 수 비례 수입 → 15기에 3골드, 25기에 5골드 → 물량을 쌓을수록 보상.


[탈피의 방] T2
구성: 칼날 유충 ×2, 변태 수호충 ×1             (3유닛, ATK 7, HP 55)
카드 태그: #포식종 #변태
트리거: 다른 카드의 부화 효과 발동 시
효과: 이 카드에서 변태 3기.
발동 상한: 2회/R
★2: 변태 2기. 발동 상한 3회/R.
★3: 변태 2기. 발동 상한 3회/R. 변태 시 이 카드 ATK +5%(성장 누적).

  변태 엔진. 유충 둥지의 인접 부화로 R1부터 유충 공급 가능.
  R1: 둥지 인접 부화 1기 → 탈피의 방 유충 3기 → 변태 발동(3기→수호충 1기).
  CP 비교: 유충 3기 CP=180 vs 수호충 1기 CP=180. ★1 CP 보존, ★2(2기→1기) CP +50%.
  METAMORPHOSIS 이벤트 emit → 적응 갑각, 포식자의 사냥 반응.


[군체 감각] T2
구성: 산성 거미 ×2, 칼날 유충 ×1               (3유닛, ATK 5, HP 34)
카드 태그: #포식종 #전투
트리거: 전투 시작 시
효과: 전체 포식종 유닛에 전투 버프 — 이 카드 유닛 수 3기당 ATK +10%
★2: 3기당 ATK +12%. 추가로 유닛 수 10기 이상이면 적 전체 AS -10%.
★3: 2기당 ATK +12%. 유닛 10기 이상이면 AS -15%, ATK -10%.

  물량 보상 카드. 유충을 많이 모을수록 전투력 버프.
  T2라 초반 급하지 않고, 유닛이 쌓인 뒤 투입해도 됨.
  배치: 유충 둥지 인접으로 부화 받아 자기 유닛 축적 → 버프 극대화.


[변태 수확] T2  ← 진화형 경제
구성: 산성 거미 ×1, 독침 사수 ×1               (2유닛, ATK 5, HP 27)
카드 태그: #포식종 #경제
트리거: 다른 카드의 변태 효과 발동 시
효과: 1 테라진. 이 카드에 부화 1기.
발동 상한: 1회/R
★2: 1 테라진. 부화 1기. 변태한 유닛 ATK +5%(성장). 2회/R.
★3: 1 테라진. 부화 2기. 전체 포식종 ATK +3%(성장). 2회/R.

  진화형 경제 카드. 변태가 발동할수록 테라진 수입 → 업그레이드 자금.
  ★1: 1테라진/R. ★2에서 2회/R = 2테라진/R로 ★2 합성 유인.
  독 양식장(물량형 골드) vs 변태 수확(진화형 테라진) = 경제 카드 분기.
  부화도 함 → 유충 둥지 대안 보강. 탈피의 방 없이는 경제 미작동.


[여왕의 산란] T3
구성: 여왕충 ×1, 갑충 일꾼 ×2, 칼날 유충 ×2   (5유닛, ATK 9, HP 90)
카드 태그: #포식종 #부화
트리거: 라운드 시작
효과: 이 카드에 부화 3기. 오른쪽 인접 카드에 부화 1기.
★2: 부화 4기(자기), 양쪽 인접 부화 2기
★3: 부화 5기(자기), 양쪽 인접 부화 3기, #여왕 유닛 HP +30%(성장 누적, 1회)

  대규모 부화. 유충 둥지의 상위 호환.
  인접 대량 배분으로 탈피의 방/군체 감각 등에 유충 공급.
  체인: [여왕] 부화 → [탈피의 방] 변태 반응 (인접 배치 시)


[적응 갑각] T3
구성: 뿔 돌진자 ×1, 돌격 전사충 ×1, 갑충 일꾼 ×1  (3유닛, ATK 9, HP 75)
카드 태그: #포식종 #강화
트리거: 다른 카드의 변태 효과 발동 시
효과: 카드에 #갑충 유닛이 있는 전체 포식종 카드 ATK+HP +5%(성장 누적). 오른쪽 인접 카드에 부화 1기.
발동 상한: 2회/R
★2: #갑충 카드 ATK+HP +7%. 인접 부화 1기. 2회/R.
★3: 전체 포식종 카드 ATK+HP +7%. 양쪽 인접 부화 1기씩. 3회/R.
     변태 원본 카드에 방어막(기본HP 20%).

  변태 반응 강화 + 부화 중계. 변태를 받아 강화하면서 인접에 유충을 공급.
  진화형 배치 [둥지]→[탈피]→[갑각]→[포식자]: 갑각이 포식자에 부화 공급.
  #갑충 = 일꾼, 전사충, 돌진자, 수호충 (4종) → 대부분의 포식종 유닛이 혜택.
  인접 부화 = HATCH 이벤트 emit → 탈피의 방이 다시 반응 가능 (추가 체인).


[기생 진화] T4
구성: 비행 포식자 ×2, 독침 사수 ×1             (3유닛, ATK 9, HP 55)
카드 태그: #포식종 #전투
트리거: 전투 종료 시
효과: 이번 전투에서 생존한 이 카드 유닛 1기당 이 카드에 부화 1기 (최대 3기/R).
      전투 승리 시 추가로 이 카드에서 변태 2기.
★2: 생존 유닛당 부화 2기 (최대 5기/R). 승리 시 변태 2기 + 생존 유닛 HP 15% 회복.
★3: 생존 유닛당 부화 2기 (최대 5기/R). 승패 무관 변태 2기. 생존 유닛 HP 20% 회복.
     변태된 유닛에 방어막(기본HP 30%).

  > 강화 과다 해소: ★2/★3의 ATK% 영구 성장 → HP 회복으로 대체.
  > "전투에서 살아남으면 회복하고 더 많이 낳는다" — 전투 생존 피드백.

  전투→생존 피드백 루프. 부화 상한으로 선형 증가 보장 (지수적 증가 방지).
  R7합류 시 R14까지: 3기(시작) + 3기/R × 8R = ~27기. 전투 생존 불문 상한.
  비행 포식자(AS 0.5) = 빠른 공격으로 적 처치 유리 → 생존율 높음.
  ★3에서는 패배해도 변태 + HP 회복 + 방어막 → 역전 가능성.


[포식자의 사냥] T4
구성: 거대 포식자 ×1, 뿔 돌진자 ×1             (2유닛, ATK 12, HP 60)
카드 태그: #포식종 #포식
트리거: 다른 카드의 변태 효과 발동 시
효과: 이 카드에서 변태 2기. 이 카드 유닛 ≤5기이면 ATK +30%(전투 버프).
발동 상한: 1회/R
★2: 변태 2기. ≤5기이면 ATK +50%. 발동 상한 2회/R.
★3: 변태 1기. ≤5기이면 ATK ×2(곱연산, 전투). 적 처치 시 HP 30% 회복.
     발동 상한 2회/R.

  먹이사슬 최상위. 변태 체인의 정점.
  탈피의 방의 변태 이벤트에 반응 → 이중 변태 체인.
  소수정예 조건(≤5기) = 포식자가 적을수록 더 강해지는 정점 보상.
  ★3에서 변태 1기(효율 극대화) + ATK ×2 + HP 회복 → 먹이사슬 왕.


[군체 초월] T5
구성: 거대 포식자 ×1, 여왕충 ×1, 변태 수호충 ×1  (3유닛, ATK 15, HP 105)
카드 태그: #포식종 #군체
트리거: 라운드 시작
효과: 이 카드에 부화 3기.
      전체 포식종 카드에 부화 1기씩.

[고유효과 — 포식 본능] 지속 효과:
  전투 중 포식종 유닛 사망 시, 생존 포식종 유닛에 ATK +3% 누적 (전투 한정).
  이 카드의 유닛은 적 처치 시 HP 10% 회복.

★2: 부화 4기(자기) + 전체 부화 2기. 사망 시 ATK +5%. 적 처치 시 HP 15% 회복.
★3: 부화 4기(자기) + 전체 부화 2기. 사망 ATK +5% + HP 15% 회복.
     매 라운드 자동 변태 1회, 변태 발생 시 이 카드 강화(ATK +5%, 영구 누적).

  > 강화 과다 해소: 매 라운드 전체 ATK +5% 영구 누적 → 전체 부화로 대체.
  > "캡스톤이 물량 공급 = 물량 테마 정체성 극대화. ATK 성장은 사망 광폭화에 집중."

  포식종 캡스톤. 2중 역할:
  1) 물량 공급 — 매 라운드 전체 부화로 유충 지속 공급. 물량형/진화형 모두 수혜.
     R10 합류 시 5R × 전체 1기/카드 = 카드당 +5기. 진화형은 즉시 변태 연료.
  2) 전투 동적 스케일링 — "약한 자가 쓰러지면 강한 자가 더 강해진다."
     물량형: 유충 대량 사망 → 생존자 광폭화 (유충이 연료)
     진화형: 소수 포식자가 유충 희생으로 ATK 폭증 (자연선택)
```

## ★2 합성 유인 검증

```
둥지:   ★1×2 = 4+2기/R. ★2×1 = 4+2기/R + 슬롯.      ✓ 동등+슬롯
양식:   최대 3g→5g. 패배 페널티 제거. floor(유닛/5)g.     ✓ 수치
탈피:   3기소모→2기소모. 변태 효율 50% 향상.             ✓ 수치
감각:   3기당10%→3기당12%. 효율 20% 향상.                ✓ 수치
수확:   상한 1→2회/R. ATK +5% 추가.                     ✓ 수치+효과
여왕:   부화 3→4, 인접 1→양쪽2. 총 4→8기/R.             ✓ 수치
갑각:   5%→7% + 인접부화 유지. 상한 2→2.                   ✓ 수치
기생:   부화 1→2(상한3→5), HP 15% 회복 추가.              ✓ 수치+효과
포식:   상한 1→2회/R. ATK 30%→50%.                     ✓ 수치
초월:   부화 2→3(자기)+전체1→2, 사망 ATK +3%→+5%.       ✓ 수치+물량
```

## 체인 흐름도

```
부화 생성:   유충 둥지(자기2+인접1), 독 양식장(자기1), 여왕의 산란(자기3+인접1),
             군체 초월(자기3+전체1/R), 기생 진화(전투 후), 변태 수확(변태 시 1기)
             ↓ HATCH 이벤트
변태 반응:   탈피의 방(변태 3기), 포식자의 사냥(변태 2기)
             ↓ METAMORPHOSIS 이벤트
강화+중계:   적응 갑각(#갑충 카드 ATK+HP +5% + 인접 부화 1기)
             변태 수확(1 테라진 + 부화 1기)
             ↓ HATCH 이벤트 (2차 체인)
             탈피의 방 추가 반응 가능

물량 지원:   군체 감각(유닛수 비례 ATK 전투버프)
경제:        독 양식장(물량형 골드) / 변태 수확(진화형 테라진)
피드백:      기생 진화(전투 생존→부화→변태)
캡스톤:      군체 초월(전투 중 사망→생존자 광폭화)
```

## 배치 권장

```
물량형:   [유충 둥지]→[독 양식장]→[군체 감각]→[여왕의 산란]→[기생 진화]→[군체 초월]
          둥지가 양식장에 인접 부화, 여왕이 기생진화에 인접 부화.
          군체 감각은 자체 유닛 축적 위해 부화 인접.

진화형:   [유충 둥지]→[탈피의 방]→[변태 수확]→[적응 갑각]→[포식자의 사냥]→[군체 초월]
          둥지→탈피(인접 부화→변태), 탈피 변태→수확(테라진)+갑각(강화)→포식자(이중 변태).
          변태 수확이 테라진 공급 → 업그레이드 자금.

하이브리드: [유충 둥지]→[탈피의 방]→[여왕의 산란]→[적응 갑각]→[기생 진화]→[군체 초월]
            둥지→탈피(인접 변태), 여왕(대량 부화+인접 공급).
```

## 빌드 경로

```
물량형:   유충둥지 + 독양식장 + 여왕산란 + 군체감각 + 군체초월
          → 유충 대량 생산, 변태 안 함, 물량 전투 버프 + 사망 광폭화
          → R14 예상: ~50유닛, 유충 사망으로 포식자/수호충 ATK 폭증

진화형:   유충둥지 + 탈피의방 + 변태수확 + 적응갑각 + 포식자사냥 + 군체초월
          → 유충 생산 → 즉시 변태 소모, 상위 유닛 육성, #갑충 강화, 테라진 수입
          → R14 예상: ~20유닛, 소수 정예가 사망 광폭화로 최종 2~3기가 초강화

하이브리드: 유충둥지 + 여왕산란 + 탈피의방 + 적응갑각 + 군체초월
          → 물량 생산 + 일부 변태, 양쪽 보너스 수취
          → R14 예상: ~35유닛, 균형 잡힌 구성
```
