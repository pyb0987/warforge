> ⚠ 이 카드들은 시스템 검증용 임시 카드입니다. 확정 카드 풀은 테마별로 별도 설계됩니다.
> 키워드가 구 체계(📈강화/🌱배양)를 사용합니다. 신규 체계(개량/제조 등)로 전환 필요.

## 예시 카드 10장 (증기기관 테마)

→ [DESIGN.md 원본의 🃏 카드 풀 섹션 참조]

1. 증기 발생기 — 라운드 시작, ⚡충전+강화 initiator
2. 기어 훈련소 — ⚡충전 reactor, #기어 강화
3. 녹슨 역장 — 📈강화 reactor, #상태 유닛 강화
4. 태양 용광로 — 구매 시, ⚡충전
5. 마력 결집기 — ⚡충전 reactor, 방어막+자신 강화
6. 증기 보일러 — 📈강화 reactor, #동력 HP 강화
7. 대장간 장인 — 📈강화 amplifier, #금속 추가 강화
8. 자동인형 군단 — 🛡버프 reactor
9. 전쟁 기계 — ⚔공격 reactor, 영구 강화
10. 종합 창고 — 판매 시, 경제

## 시너지 흐름 (예시 아키타입)

### 아키타입 A: ⚙️ 부품 조합
```
[증기 발생기] → ⚡충전 → [기어 훈련소] → 📈강화 → [대장간 장인] → 추가 강화
```

### 아키타입 B: 💨 증기 폭풍
```
[증기 발생기] → ⚡충전 → [증기 보일러] → HP 강화 → [태양 용광로] → 추가 ⚡충전
```

### 아키타입 C: 🛡️ 녹슨 방어
```
[녹슨 역장] → 📈강화 반응 → [전쟁 기계] → 영구 강화 → [자동인형 군단] → 버프 중첩
```

## 빌드 아키타입 (초안)

| 빌드명 | 핵심 성장 루프 | 플레이 스타일 |
|--------|----------------|---------------|
| 물량 | 유닛추가→유닛추가 | 유닛 수를 연쇄적으로 불려 물량으로 압도 |
| 정예 강화 | 강화→강화→유닛추가 | 소수 유닛 스탯을 극한까지 성장 |
| 합성 촉진 | ⬆합성→유닛추가/강화 보너스 | 빠른 ★ 업그레이드로 질적 우위 선점 |
| 사후 성장 | 전투 💀사망 → 라운드 후 강화/유닛추가 | 전투 피해를 성장 자원으로 전환 |
| 복합 체인 | 유닛추가 + 강화 교차 체인 | 두 축을 교차하는 긴 체인으로 폭발적 성장 |
