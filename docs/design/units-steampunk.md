## 비주얼 컨셉

```
태엽 거미:    황동 기계 거미. 6개 가느다란 다리. 몸통에 태엽 노출. 단안 붉은 렌즈.
             떼지어 달려드는 이미지. 납작한 원형 + 방사형 다리.
             색감: 어두운 황동, 기름때, 붉은 렌즈.

증기 쥐:     기계 쥐. 등에 소형 증기 분사구. 달리면서 증기 자국.
             입에서 근거리 증기 분사(Range 2). 금속 꼬리가 안테나처럼.
             색감: 기름 묻은 회철색, 배기구에서 흰 증기.

톱날 구동체:  무한궤도 위 거대 회전 톱날이 장착된 소형 전차.
             톱날이 앞으로 향하고 적에게 돌진. 기어와 체인 노출.
             SC1 벌처처럼 낮고 공격적. 색감: 녹슨 철, 톱날 은색, 스파크.

증기 전갈:   4족 보행 기계 전갈. 꼬리에 증기 분사 침(Range 2).
             집게로 근접 공격도 가능. 등에 작은 보일러.
             어둡고 위협적. 색감: 어두운 구리, 꼬리 끝 붉은 증기, 기름 코팅.

철갑 게:     기계 게. 두꺼운 철판 갑각. 거대한 집게 2개.
             옆으로 걸으며 접근. 넓은 전면 장갑이 방패 역할.
             색감: 어두운 강철, 리벳, 긁힌 자국, 녹.

증기 타이탄:  4족 보행 거대 기관. 게임 내 가장 큰 스팀펑크 유닛.
             등에 거대한 굴뚝이 검은 연기를 뿜음. 다리는 유압 피스톤.
             걸어다니는 보일러/용광로. 내부에서 붉은 빛.
             색감: 검은 철, 녹슨 파이프, 검은 연기, 내부 용광로 붉은빛.
             → 보드에 올라오면 "왔다" 하는 존재감.

황동 포차:   3~4바퀴 이동식 대포. 황동 포신 돌출.
             뒤에 탄약 상자. 발사 시 연기와 섬광. 투박하고 실용적.
             색감: 황동 포신, 나무+철 차체, 발사 시 주황 섬광.

전기 드론:   공중 부유 구형 드론. 전기 아크가 주변에 튐.
             4개 작은 프로펠러. 하단 전기 방출구.
             불안정한 전기 이펙트. 색감: 어두운 회색, 파란/흰 전기 아크.

증기 터렛:   넓고 낮은 고정 포대. 거대한 증기 대포.
             뒤에 증기 탱크 연결. 발사 시 거대한 증기 구름.
             거의 움직이지 않는 "고정 화력 거점".
             색감: 강철 포대, 구리 증기탱크, 흰 연기+주황 불꽃.

태엽 정찰기:  외발 바퀴(모노휠) 로봇. 상체에 망원경 센서.
             안테나 1개. 빠르게 굴러다니며 정찰. 작지만 어두운 톤.
             색감: 어두운 황동, 센서 렌즈 초록, 안테나 은색.
```

## 실루엣 차별화

```
태엽 거미:   납작+다리 방사 (절지형)
증기 쥐:    쥐+등 배기구 (동물형)
톱날 구동체: 무한궤도+회전 톱날 (전차형)
증기 전갈:   4족+꼬리 침 (절지형, 거미보다 큼)
철갑 게:    넓은 전면+집게 (갑각형)
증기 타이탄: 4족 거대+굴뚝 (거대 기관)
황동 포차:   바퀴+포신 돌출 (차량형)
전기 드론:   구형+프로펠러 (비행체)
증기 터렛:   낮고 넓은+대포 (고정형)
태엽 정찰기: 외발 바퀴+센서 (단륜형)

→ 10종 모두 실루엣만으로 즉시 구분 가능
→ 인간형 0종
```
