# 군대 카드 풀 (10장)

> 효과/수치 상세: `data/cards/military.yaml` 참조 (단일 진실 소스).
> 이 문서는 설계 의도와 빌드 전략을 기술합니다.

> 유닛 풀: units-military.md 참조.
> 핵심 판타지: "신병이 정예로 거듭나는 군대" — 훈련→승급→(강화)의 쾌감.
> 고유 엔진: 인접 훈련 전파 + 계급 임계점 해금 + 징집 3택1(기수 차등).
> 테마 정체성: 드루이드(🌳축적), 스팀펑크(제조↔개량), 포식충(부화→변태), **군대(훈련→승급→강화)**.

## 계급 시스템

```
⭐ = 카드별 내부 카운터. 비전투. "계급 N" 표시.
소모 없음. 누적 전용.

훈련 = 이 카드의 계급 +1
시작 계급 = 0.

훈련 전파:
  훈련소/특수작전대: 자기 훈련 + 인접 군대 카드에도 훈련(계급+1).
  군사학교: 훈련 반응 → 추가 훈련(증폭).
  통합사령부(T5): 전체 군대 카드 훈련(계급+1).
  → 정예형은 훈련소 주변에 카드를 배치해서 계급 전파.
  → 물량형은 전진기지 주변에 배치해서 징집 전파.
  → 배치 순서가 전략적 결정.

임계점 보너스 (시스템 자동 해금, 수치는 YAML 참조):
  계급 3:  이 카드 유닛 ATK 강화. 이 카드의 비(강화) 유닛 1기를 (강화)로 변환.
  계급 5:  이 카드 유닛 ATK 추가 강화 + AS 향상. 비(강화) 유닛 1기 추가 (강화) 변환.
  계급 8:  이 카드 유닛 ATK 추가 강화 + AS 향상. 리더 유닛(전술 지휘관) 1기 생성.
           이 카드의 모든 유닛을 (강화)로 변환.

임계점 보너스는 카드 modifier로 누적 (성장과 동일 스택).
계급이 임계점 미만으로 떨어지는 일은 없음 (소모 없음).
(강화) 변환: 카드 내 가장 약한(CP) 비(강화) 유닛부터 변환.
(강화) 유닛은 기존 유닛 타입의 상위 스탯 버전 (units-military.md 참조).

징집:
  이 카드에 유닛 추가. 징집 풀에서 무작위 3종 제시, 플레이어 1택.
  기수 차등 (수치는 YAML 참조):
    신병:         다수 배치 (물량형)
    정찰 드론:    중간 배치
    강습 바이커:  중간 배치
    보병:         소수 배치
    에너지 방패병: 소수 배치
    플라즈마 사수: 소수 배치 (화력형)
  → "신병 다수(물량) vs 사수 소수(화력)" 트레이드오프.

징집 제외 (4종 — 임계점/리더/카드전용):
  저격 드론, 궤도 포대, 전술 지휘관, 중장갑 워커
  → 훈련 임계점 배치, 리더 생성, 카드 composition으로만 등장.

(강화) 버전:
  징집 풀 6종 각각에 (강화) 버전 존재 (units-military.md 참조).
  (강화) 기수도 동일 차등 패턴 (수치는 YAML 참조).
  징병국 ★3: 징집 유닛이 해당 유닛의 (강화) 버전으로 자동 변환.

★합성 시 계급: 3장 중 최고 계급을 승계 (max, 합산 아님).
```

## 서브 전략: 정예형 vs 물량형

```
정예형 (소수 고계급 → 전원 (강화) → 부활):
  훈련소(T1) → 군사학교(T2) → 보급(T2) → 전술사령부(T3) → 특수작전대(T4) → 통합사령부(T5)

  배치: [보급][전술사령부][훈련소][군사학교][특수작전대][...]
        ← 훈련소 인접 3장이 계급 수혜 →

  훈련소 인접+군사학교 증폭 = 인접 카드 계급 빠르게 상승.
  훈련소 자체: ★2 가속으로 빠른 임계점 도달.
  인접 카드: 빠른 계급 상승 → 전술사령부 ★3 고위 임계 달성.
  (강화) 경로: 계급3→1기, 계급5→+1기, 계급8→전원.
  유닛 수 중간. (강화) 비율 높음.
  통합사령부: 전원 부활 = 실질 전투력 대폭 증가 (수치는 YAML 참조).

물량형 (대규모 징집 → (강화) 탱킹 → 부활):
  전진기지(T1) → 징병국(T2) → 보급(T2) → 돌격편대(T3) → 군수공장(T4) → 통합사령부(T5)

  배치: [돌격편대][전진기지][징병국][보급][군수공장][...]

  징집 다수/R. 신병 선택으로 물량 극대화.
  후반에 대규모 유닛. (강화)는 징병국 ★2~3으로.
  돌격편대: (강화)=2기 카운트 → (강화) 보유할수록 ATK 버프 효율 향상.
  통합사령부: (강화) 소수가 부활 → 전열 유지.
  군수공장: 신병 다수 징집 → 카운터 빠르게 축적 → 테라진 수입.

공유 카드: 보급(T2), 전술사령부(T3)
정예 전용: 훈련소(T1), 군사학교(T2), 특수작전대(T4) = 3장
물량 전용: 전진기지(T1), 징병국(T2), 돌격편대(T3), 군수공장(T4) = 4장
양쪽 캡스톤: 통합사령부(T5)
분기 시점: T1 (훈련소 vs 전진기지 선택)

하이브리드 벌점:
  훈련소는 유닛 추가 없음(임계점 3기만) → 물량 기여 미미.
  전진기지는 훈련 없음 → 계급 기여 0.
  8장 필드에서 양쪽 핵심 모두 넣으면 T3~T4 공백.
```

## ★2 합성 유인 검증

```
훈련소:     ★1=자기+1+인접+1, ★2=자기+2+인접+1. 자기만 가속.               ✓ 가속
전진기지:   ★1=징집, ★2=징집+인접. 슬롯 절약 + 물량 확대.                   ✓ 확장
군사학교:   ★1=추가+1, ★2=+1+ATK 성장+횟수 증가. 수치+새 효과.             ✓ 수치
징병국:     ★1=추가 징집, ★2=수량 증가+일부 (강화)+횟수 증가. 수치+새효과.   ✓ 수치
보급:       ★1=기본+비례, ★2=기본 증가+비례 강화+패배 전액. 수치+안정.       ✓ 수치
전술사령부:  ★1=계급 비례+유닛 비례, ★2=계수 강화. 수치.                     ✓ 수치
돌격편대:   ★1=유닛 비례, ★2=계수 강화+조건 완화. 수치.                     ✓ 수치
특수작전대:  ★1=계급 임계→리더, ★2=임계 완화→리더 강화. 조건 완화.           ✓ 조건
군수공장:   ★1=임계→테라진, ★2=임계 완화+레어 할인+ATK 성장, ★3=임계 추가 완화+에픽 포함+테라진 증가. ✓ 수치
통합사령부:  ★1=부분 HP 부활, ★2=HP 증가+ATK 버프, ★3=훈련+2+다수 부활(HP100%+ATK 누적+방어막)+전원 부활. ✓ 수치
```

## 계급 흐름도

```
계급 생성:    훈련소(자기+인접), 특수작전대(자기+인접), 통합사령부(전체, ★3는 +2/R)
계급 증폭:    군사학교(훈련 반응→추가+1, 횟수 제한)
계급→(강화):  임계점 3/5/8에서 순차 (강화) 변환. 훈련소 임계점 배치유닛은 (강화).
계급→보너스:  시스템 임계점 3/5/8 자동 해금, 훈련소 임계점 배치, 특수작전대 리더
계급→후반:    계급 고위 임계점 (훈련소 ★3 곱연산, 전술사령부 ★3 AS, 특수작전대 ★3 리더 확산)
계급→경제:    보급 ★3(이 카드 계급 임계+→테라진)
계급→방어:    전술사령부(이 카드 계급 비례 방어막)

징집 생성:    전진기지(기수 차등 3택1)
징집 증폭:    징병국(징집 반응→추가 무작위 징집)
징집→(강화):  징병국 ★2(일부), ★3(전원)
징집→전투:    돌격편대(유닛 수 × ATK%, (강화)=2기 카운트)
징집→경제:    군수공장(징집 횟수→카운터→테라진. 기수 무관 +1/회)

(강화)→방어:  전술사령부((강화) 유닛 추가 방어막)
(강화)→전투:  돌격편대((강화) 2기 카운트)
(강화)→부활:  통합사령부(최종 작전: (강화) 사망 시 부활)
```

## 타임라인 — 정예형

```
R1:    [훈련소] 배치. 자기 계급+1. 초반 전투. 빠듯.
R2:    계급 2. 인접에 보급 배치 → 보급도 계급+1/R 시작.
R3:    훈련소 계급 3 → ATK 강화 + 보병(강화) 배치. "첫 정예 탄생." (수치는 YAML 참조)
       군사학교 합류 → 인접 카드 계급 가속.
R4:    전술사령부 배치(훈련소 인접). 전술사령부 계급 상승 시작.
R5:    훈련소 계급 5 → 사수(강화) 배치 + 추가 (강화). "정예 확대."
       보급 계급 임계 → ★3이면 테라진 해금.
R7:    특수작전대 합류(보드 반대편). 인접 카드에 훈련 전파 시작.
R8:    훈련소 ★2 계급 고위 임계 달성 → ATK 곱연산.
       전술사령부 계급 고위 임계 임박.
       훈련소 계급 8 → 워커 배치! 전원 (강화)! + 리더.
R10:   통합사령부 ★3 합류. 전체 +2/R (계급 가속 2배). (강화) 전원 다수 부활.
       "전원 부활 + 누적 ATK 버프 = 죽을수록 강해지는 군대."
       통합사령부 계급 임계 → 전 유닛 부활 해금.
R12:   전술사령부 계급 고위 임계 → AS 향상. 특수작전대 계급 고위 임계 → 리더 버프 확산. 사실상 불패.
```

## 타임라인 — 물량형

```
R1:    [전진기지] 배치. 징집 → 초반부터 물량 우세.
       3택1: "신병 다수(물량) vs 사수 소수(화력)" 매 라운드 고민. (카운터는 횟수 기준, 기수 무관)
R3:    징병국 합류. R당 징집 수 증가.
R5:    보급+돌격편대 합류.
       유닛 다수 확보. 돌격편대: 유닛 수 비례 ATK 전투 버프 (수치는 YAML 참조).
R6:    징병국 ★2 → 징집 일부 (강화). "첫 정예 합류."
R7:    군수공장 합류. 징집 횟수 기준 카운터 축적 시작.
R9:    유닛 다수 + (강화) 일부. 돌격편대 버프 증가.
R10:   통합사령부 ★3 합류. 전체 +2/R. (강화) 부활 발동.
       군수공장 ★3: 에픽 할인 + 테라진 수입 증가.
       "에픽으로 무장한 대군이 여러 번 일어남 = 물량형 최종 형태."
R14:   유닛 대폭 증가 + (강화) 비율 증가. 돌격편대 버프 극대화.
       (강화) 부활 = 전투 지속력 폭발.
```
