> 핵심 판타지: "신병이 정예로 거듭나는 군대" — 훈련→승급→(강화)의 쾌감.
> 고유 엔진: 훈련 범위 전파 + 계급 milestone 해금 + 징집 3택1(기수 차등).
> 테마 정체성: 드루이드(🌳축적), 스팀펑크(제조↔개량), 포식충(부화→변태), **군대(훈련→승급→강화)**.

## ★/R 두 축 분리 원칙 (2026-04-16 재설계)

각 카드는 **두 개의 독립 진행 축**을 가진다:
- **★ 축**: 양적 스케일링 (수치 증가, 수량 증가, 계수 강화)
- **R 축**: 질적 전환 (효과의 성격/범위/해금 단계 변화)

**R milestone**: 계급 4 (R4), 계급 10 (R10) 두 단계.
(이전 계급 3/5/8 임계점 시스템 폐지)

### 공통 R 효과
- **R4 도달**: 이 카드의 비(강화) 유닛 중 절반이 (강화)로 변환
- **R10 도달**: 이 카드의 모든 유닛이 (강화)로 변환

## 계급 시스템

```
⭐ = 카드별 내부 카운터. 비전투. "계급 N" 표시.
소모 없음. 누적 전용.

훈련 = 이 카드의 계급 +1 (유닛은 추가되지 않음)
징집 = 이 카드에 유닛 추가 (계급은 오르지 않음)
시작 계급 = 0.

훈련 전파 (★1 기준, ★/R로 확장):
  훈련소: self + 오른쪽 인접 (R0) → 양쪽 (R4) → 전체 군대 카드 (R10)
  통합사령부(T5): 전체 군대 카드 +1/R (★3: +2/R)
  군사학교: 훈련 이벤트 감지 → 추가 훈련 (증폭자)
  → 정예형은 훈련소 주변에 카드를 배치해서 계급 전파.
  → 물량형은 징병국 주변에 배치해서 징집 전파.
  → 배치 순서가 전략적 결정.

계급 milestone (시스템 자동 해금, 수치는 YAML 참조):
  계급 4 (R4):  이 카드 비(강화) 유닛의 절반이 (강화)로 변환 + 카드별 R4 고유 효과.
  계급 10 (R10): 이 카드의 모든 유닛이 (강화)로 변환 + 카드별 R10 고유 효과.

계급 milestone은 카드 modifier로 누적 (성장과 동일 스택).
계급이 milestone 미만으로 떨어지는 일은 없음 (소모 없음).
(강화) 변환: 카드 내 가장 약한(CP) 비(강화) 유닛부터 변환.
(강화) 유닛은 기존 유닛 타입의 상위 스탯 버전 (units-military.md 참조).

징집:
  이 카드에 유닛 추가. 징집 풀에서 무작위 3종 제시, 플레이어 1택.
  기수 차등 (수치는 YAML 참조):
    신병:         다수 배치 (물량형)
    정찰 드론:    중간 배치
    강습 바이커:  중간 배치
    보병:         소수 배치
    에너지 방패병: 소수 배치
    플라즈마 사수: 소수 배치 (화력형)
  → "신병 다수(물량) vs 사수 소수(화력)" 트레이드오프.

징집 제외 (R0 기본 상태, 4종 — 리더/카드전용):
  저격 드론, 궤도 포대, 전술 지휘관, 중장갑 워커
  → 징병국 R4에서 (강화) 유닛 풀 해금, R10에서 엘리트 유닛 풀 해금.

(강화) 변환 경로:
  - 징병국 R4/R10 풀 확장
  - 통합사령부/군사학교 효과 기반 개별 변환
  - 공통 R 효과 (모든 카드 R4/R10에서 자동 변환)

★합성 시 계급: 3장 중 최고 계급을 승계 (max, 합산 아님).
```

## ★/R 매트릭스 요약

| 카드 | ★ 축 | R 축 |
|---|---|---|
| 훈련소 | 훈련 수량 (self 가속) | 전파 범위 (오른쪽→양쪽→전체 군대) |
| 징병국 | 징집 수량/범위 | 징집 풀 등급 (일반→강화→엘리트) |
| 전진기지 | 반응 수량 / (강화) 비율 | 증원 범위 (대상→대상+인접→전체) |
| 군사학교 | max_act / 훈련량 / enhance | 훈련 질적 전환 (강화 변환 / 랜덤 강화 유닛 추가) |
| 보급부대 | 골드 계수 | 자원 해금 (골드만 → +테라진 → +테라진 확대+골드) |
| 전술사령부 | rank_buff 계수 | 버프 차원 추가 (HP/rank → AS) |
| 돌격편대 | 바이커 생산 (1→2→4기) | R4: swarm_buff 해금 / R10: 라이프스틸 10% |
| 특수작전대 | 크리 배율 × 저격드론 생산 (2/3/6×0/1/3) | 치명타 확률 + 스플래시 |
| 군수공장 | counter → 전군 ATK% (+R+1@★3) | R4: 상점 슬롯/할인 / R10: 슬롯+할인 누적 |
| 통합사령부 | 훈련량 + 부활 HP (25/50/100%) | 부활 범위 (자기강화→자기전체→인접 3장 전체) |

  ★3+R10 조합: "배율 6 × 30% 확률 × 스플래시 50%" = 엔드게임 DPS 폭발.

  R10 "군수 최적화": 슬롯 추가 +1 누적 (총 +2), 할인 추가 -1 누적 (총 -2).
  → 테라진 직접 공급은 보급부대와 분리. 군수공장은 업그레이드 효율 담당.


[통합 사령부] T5 — 캡스톤 (훈련 + 부활)
구성: 전술 지휘관 ×1, 중장갑 워커 ×1, 궤도 포대 ×1

  R4: 부활 대상이 이 카드 (강화) → 이 카드 모든 유닛 (비강화 포함).

  → ★과 R 독립 축. ★3+R10 = 인접 3장의 모든 유닛 HP 100% 부활 (전투당 1회).

## 서브 전략: 정예형 vs 물량형

```
정예형 (소수 고계급 → 전원 (강화) → 부활):
  훈련소(T1) → 군사학교(T2) → 보급(T2) → 전술사령부(T3) → 특수작전대(T4) → 통합사령부(T5)

  배치: [보급][전술사령부][훈련소][군사학교][특수작전대][...]
        ← 훈련소 R4+ 양쪽 전파: 인접 2장 수혜 →

  R0: 훈련소가 self + 오른쪽 전파. ★2 self 가속.
  R4: 훈련소 range 확장 + 모든 R4 카드 비(강화) 절반 변환.
  R10: 전체 군대 훈련 가속 + 전원 (강화).
  특수작전대: 크리 + 저격 드론 다수 → 정예형 DPS 피크.
  통합사령부: 전원 부활 = 실질 전투력 대폭 증가.

물량형 (대규모 징집 → (강화) 탱킹 → 부활):
  징병국(T1) → 전진기지(T2) → 보급(T2) → 돌격편대(T3) → 군수공장(T4) → 통합사령부(T5)

  배치: [돌격편대][징병국][전진기지][보급][군수공장][...]

  R0: 징병국 self 징집 + 전진기지 반응 증폭. 돌격편대 바이커 생산.
  R4: 징병국 풀 (강화) 해금 + 전진기지 반응 범위 확장 + 돌격편대 swarm_buff 해금.
  R10: 징병국 엘리트 풀 + 전진기지 전체 군대 증원 + 돌격편대 라이프스틸.
  군수공장: 업그레이드 상점 최적화 + 전군 ATK 버프.
  통합사령부: (강화) 유닛 부활로 전열 유지.

공유 카드: 보급(T2), 전술사령부(T3), 군수공장(T4)
정예 전용: 훈련소(T1), 군사학교(T2), 특수작전대(T4) = 3장
물량 전용: 징병국(T1), 전진기지(T2), 돌격편대(T3) = 3장
양쪽 캡스톤: 통합사령부(T5)
분기 시점: T1 (훈련소 vs 징병국 선택)
```

## 타임라인

> **표기 규약**: 아래 "R1~R15"는 **라운드 번호**이다.
> 카드별 계급 milestone은 "**계급 4**", "**계급 10**"으로 구분해 표기한다.
> 각 카드는 훈련 여부/빈도에 따라 라운드 번호와 관계없이 계급이 쌓인다.

### 정예형

```
Round 1:    [훈련소] 배치. 자기 계급+1/R. 오른쪽 인접 카드에도 +1/R 시작.
Round 2:    계급 2. 인접에 보급 배치 → 보급도 계급+1/R 시작.
Round 3:    군사학교 합류 → 훈련 이벤트 증폭 시작.
Round 4:    ★ 훈련소 "계급 4" milestone 달성:
             → 훈련소 비(강화) 절반 변환 ("첫 정예 탄생")
             → 훈련 범위 양쪽으로 확장 → 보급/전술사령부도 수혜.
Round 5-6:  전술사령부 배치. 양쪽 훈련 수혜. 특수작전대 "계급 4" 돌파 시 크리+스플래시.
Round 7:    특수작전대 합류. 저격 드론 생산 + 크리 작동 시작.
Round 8-9:  훈련소 ★2 가속. 훈련소 계급 6~8 도달.
Round 10:   ★ 훈련소 "계급 10" milestone 달성:
             → 훈련소 전원 (강화) 변환
             → 훈련 범위 전체 군대 카드로 확장 (모든 군대 +1/R)
             → 통합사령부 "계급 10" 도달 시 인접 3장 전체 부활.
Round 12:   정예형 엔드게임. ★3+계급 10 스택: 특수작전대 배율 6 × 확률 30% × 스플래시 50%.
```

### 물량형

```
Round 1:    [징병국] 배치. 매 RS 신병 풀 3택1 2기.
Round 3:    전진기지 합류. 징집 이벤트마다 추가 증원 시작.
Round 4:    ★ 징병국 "계급 4" milestone:
             → 징병국 비(강화) 절반 변환
             → 징집 풀에 (강화) 유닛 포함 ("첫 정예 등장")
            돌격편대 "계급 4" milestone: swarm_buff 해금 — 물량 수량 비례 ATK 버프 작동.
Round 5:    보급부대 + 돌격편대 합류. 바이커 생산 + swarm_buff 상승.
Round 7:    군수공장 합류. counter 축적 시작. 업그레이드 상점 혜택.
Round 9:    군수공장 counter 임계 달성 → 전군 ATK +5~10% 영구.
Round 10:   ★ 전진기지 "계급 10" milestone:
             → 전진기지 전원 (강화)
             → 반응 범위 전체 군대 카드로 확장 (징집 1회 = 전군 증원)
            돌격편대 "계급 10" milestone: 전군 라이프스틸 10% 해금.
            군수공장 "계급 10" 도달 시 상점 슬롯 +2, 할인 -2 누적.
Round 12:   물량형 엔드게임. 통합사령부 "계급 10" 인접 3장 부활 + 라이프스틸 + 전군 ATK 영구.
```

## 구현 필요 항목 (YAML placeholder → 구현 스프린트)

이 재설계에서 YAML에 선언되었으나 실제 combat/system 로직 구현이 필요한 효과:

| 효과 | 카드 | 구현 난이도 |
|---|---|---|
| `crit_buff` (카드 유닛에 치명타 mechanic 적용) | 특수작전대 | MEDIUM |
| `crit_splash` (크리 발동 시 인접 적 스플래시) | 특수작전대 | MEDIUM |
| `revive` + `revive_scope_override` | 통합사령부 | HARD (현재 placeholder) |
| `upgrade_shop_bonus` (슬롯+할인 카드 효과) | 군수공장 | HARD |
| `rank_buff_hp` (hp_per_rank 조합) | 전술사령부 | MEDIUM |
| `enhance_convert_target` (훈련 시 강화 변환) | 군사학교 R4 | MEDIUM |
| `spawn_enhanced_random` (랜덤 강화 유닛 추가) | 군사학교 R10 | MEDIUM |
| `conscript_pool_tier` (풀 등급 확장) | 징병국 R4/R10 | MEDIUM |
| `lifesteal` on swarm_buff | 돌격편대 R10 | EASY (mechanic 이미 존재) |
| `enhance_convert_card` (공통 R4/R10 변환) | 모든 카드 | MEDIUM |
| `spawn_unit` (카드에 특정 유닛 추가) | 돌격편대/특수작전대 | EASY (conscript 재사용) |
| `grant_gold`, `grant_terazin` | 보급부대 R10 | EASY |

기존 구현 재사용:
- `train`, `conscript`, `enhance`, `counter_produce`, `swarm_buff`, `rank_buff`, `economy`
- `splash`, `critical`, `armor_pierce`, `chain_explosion` (mechanics_handler.gd)
- Upgrade shop slot via `Talisman` / Commander `단조사` discount
