# 군대 카드 풀 (10장)

> 유닛 풀: units-military.md 참조.
> 핵심 판타지: "신병이 정예로 거듭나는 군대" — 훈련→승급→(강화)의 쾌감.
> 고유 엔진: 인접 훈련 전파 + 계급 임계점 해금 + 징집 3택1(기수 차등).
> 테마 정체성: 드루이드(🌳축적), 스팀펑크(제조↔개량), 포식충(부화→변태), **군대(훈련→승급→강화)**.

## 계급 시스템

```
⭐ = 카드별 내부 카운터. 비전투. "계급 N" 표시.
소모 없음. 누적 전용.

훈련 = 이 카드의 계급 +1
시작 계급 = 0.

훈련 전파:
  훈련소/특수작전대: 자기 훈련 + 인접 군대 카드에도 훈련(계급+1).
  군사학교: 훈련 반응 → 추가 훈련(증폭).
  통합사령부(T5): 전체 군대 카드 훈련(계급+1).
  → 정예형은 훈련소 주변에 카드를 배치해서 계급 전파.
  → 물량형은 전진기지 주변에 배치해서 징집 전파.
  → 배치 순서가 전략적 결정.

임계점 보너스 (시스템 자동 해금):
  계급 3:  이 카드 유닛 ATK +15%. 이 카드의 비(강화) 유닛 1기를 (강화)로 변환.
  계급 5:  이 카드 유닛 ATK +25%, AS ×0.85. 비(강화) 유닛 1기 추가 (강화) 변환.
  계급 8:  이 카드 유닛 ATK +35%, AS ×0.85. 리더 유닛(전술 지휘관) 1기 생성.
           이 카드의 모든 유닛을 (강화)로 변환.

임계점 보너스는 카드 modifier로 누적 (성장과 동일 스택).
계급이 임계점 미만으로 떨어지는 일은 없음 (소모 없음).
(강화) 변환: 카드 내 가장 약한(CP) 비(강화) 유닛부터 변환.
(강화) 유닛은 기존 유닛 타입의 상위 스탯 버전 (units-military.md 참조).

징집:
  이 카드에 유닛 추가. 징집 풀에서 무작위 3종 제시, 플레이어 1택.
  기수 차등:
    신병:         3기 배치
    정찰 드론:    2기 배치
    강습 바이커:  2기 배치
    보병:         1기 배치
    에너지 방패병: 1기 배치
    플라즈마 사수: 1기 배치
  → "신병 3기(물량) vs 사수 1기(화력)" 트레이드오프.

징집 제외 (4종 — 임계점/리더/카드전용):
  저격 드론, 궤도 포대, 전술 지휘관, 중장갑 워커
  → 훈련 임계점 배치, 리더 생성, 카드 composition으로만 등장.

(강화) 버전:
  징집 풀 6종 각각에 (강화) 버전 존재 (units-military.md 참조).
  (강화) 기수도 차등: 신병(강화) 3기, 드론/바이커(강화) 2기, 나머지 1기.
  징병국 ★3: 징집 유닛이 해당 유닛의 (강화) 버전으로 자동 변환.

★합성 시 계급: 3장 중 최고 계급을 승계 (max, 합산 아님).
```

구성: 신병 ×2, 보병 ×1                      (3유닛, ATK 12, HP 110)
카드 태그: #군대 #훈련
트리거: 라운드 시작
효과: 이 카드 훈련(계급+1). 인접 군대 카드에도 훈련(계급+1).

      계급 5 도달 시: 플라즈마 사수(강화) 1기 배치.

★2: 이 카드 계급+2. 인접 훈련은 +1 유지. 동일 임계점.
★3: 이 카드 계급+2. 임계점 배치 유닛이 즉시 ATK +20%.

  정예형의 핵심 스타터. 자기 훈련 + 인접 전파.

  ★3 계급 15: +2/R이면 R8에 도달. 정예형 후반 보상.
  "달성의 쾌감": 임계점마다 새 정예 병종 합류 + 인접 카드도 성장.


[전진 기지] T1 — 물량 스타터
구성: 신병 ×3, 정찰 드론 ×1                 (4유닛, ATK 12, HP 110)
카드 태그: #군대 #전선
트리거: 라운드 시작
효과: 이 카드 징집 2기 (3택1).
★2: 징집 2기 + 인접 군대 카드에 징집 1기 (무작위, 플레이어 선택 없음).
★3: 징집 3기 + 양쪽 인접 각 징집 1기 (무작위).

  물량형의 핵심 스타터. R1부터 6유닛.
  이 카드 징집만 3택1(플레이어 결정). 인접 징집은 무작위(속도 우선).
  ★3: 유닛 10기 조건이 물량형에서 자연 달성 → 전투 보너스.


[군사 학교] T2 — 정예 핵심 (훈련 증폭)
구성: 전술 지휘관 ×1, 보병 ×1               (2유닛, ATK 10, HP 105)
카드 태그: #군대 #훈련
트리거: [고유효과] 다른 카드의 훈련 효과 발동 시
효과: 훈련이 발동된 카드에 추가 훈련(계급+1).
발동: 이 카드의 총 반응 횟수 최대 2/R.
★2: 추가 계급+1 + 훈련이 발동된 카드 ATK +2%(영구 누적). 총 반응 최대 3/R.
★3: 추가 계급+2 + 훈련이 발동된 카드 ATK +3%(영구 누적). 총 반응 최대 3/R.

  훈련소가 자기+인접에 훈련 → 군사학교 반응 → 훈련된 카드에 추가 계급.
  배치 예: [전술사령부][훈련소][군사학교] → 전술사령부가 +1(인접)+1(군사학교)=+2/R.
  "총 반응 최대 N/R" = 군사학교 자체의 라운드당 총 발동 횟수 상한.
  ★3 15R 실측: +3% × 3/R × 15R = +135% ATK 누적 (SC1 안전 범위).


[징병국] T2 — 물량 핵심 (징집 증폭)

★2: 추가 징집 2기(1기는 (강화) 버전). 총 반응 최대 3/R.
★3: 추가 징집 2기(전원 (강화) 버전). 총 반응 최대 3/R.

  전진기지의 징집 2기 → 징병국 반응 → 추가 1기 = R당 3기.
  징병국의 추가 징집은 무작위(3택1 아님) = 속도 유지.
  ★2: (강화) 접근 시작. ★3: 전원 (강화) → "전군 무장 완료".


[보급 부대] T2 — 공유 경제
구성: 정찰 드론 ×1, 강습 바이커 ×1           (2유닛, ATK 8, HP 60)

  훈련소 인접이 아니면 계급이 안 올라 테라진 미해금 → 배치 결정.

      (강화) 유닛 각각에 추가 방어막 +5%(기본 HP, 유닛 개별 적용).
★2: 계급 × 3% 방어막. ATK +유닛 수 × 0.8%. (강화) 개별 추가 방어막 +8%.
★3: 계급 × 4% 방어막. ATK +유닛 수 × 1%. (강화) 개별 추가 방어막 +10%.
     계급 15: 전체 군대 유닛 AS ×0.85 (엘리트 지휘).

  "이 카드 계급" → 훈련소 인접이면 +1/R(+군사학교면 +2/R). 비인접이면 0.
  정예형: 훈련소 옆 배치 → 계급↑ → 방어막 두꺼움 + ★3 계급 15 AS 보너스.
  물량형: 계급 낮지만 유닛 수 ATK 버프가 핵심.
  (강화) 유닛이 탱킹 → "강화 장갑의 판타지".

★3: 유닛 수 × 2%. 10기 이상이면 MS +1. 20기 이상이면 AS ×0.85. (강화) 2기 카운트.

  (강화) 유닛의 2배 카운트 = (강화) 투자 보상.
    물량형: 실 20기 중 (강화)5기 → 카운트 25기 = +25%.
    정예형: 실 12기 중 (강화)8기 → 카운트 20기 = +20%.

      테라진 2/발동으로 꾸준한 수입 + 할인으로 구매 효율 극대화.

## 서브 전략: 정예형 vs 물량형

```
정예형 (소수 고계급 → 전원 (강화) → 부활):
  훈련소(T1) → 군사학교(T2) → 보급(T2) → 전술사령부(T3) → 특수작전대(T4) → 통합사령부(T5)

  배치: [보급][전술사령부][훈련소][군사학교][특수작전대][...]
        ← 훈련소 인접 3장이 계급 수혜 →

  훈련소 +1/R(인접) + 군사학교 +1/R(증폭) = 인접 카드 +2/R.
  훈련소 자체: ★2 +2/R → R8에 계급 16 (계급15 임계점 도달).
  인접 카드: +2/R → R8에 계급 16. 전술사령부 ★3 계급15 = AS ×0.85.
  (강화) 경로: 계급3→1기, 계급5→+1기, 계급8→전원.
  유닛 수 ~15기. (강화) 비율 100%.
  통합사령부: 전원 부활 1회 = 실질 30기 전투력.

물량형 (대규모 징집 → (강화) 탱킹 → 부활):
  전진기지(T1) → 징병국(T2) → 보급(T2) → 돌격편대(T3) → 군수공장(T4) → 통합사령부(T5)

  배치: [돌격편대][전진기지][징병국][보급][군수공장][...]

  징집 2~5기/R. 신병 3기 선택으로 물량 극대화.
  R15에 30+유닛. (강화)는 징병국 ★2~3으로.
  돌격편대: (강화)=2기 카운트. 실20기+(강화)5기→카운트25=ATK+25%.
  통합사령부: (강화) 소수가 부활 → 전열 유지.
  군수공장: 신병 3기 징집 = 카운터 +3 → 테라진 빠르게 축적.

공유 카드: 보급(T2), 전술사령부(T3)
정예 전용: 훈련소(T1), 군사학교(T2), 특수작전대(T4) = 3장
물량 전용: 전진기지(T1), 징병국(T2), 돌격편대(T3), 군수공장(T4) = 4장
양쪽 캡스톤: 통합사령부(T5)
분기 시점: T1 (훈련소 vs 전진기지 선택)

하이브리드 벌점:
  훈련소는 유닛 추가 없음(임계점 3기만) → 물량 기여 미미.
  전진기지는 훈련 없음 → 계급 기여 0.
  8장 필드에서 양쪽 핵심 모두 넣으면 T3~T4 공백.
```

## ★2 합성 유인 검증

```
훈련소:     ★1=자기+1/R+인접+1, ★2=자기+2/R+인접+1. 자기만 가속.           ✓ 가속
전진기지:   ★1=징집2기, ★2=2기+인접1기. 슬롯 절약 + 물량 확대.              ✓ 확장
군사학교:   ★1=추가+1(2/R), ★2=+1+ATK2%(3/R). 수치+새 효과.              ✓ 수치
징병국:     ★1=추가1기(2/R), ★2=2기(1기 강화)(3/R). 수치+새효과.            ✓ 수치
보급:       ★1=1골드+0.5/카드, ★2=2골드+1/카드+패배전액. 수치+안정.          ✓ 수치
전술사령부:  ★1=계급2%+유닛0.5%, ★2=3%+0.8%. 수치.                         ✓ 수치
돌격편대:   ★1=유닛×1%, ★2=×1.5%+조건완화. 수치.                          ✓ 수치
특수작전대:  ★1=계급8→리더, ★2=계급6→리더+강화. 조건 완화.                  ✓ 조건
군수공장:   ★1=카10→테라진1+ATK%, ★2=카8→테라진1+레어25%할인+ATK%, ★3=카6→테라진3+에픽4테라진상시+ATK%. ✓ 수치
통합사령부:  ★1=HP50%부활, ★2=HP75%+ATK15%, ★3=훈련+2+10회(매부활HP100%+ATK+15%누적+방어막20%)+계급8+전원부활. ✓ 수치
```

## 계급 흐름도

```
계급 생성:    훈련소(자기+인접, +1/R), 특수작전대(자기+인접, +1/R), 통합사령부(전체, +1/R; ★3는 +2/R)
계급 증폭:    군사학교(훈련 반응→추가 +1, 2~3/R)
계급→(강화):  임계점 3(1기), 5(+1기), 8(전원). 훈련소 배치유닛은 (강화).
계급→보너스:  시스템 임계점(3/5/8), 훈련소 임계점 배치, 특수작전대 리더
계급→후반:    계급 15 임계점 (훈련소 ★3 곱연산, 전술사령부 ★3 AS, 특수작전대 ★3 리더 확산)
계급→경제:    보급 ★3(이 카드 계급 5+→테라진 1/R)
계급→방어:    전술사령부(이 카드 계급 × 2% 방어막)

징집 생성:    전진기지(2기/R, 3택1 기수차등)
징집 증폭:    징병국(징집 반응→추가 1기 무작위, 2/R)
징집→(강화):  징병국 ★2(1기/R), ★3(전원)
징집→전투:    돌격편대(유닛 수 × ATK%, (강화)=2기 카운트)
징집→경제:    군수공장(징집 횟수→카운터→테라진. 기수 무관 +1/회)

(강화)→방어:  전술사령부((강화) 유닛 추가 방어막 +5%)
(강화)→전투:  돌격편대((강화) 2기 카운트)
(강화)→부활:  통합사령부(최종 작전: (강화) 사망 시 부활)
```

## 타임라인 — 정예형

```
R1:    [훈련소] 배치. 자기 계급+1. 3유닛으로 전투. 빠듯.
R2:    계급 2. 인접에 보급 배치 → 보급도 계급+1/R 시작.
R3:    훈련소 계급 3 → ATK+15% + 보병(강화) 배치 + 신병 1기 (강화). "첫 정예 탄생."
       군사학교 합류 → 인접 카드 실질 +2/R.
R4:    전술사령부 배치(훈련소 인접). 전술사령부 계급+2/R 시작.
R5:    훈련소 계급 5(★1+1/R) → 사수(강화) + 추가 (강화). "정예 확대."
       보급 계급 5(인접+군사학교) → ★3이면 테라진 해금.
R7:    특수작전대 합류(보드 반대편). 인접 2~3장에 훈련 전파 시작.
R8:    훈련소 ★2: 계급 16 → 계급 15 달성! → ATK ×1.3.
       전술사령부 ★2: 계급 ~14. AS 보너스 임박.
       훈련소 계급 8 → 워커 배치! 전원 (강화)! + 리더.
R10:   통합사령부 ★3 합류. 전체 +2/R(계급 가속 2배). (강화) 전원 3회 부활.
       "전원 3회 부활 × 1회차 ATK+50%+AS×0.75 = 죽을수록 강해지는 군대."
       통합사령부 계급 0 → +2/R. R4(계급8) → 전 유닛 부활 해금.
R12:   전술사령부 계급 15 → AS ×0.85. 특수작전대 계급 15 → 리더 버프 확산.
       통합사령부 계급 8 달성(R14) → 전체 군대 유닛 3회 부활. 사실상 불패.
```
