# 리플레이 시스템 (런 외부 선택)

게임 시작 전 선택하는 3가지 시스템. N × M × 난이도 조합으로 리플레이성 확보.

## 1. 플레이특성 (커맨더)

런 시작 시 1명 선택. 게임 규칙 자체를 변경하여 최적 전략이 달라짐.

| 커맨더 | 특성 | 유도하는 전략 |
|--------|------|-------------|
| 🎲 도박꾼 | 리롤 50% 무료. ★3 합성 시 골드 절반 환급 | ★3 올인, 공격적 리롤 |
| 🌱 양성가 | 유닛추가 유닛 HP +5. 유닛 15+ 카드 발동상한 +1 | 물량 극대화 |
| ⚒️ 단조사 | 업그레이드 슬롯 +1 (6개). 커먼 자원비용 -1 | 업그레이드 몰빵 정예 |
| 📐 전략가 | 필드 시작 +1 (7장). 인접 조건 2칸으로 확장 | 넓은 필드, 긴 체인 |
| 📚 수집가 | 서로 다른 카드 종류 1개당 전체 ATK +1 | 다양한 카드, 합성 안 함 |
| ⚔️ 약탈자 | 전투 승리 시 골드 +2. 3연승마다 커먼 업그레이드 | 연승 유지, 전투 중심 |
| 💰 연금술사 | 판매 손실 없음. 매 라운드 골드 +1 | 자유 실험, 빌드 전환 |

설계 검증: 수집가(합성 안 함) vs 도박꾼(★3 올인) = **완전히 반대 전략**. 같은 게임이 다르게 느껴짐.

## 2. 장신구/부적

커맨더와 조합하여 세부 전술을 조절. 커맨더가 "방향", 부적이 "세기".
상세: [talismans.md](talismans.md)

**성장 부적:**
- 🔥 부싯돌: 매 라운드 첫 성장 이벤트 효과량 2배
- 💧 수은 방울: 강화 효과 발동 시 효과량 +25%
- 🥚 깨진 알: ★2 이상 카드의 유닛 추가 시 +1기

**전투 부적:**
- 💀 금간 해골: 유닛 첫 치사 피격 HP 1 생존 (유닛당 1회)
- 🥁 전쟁 북: 아군 유닛 수 > 적이면 적 전체 ATK -10%

**경제 부적:**
- 👁️ 유리 눈: 리롤 시 보유 카드 등장 확률 ×1.15
- 🪙 양면 동전: 매 라운드 상점 카드 1장 50% 할인 + 1장 50% 할증
- 🔧 녹슨 렌치: 업그레이드 분리 가능, 테라진 50% 환급

**빌드 부적:**
- 👜 터진 자루: 업그레이드 상점 표시 +1 (2→3)
- 🔌 구리 전선: 풀슬롯 카드의 업그레이드 효과가 인접 카드에 30% 전파

**변칙 부적:**
- 🎲 황금 주사위: 보스 보상 선택지 +2 (4→6)
- 🏺 영혼 항아리: 라운드 첫 판매 시 유닛 절반을 필드 카드에 배분

## 커맨더 × 부적 핵심 시너지

```
도박꾼  × 유리 눈    = ★3 올인 + 카드 찾기 가속. 가장 직관적 조합.
도박꾼  × 황금 주사위 = 보스 6택1. ★승급 + 영구규칙 최적 선택.
양성가  × 금간 해골  = 대군 전원 1회 생존. 물량의 압도적 생존력.
양성가  × 전쟁 북    = 항상 수적 우위 → 적 ATK -10%. 물량의 전투 보상.
단조사  × 구리 전선  = 6슬롯 풀채우기 → 업그레이드 인접 전파. 정예 허브.
전략가  × 황금 주사위 = 📐구조 확률 67%. 필드 9장 가능성 대폭 증가.
수집가  × 양면 동전  = 할인 카드 다양하게 활용. "싸니까 사볼까?" 탐색.
수집가  × 영혼 항아리 = 판매로 유닛 재분배. 합성 대신 판매 성장.
약탈자  × 전쟁 북    = 연승으로 병력 유지 + 수적 우위 디버프. 눈덩이.
연금술사 × 녹슨 렌치 = 에픽 구매 + 기존 업글 분리 교체. 최적 빌드.
```

안티 시너지(-)도 의도적: 수집가 + 깨진 알 = 합성 안 하는데 ★2 조건. 어렵지만 "1장만 ★2" 고급 플레이 가능.

## 3. 난이도 (누적식)

각 단계가 이전 모든 효과에 누적.

```
난이도 1 ⬜ 신병:   기본
난이도 2 🟩 병장:   적 HP ×1.2
난이도 3 🟦 하사:   시작 골드 10 (13→10)
난이도 4 🟪 중사:   적 유닛 수 ×1.3
난이도 5 🟧 소위:   R5/R10 보스가 커먼 업그레이드 보유
난이도 6 🟥 대위:   상점 4장 (5→4). 리롤 2골드
난이도 7 ⬛ 장군:   적 ATK ×1.3. 보스 레어/에픽 업그레이드 보유
난이도 8 💀 전설:   플레이어 HP 20 (30→20)
```

난이도 5+에서 **적이 업그레이드를 보유** = 질적으로 다른 경험. 단순 숫자 인플레이션이 아님.

## 해금 (메타 진행 루프)

```
초기 해금: 커맨더 2종 (도박꾼, 양성가) + 부적 3종 + 난이도 1

업적으로 해금:
  "유닛 50+ 필드"          → 양성가 해금
  "업그레이드 10+ 장착"     → 단조사 해금
  "서로 다른 카드 7종 필드"  → 수집가 해금
  "5연승"                  → 약탈자 해금
  "한 런에 카드 10장+ 판매" → 연금술사 해금

클리어 보상:
  난이도 N 클리어 → 난이도 N+1 해금 + 전용 부적 해금

리플레이 볼륨:
  커맨더 7 × 부적 12 × 난이도 8 = 672 시작 조합
  실질 의미 있는 조합 ~200가지 × 25분 = 수백 시간
```
