# 리플레이 시스템 (런 외부 선택)

게임 시작 전 선택하는 3가지 시스템. N × M × 난이도 조합으로 리플레이성 확보.

## 1. 플레이특성 (커맨더)

런 시작 시 1명 선택. 게임 규칙 자체를 변경하여 최적 전략이 달라짐.

| 커맨더 | 특성 | 유도하는 전략 |
|--------|------|-------------|
| 🎲 도박꾼 | 리롤 50% 무료. ★3 합성 시 골드 절반 환급 | ★3 올인, 공격적 리롤 |
| 🌱 양성가 | 유닛추가 유닛 HP +5. 유닛 15+ 카드 발동상한 +1 | 물량 극대화 |
| ⚒️ 단조사 | 업그레이드 슬롯 +1 (6개). 커먼 자원비용 -1 | 업그레이드 몰빵 정예 |
| 📐 전략가 | 필드 시작 +1 (7장). 인접 조건 2칸으로 확장 | 넓은 필드, 긴 체인 |
| 📚 수집가 | 서로 다른 카드 종류 1개당 전체 ATK +1 | 다양한 카드, 합성 안 함 |
| ⚔️ 약탈자 | 전투 승리 시 골드 +2. 3연승마다 커먼 업그레이드 | 연승 유지, 전투 중심 |
| 💰 연금술사 | 판매 손실 없음. 매 라운드 골드 +1 | 자유 실험, 빌드 전환 |

설계 검증: 수집가(합성 안 함) vs 도박꾼(★3 올인) = **완전히 반대 전략**. 같은 게임이 다르게 느껴짐.

## 2. 장신구/부적

커맨더와 조합하여 세부 전술을 조절. 커맨더가 "방향", 부적이 "세기".

**체인 부적:**
- ⚡ 가속기어: "첫 체인 활성화 효과 2배"
- 🔄 잔향: "체인 10+ 달성 시 마지막 이벤트 1회 추가 발동"
- 🌿 과성장: "유닛추가 시 10% 확률로 2기 추가"

**전투 부적:**
- 🗡️ 선제타격: "전투 시작 시 아군 첫 공격 ATK ×2"
- 🛡️ 최후의보루: "아군 유닛 첫 치명타 시 HP 1로 생존 (유닛당 1회)"
- 💀 사후폭발: "아군 사망 시 주변 적에게 ATK 50% 피해"

**경제 부적:**
- 🏪 단골할인: "같은 카드 3번째 구매 시 1골드 할인"
- 📦 잉여재고: "상점에 업그레이드가 매 라운드 등장"
- 🔨 해체: "업그레이드 제거 시 전투자원 절반 환급"

**빌드 부적:**
- ⬆️ 속성계승: "★합성 시 하위 ★ 업그레이드 유지"
- 🏷️ 태그공명: "같은 태그 3종+ 필드 시 해당 태그 유닛 HP +10"
- 📈 만렙효과: "★3 카드 체인 효과가 인접→전체 아군 대상"

## 커맨더 × 부적 핵심 시너지

```
도박꾼  × 만렙효과  = ★3 빌드의 극한 (체인 전체 적용)
양성가  × 최후의보루 = 대군이 전부 1회 사망 면역 (물량 극대화)
단조사  × 잉여재고  = 매 라운드 업그레이드 → 에픽 2개 가능
전략가  × 잔향      = 긴 체인의 마지막이 2번 터짐 (폭발적)
수집가  × 태그공명  = 다양한 카드의 태그 시너지 보상
약탈자  × 선제타격  = 선공으로 연승 시작 → 눈덩이
연금술사 × 해체     = 업그레이드 자유 교체 → 실험 플레이
```

안티 시너지(-)도 의도적: 수집가 + 만렙효과 = ★3 안 만드는데 ★3 보상. 어렵지만 "1장만 ★3" 고급 플레이 가능.

## 3. 난이도 (누적식)

각 단계가 이전 모든 효과에 누적.

```
난이도 1 ⬜ 신병:   기본
난이도 2 🟩 병장:   적 HP ×1.2
난이도 3 🟦 하사:   시작 골드 10 (13→10)
난이도 4 🟪 중사:   적 유닛 수 ×1.3
난이도 5 🟧 소위:   R5/R10 보스가 커먼 업그레이드 보유
난이도 6 🟥 대위:   상점 4장 (5→4). 리롤 2골드
난이도 7 ⬛ 장군:   적 ATK ×1.3. 보스 레어/에픽 업그레이드 보유
난이도 8 💀 전설:   플레이어 HP 20 (30→20)
```

난이도 5+에서 **적이 업그레이드를 보유** = 질적으로 다른 경험. 단순 숫자 인플레이션이 아님.

## 해금 (메타 진행 루프)

```
초기 해금: 커맨더 2종 (도박꾼, 양성가) + 부적 3종 + 난이도 1

업적으로 해금:
  "유닛 50+ 필드"          → 양성가 해금
  "업그레이드 10+ 장착"     → 단조사 해금
  "서로 다른 카드 7종 필드"  → 수집가 해금
  "5연승"                  → 약탈자 해금
  "한 런에 카드 10장+ 판매" → 연금술사 해금

클리어 보상:
  난이도 N 클리어 → 난이도 N+1 해금 + 전용 부적 해금

리플레이 볼륨:
  커맨더 7 × 부적 12 × 난이도 8 = 672 시작 조합
  실질 의미 있는 조합 ~200가지 × 25분 = 수백 시간
```
