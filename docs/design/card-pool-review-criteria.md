# 테마 카드 풀 평가 Criteria

multi-review로 테마 카드 풀을 평가할 때 사용하는 4개 축.
각 축은 독립된 Critic에게 배정한다.

## A. 구조 건전성 (Structural Soundness)

| 항목 | 질문 | 측정 방법 |
|------|------|-----------|
| 필수카드 부재 | 이 카드 없이 테마가 기능하는가? | 각 카드를 1장씩 제거한 9장 덱 시뮬 (10장 중 1장 빠진 상태) |
| 체인 진입점 다중화 | 체인을 시작할 수 있는 카드가 2장 이상인가? | 라운드 시작 / 자기 발동 카드 수 |
| 티어 곡선 | T1~T5 분포가 자연스러운가? | 기준: T1:2, T2:2~3, T3:2, T4:2, T5:1 |
| 역할 커버리지 | 체인 스타터, 리액터, 이코노미, 전투 최소 1장씩 있는가? | 역할 태깅 |

## B. 서브 전략 분기 (Build Diversity)

| 항목 | 질문 | 측정 방법 |
|------|------|-----------|
| 경로 독립성 | 서브 전략 A만으로 런 완주 가능한가? B만으로? | 경로별 독립 시뮬 |
| 선택 긴장감 | A 경로 카드를 잡으면 B 경로가 약해지는가? | 카드 간 시너지/반시너지 매트릭스 |
| 분기 시점 | 드래프트 언제 분기가 결정되는가? | T1~T2에서 결정이 이상적 |
| 하이브리드 벌점 | 양쪽 다 가져가면 어느 쪽도 극대화 못하는가? | 혼합 덱 vs 집중 덱 성능 비교 |

## C. 성장 & 스케일링 건강성 (Growth Health)

| 항목 | 질문 | 측정 방법 |
|------|------|-----------|
| 성장 카드 밀도 | 다른 카드/유닛을 성장시키는 카드가 2~3장 이내인가? | 성장 효과 카드 수 (자기 성장 제외) |
| 자기 vs 외부 성장 | 자기 카드 내 성장과 외부 성장 비율이 적절한가? | 효과 대상 분류 |
| 스노우볼 제어 | 성장이 선형에 가까운가, 지수적인가? | 라운드별 스탯 곡선 시뮬 |
| 초기 생존력 | 성장 없이 T1~T2 카드만으로 라운드 3~5 버틸 수 있는가? | 초기 전투 시뮬 승률 (CP는 참고) |

## D. 플레이어 경험 (Player Experience)

| 항목 | 질문 | 측정 방법 |
|------|------|-----------|
| 드래프트 선택 의미 | 매 드래프트에서 진짜 고민이 있는가? | 카드 간 기회비용 존재 여부 |
| 리플레이 다양성 | 같은 테마 3런 시 다른 덱이 나오는가? | 최종 덱 구성 엔트로피 |
| 테마 정체성 | 이 테마만의 "느낌"이 플레이에서 체감되는가? | 키워드 차별화 (정성적) |
| 패배 보상 | 약한 런에서도 학습/발전감이 있는가? | 이코노미 회복 카드 존재 여부 |

## E. 설계 안전성 체크리스트 (Design Safety)

카드 설계 시 **반드시 사전 검증**해야 하는 항목. 과거 반복 발견된 문제 패턴.

### 메커니즘 순수성

| # | 항목 | 질문 | 예시 |
|---|------|------|------|
| E1 | 전용 카드 메커니즘 순수성 | 전용 카드(공유 아님)가 양쪽 서브 전략의 핵심 키워드를 동시에 보유하지 않는가? | 군대 훈련소가 훈련+징집 동시 보유 → 하이브리드 벌점 희석 → 훈련소에서 징집 제거 |
| E2 | 캡스톤 공정성 | T5 캡스톤이 양쪽 서브 전략 모두를 보상하는가? (한쪽만 보상하면 다른 쪽 후반 열세) | 군대 통합사령부가 "평균 계급" 기반 → 정예만 보상 → 이중 조건(계급 OR 유닛 수)으로 수정 |
| E3 | 루프 제어 | 같은 카드 2장이 서로 무한 트리거하지 않는가? [고유효과]가 필요한 곳에 명시되어 있는가? 발동 상한 N/R이 모든 ON_EVENT 리액터에 있는가? 리액터가 emit하는 이벤트가 자기 트리거 이벤트와 순환하는지 체인 그래프에서 확인했는가? | 스팀펑크 조립라인 2장 루프 → [고유효과] 추가. 포식종 적응갑각(METAMORPHOSIS→부화emit)↔탈피의방(HATCH→변태emit) 순환 → 각각 2회/R 상한으로 제어 |

### 효과 텍스트 명확성

| # | 항목 | 질문 | 예시 |
|---|------|------|------|
| E4 | 대상 명시 | 효과 대상이 "유닛" vs "카드"로 명확히 구분되어 있는가? 개량/성장은 카드 귀속 modifier인데 "유닛 개량"이라고 쓰면 혼동. | 스팀펑크 조립소 ★3 "유닛 즉시 개량" → "해당 카드 개량"으로 수정 |
| E5 | 태그 대상 범위 | 태그 지정 효과(#태엽 ATK +5%)가 카드 전체 적용인지 태그 유닛만 적용인지 명시되어 있는가? | 스팀펑크 공방 "#태엽 유닛 개량" → 카드 modifier 원칙상 태그는 조건, 개량은 카드 전체 |
| E6 | 지속 범위 명시 | 효과가 "이번 전투" / "이번 라운드" / "영구 누적" 중 어디에 해당하는지 명시되어 있는가? | 이자기 리롤 보너스 "런 누적" vs "라운드당" 미정의 → "이번 라운드 리롤 횟수" + 최대 5회분으로 확정 |

### 수치 안전성

| # | 항목 | 질문 | 예시 |
|---|------|------|------|
| E7 | ★ 진화 체감성 | ★2 수치가 SC1 스케일(ATK 1~20)에서 체감 가능한가? 전투 결과에 유의미한 차이를 만드는가? | 이자기 ★2 +2%/리롤 → 리롤 5회에도 ATK +0.48/유닛 = 체감 불가 → +5%/리롤로 상향 |
| E8 | 생성 메커니즘 대상 고정 | 유닛 생성(부화/번식 등)이 항상 의도한 유닛 타입을 생성하는가? | 포식종 부화: 변태로 유충 소모 후 "최약"이 수호충 → 부화=항상 칼날유충으로 수정 |
| E9 | 경제 보상 상한 | 골드/테라진 수입이 라운드당 최대 얼마인가? 다른 테마 경제 카드와 비교하여 과도하지 않은가? | 변태 수확 2테라진/R → 1테라진/R로 하향 |
| E10 | 경제 공식 테마 연동 | 경제 공식이 해당 테마의 핵심 자원/메커니즘과 연동되는가? 비례 보상에 floor가 명시되어 있는가? | 포식종: 유닛 수 비례(floor(유닛수/5)). 드루이드: 🌳 비례(🌳÷3). 스팀펑크: 열(🔥) 비례. 군대: ???(설계 시 결정) |
| E11 | 지수적 증가 방지 | 자기 강화 루프의 15R 내 실측 도달값을 계산했는가? 하드 상한이 없으면 물리적 한계값이 건강한지 검증했는가? 피드백 루프가 지수적으로 폭주하지 않는가? | 기생 진화: 생존당 부화 무제한→지수증가 → 최대 3기/R 상한. 세계수: 상한 없지만 floor(65/30)=2→×1.3(★1) = 15R 내 물리적 한계가 건강 |
| E12 | 자원 이전 악용 | 판매/흡수 등으로 자원(🌳, 유닛 등)을 특정 카드에 집중했을 때 정상 경로 대비 +50% 이상 효과 증가가 발생하는가? 발생하면 계수 하향 또는 조건 상향 필요. | 뿌리깊은 자: 판매로 🌳 집중 시 성장 39.6%/R → 계수 1.2%→0.8% 하향으로 26.4%/R(정상 대비 +40%) |
| E13 | 인지 부하 통일 | 유사한 역할의 상한 수치가 통일되어 있는가? | 포자 구름: AS 상한 60%, ATK 상한 50% → 모두 50%로 통일 |
| E14 | ★ 승급 단조성 | ★N+1의 모든 수치 라인이 ★N 이상인가? 효과 종류가 바뀌면(자기→전체 등) 해당 라인을 명시적으로 비교 표기했는가? | 변태 수확 ★2 ATK+5%(자기)→★3 전체+3% = 비교 어려움 → 비교 노트 부기 |

**적용 시점**: 카드 설계 완료 후, multi-review 실행 전에 E1~E14를 자가 점검.
veto 조건이 아닌 사전 체크리스트 — 위반 시 설계를 수정한 후 multi-review 진입.

## 판정 기준

| 조건 | 판정 |
|------|------|
| 4축 모두 ≥ 7 AND veto 없음 | **PASS** |
| 임의 축 veto | **VETO** — 해당 축 사유 제시 |
| 평균 ≥ 7 BUT 일부 < 7 | **MIXED** → Synthesis |
| 평균 < 7 | **FAIL** → Synthesis |

## Veto 조건 (축별)

- **A (구조)**: 필수카드가 1장이라도 존재하면 veto
- **B (분기)**: 서브 전략이 1개뿐이거나, 하이브리드가 집중보다 강하면 veto
- **C (성장)**: 외부 성장 카드가 4장 이상이면 veto
- **D (경험)**: 3런 시 동일 덱이 나오면 veto
