# 스팀펑크 카드 풀 (10장)

> 효과/수치 상세: `data/cards/steampunk.yaml` 참조 (단일 진실 소스).
> 이 문서는 설계 의도와 빌드 전략을 기술합니다.

> 유닛 풀: units-steampunk.md 참조.
> 핵심 판타지: "효율적 공장" — 카드 간 이벤트 체인으로 안정적 성장.
> 파이프라인: 제조(스폰) → 개량(ATK/HP 강화) → 전투/경제/집중.
> ⚡충전: 사용하지 않음. 체인은 제조↔개량으로 직접 연결.
>
> **제조 = 대상 카드 composition 내 무작위 유닛 1기 추가** (★3 특수 효과로 "가장 강한 유닛" 지정 가능).
> **개량 = 카드 귀속 누적 (Layer 1)**: 개량(ATK +5%)은 대상 카드의 성장 누적 modifier에 추가됨.
> 이후 제조된 유닛도 카드의 누적 개량 혜택을 받음.
> → 체인이 활발할수록 개량 누적↑, 여기에 업그레이드(Layer 2)가 곱해져 폭발하는 구조.
> 유닛이 다른 카드로 이동하면 원 카드의 개량 누적 소실 (기본 스탯으로 복귀).
> 상세: architecture.md "카드 레벨 스탯 modifier" 참조.

## 서브 전략: 확산 vs 집중

```
확산 경로 (넓은 보드):
  조립소(T1) → 공방(T1) → 조립라인(T3) → 전쟁기계(T4)
  인접 카드에 제조 분산, 유닛 수로 화력 스케일링.

집중 경로 (좁은 보드):
  용광로(T1) → 공방(T1) → 순환기(T2) → 과급기(T4) → 병기창(T5)
  자기 카드에 유닛+개량 집중, 테라진/업그레이드 스케일링.

공유 카드: 이자기(T2), 방벽(T3)
분기 시점: T1 (조립소 vs 용광로)
하이브리드 벌점: 8장 필드에서 양쪽 핵심 카드를 모두 넣으면 방벽/이자기 탈락 → 생존력/경제 약화
```

## 체인 흐름

```
제조 이벤트 소스:
  조립소 (라운드 시작) → 인접 제조         ← 확산 스타터
  용광로 (라운드 시작) → 자기 제조+개량    ← 집중 스타터
  순환기 (개량 반응) → 제조               ← 역순환 피드백
  이자기 (리롤 시) → 자기 제조            ← 독립 경제
  병기창 (판매 시) → 유닛 흡수

제조 반응:
  공방 → 개량(ATK) → 순환기 → 제조 → 공방 → ... (발동 상한으로 제어)
  조립라인 → 인접 추가 제조 (자기 제외) → 공방/조립라인 반응 → ...
  과급기 → 카운터 축적 → 테라진+자기 개량

전투 전환:
  전쟁 기계: #화기 사거리 + 공격 시 ATK 스케일링 (확산 페이오프)
  증기 방벽: 방어막으로 유닛 생존 시간 확보 (공유)
  과급기: 체인 활동 → 테라진+자기 개량 (집중 페이오프)
  병기창: 유닛 집중 → 소수 정예 군단 (집중 캡스톤)
    주의: 유닛 이동 시 원 카드의 개량 누적 소실, 병기창의 modifier 적용

T4 선택 (서브 전략 핵심 분기):
  전쟁 기계 — 확산: 유닛 수 → 사거리+화력
  태엽 과급기 — 집중: 체인 활동 → 테라진+자기 성장+업그레이드
```
