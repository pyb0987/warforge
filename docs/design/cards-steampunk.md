# 스팀펑크 카드 풀 (9장)

> 유닛 풀: units-steampunk.md 참조.
> 핵심 판타지: "효율적 공장" — 카드 간 이벤트 체인으로 안정적 성장.
> 파이프라인: 제조(스폰) → 개량(ATK/HP 강화) → 전투/경제/집중.
> ⚡충전: 사용하지 않음. 체인은 제조↔개량으로 직접 연결.

## 개요

| # | 이름 | 티어 | 역할 | 핵심 메커니즘 |
|---|------|------|------|-------------|
| 1 | 증기 조립소 | T1 | ① 체인 시작 | 라운드 시작 → 인접 제조 |
| 2 | 태엽 공방 | T1 | ③ ATK 개량 | 제조 → ATK 개량 |
| 3 | 증기 순환기 | T2 | 개량→제조 역순환 | 개량 → 추가 제조 (피드백 루프) |
| 4 | 증기 이자기 | T2 | ⑥ 경제 | 리롤 → 제조 |
| 5 | 조립 라인 | T3 | ② 체인 증폭 | 제조 → 추가 제조 |
| 6 | 증기 방벽 | T3 | ④ 방어 | 전투 시작 → 방어막 |
| 7 | 전쟁 기계 | T4 | ⑤ 전투 (화력) | #화기 사거리 스케일링 |
| 8 | 태엽 과급기 | T4 | ⑤ 전투 (경제) | 제조 카운터 → 테라진/업그레이드 |
| 9 | 제국 병기창 | T5 | ⑦⑧ 집중 capstone | 인접 유닛 흡수 → 집중 |

```

★2: 양쪽 인접 카드 유닛 1기 제조

★3: 3기씩 이동, [지속 효과] 이 카드의 유닛은 ATK/HP +50%,
```

## 체인 흐름

```
제조 이벤트 소스:
  조립소 (라운드 시작) → 제조
  순환기 (개량 반응) → 제조      ← 역순환 피드백
  이자기 (리롤 시) → 제조
  병기창 (라운드 시작) → 제조 (유닛 이동)

제조 반응:
  공방 → 개량(ATK) → 순환기 → 제조 → 공방 → ... (발동 상한으로 제어)
  조립라인 → 추가 제조 → 공방/조립라인 반응 → ...
  과급기 → 카운터 축적 → 테라진/업그레이드

전투 전환:
  전쟁 기계: #화기 사거리 + 공격 시 ATK 스케일링
  증기 방벽: 방어막으로 유닛 생존 시간 확보
  과급기: 체인 활동 → 업그레이드 자원 변환
  병기창: 유닛 집중 → 소수 정예 군단

T4 선택:
  전쟁 기계 — 화력 특화 (사거리, ATK 스케일)
  태엽 과급기 — 장기 투자 (테라진, 업그레이드)
```
