# 스팀펑크 카드 풀 (9장)

> 강화 수치는 기본 % (원래 스탯 기준). 곱연산은 명시적으로 표기.
> 카드 태그 1~3개 (유닛 태그와 별도).

## 개요

| # | 이름 | 티어 | 트리거 | 핵심 역할 |
|---|------|------|--------|----------|
| 1 | 증기 조립소 | T1 | 라운드 시작 | 체인 시작점 (제조+⚡충전) |
| 2 | 기어 공방 | T1 | ⚡충전 시 | 기본 개량 reactor |
| 3 | 대장간 | T2 | 개량 시 | 개량 증폭 amplifier |
| 4 | 증기 이자기 | T2 | 리롤할 때마다 | 경제 연동 — 리롤=성장 |
| 5 | 조립 라인 | T3 | 제조 시 | 제조→개량 연결 |
| 6 | 시한 보일러 | T3 | 라운드 시작 (3R 체류 조건) | 인내 보상 — 느린 폭탄 |
| 7 | 전쟁 기계 | T4 | ⚔공격 실행 시 | 전투 중 영구 개량 |
| 8 | 과부하 코일 | T4 | 라운드 시작 (비전투원) | 전투 포기→체인 강화 |
| 9 | 제국 공장 | T5 | 라운드 시작 | capstone — 전체 제조+개량+충전 |

## 상세

```
[증기 조립소] T1
카드 태그: #스팀펑크 #생산
트리거: 라운드 시작
효과: 인접 카드 유닛 1기 제조, 인접 카드 ⚡충전
★2: +자신 #기계 유닛 개량(ATK +20%)
★3: 양쪽 인접 카드 제조, 전체 아군 ⚡충전, 제조된 유닛 즉시 개량(ATK +20%, HP +20%)

[기어 공방] T1
카드 태그: #스팀펑크 #강화
트리거: ⚡충전 시
효과: 아군 카드 내 #기어 유닛 전체 개량(ATK +25%)
발동: 최대 2/R
★2: +#스프링도 대상, 최대 3/R
★3: 개량 시 인접 카드 ⚡충전 (개량→충전 브릿지), 최대 4/R

[대장간] T2
카드 태그: #스팀펑크 #강화
트리거: 개량 시
효과: 개량된 카드 내 #금속 유닛 추가 개량(ATK +20%)
발동: 최대 2/R
★2: #금속+#기어도 대상, 최대 3/R
★3: 추가 개량 시 대상 카드 유닛 1기 제조 (개량→제조 브릿지), 최대 4/R

[증기 이자기] T2
카드 태그: #스팀펑크 #경제
트리거: 리롤할 때마다
효과: 자신 유닛 전체 개량(HP +15%)
발동: 최대 3/R
★2: 개량(ATK +10%, HP +15%), 최대 4/R
★3: 리롤 시 50% 확률로 리롤 비용 환급
→ 리롤 중독이 빌드. 도박꾼 커맨더 시너지.

[조립 라인] T3
카드 태그: #스팀펑크 #생산
트리거: 제조 시
효과: 제조된 카드 내 #부품 유닛 전체 개량(ATK +20%)
발동: 최대 3/R
★2: 개량(ATK +20%, HP +10%), 최대 4/R
★3: 3회째 발동 시 자신 유닛 2기 제조 (제조 루프)

[시한 보일러] T3
카드 태그: #스팀펑크 #시간
트리거: 라운드 시작 (조건: 필드에 3라운드+ 체류)
효과: 자신 #동력 유닛 전체 개량(ATK +40%, HP +30%)
★2: 개량(ATK +50%, HP +40%)
★3: 5라운드 체류 달성 시 전체 아군 유닛 수 2배 (1회 임계점)
→ R3 배치 → R6부터 매 라운드 강한 강화, R8에 군대 2배 폭발.
→ 늦게 놓을수록 약함. 일찍 놓으면 초반 전투력 빈칸.

[전쟁 기계] T4
카드 태그: #스팀펑크 #전투
트리거: ⚔공격 실행 시
효과: 자신 #무기 유닛 개량(ATK +15%, 영구)
발동: 무제한
★2: 영구 개량 + 관통 부여
★3: #무기 → #기계 전체

[과부하 코일] T4
카드 태그: #스팀펑크 #과부하
트리거: 라운드 시작
효과: 이 카드는 전투에 참여하지 않는다. 전체 아군 카드 ⚡충전 2회씩.
★2: ⚡충전 3회씩
★3: 이 카드가 필드에 있는 한 모든 개량 효과의 % +10% 추가
→ 기어 공방의 ATK +25%가 ATK +35%로 증가하는 글로벌 버프.

[제국 공장] T5
카드 태그: #스팀펑크 #생산 #강화
트리거: 라운드 시작
효과: 전체 아군 카드 유닛 1기씩 제조, 자신 #기계 유닛 개량(ATK +30%), 전체 아군 ⚡충전
★2: 제조 2기씩, 개량(ATK +30%, HP +25%)
★3: 제조된 유닛 즉시 개량(ATK +30%, HP +30%), ⚡충전 2회씩
```

## 키워드 커버리지

```
            생성                         반응
제조        조립소(T1), 대장간★3,         조립 라인(T3)
            조립라인★3, 제국공장(T5)

개량        기어공방(T1), 조립라인(T3),    대장간(T2)
            시한보일러(T3), 전쟁기계(T4),
            과부하코일★3, 제국공장(T5)

⚡충전      조립소(T1), 과부하코일(T4),    기어공방(T1)
            기어공방★3, 제국공장(T5)
```

## 설계 공간 활용 분포

```
기본 체인:    4장 (조립소, 기어공방, 대장간, 조립라인)
경제 연동:    1장 (증기이자기 — 리롤=성장)
시간/누적:    1장 (시한보일러 — 체류 시간 보상 + 임계점 폭발)
전투 성장:    1장 (전쟁기계 — 전투 중 영구 개량)
대가형:       1장 (과부하코일 — 전투 포기→체인 강화)
capstone:     1장 (제국공장 — 올인원)
```
