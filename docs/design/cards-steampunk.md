# 스팀펑크 카드 풀 (9장)

> 유닛 풀: units-steampunk.md 참조.
> 성장(개량): 드문 효과 (9장 중 3장, 5~8%). 업그레이드가 스탯 성장 주 축.
> ⚡충전: 사용하지 않음. 체인은 제조↔개량으로 직접 연결.

## 개요

| # | 이름 | 티어 | 트리거 | 핵심 역할 |
|---|------|------|--------|----------|
| 1 | 증기 조립소 | T1 | 라운드 시작 | 체인 시작점 (제조) |
| 2 | 태엽 공방 | T1 | 제조 시 | 제조→개량 연결 |
| 3 | 증기 대장간 | T2 | 개량 시 | 개량 증폭 |
| 4 | 증기 이자기 | T2 | 리롤할 때마다 | 경제 연동 — 리롤=제조 |
| 5 | 조립 라인 | T3 | 다른 카드 제조 시 | 제조 증폭 |
| 6 | 시한 보일러 | T3 | 라운드 시작 (3R 체류) | 인내 보상 — 조건부 개량 |
| 7 | 전쟁 기계 | T4 | ⚔공격 실행 시 | 전투 중 버프 |
| 8 | 과부하 코일 | T4 | 라운드 시작 (비전투원) | 전투 포기→인접 추가 발동 |
| 9 | 제국 공장 | T5 | 라운드 시작 | capstone — 전체 제조 |

## 상세

```

구성: 태엽 거미 ×2, 증기 쥐 ×1           (3유닛, ATK 6, HP 55)

트리거: 라운드 시작
효과: 인접 카드 유닛 1기 제조
★2: 양쪽 인접 카드 제조
★3: 양쪽 인접 제조, 제조된 유닛 즉시 개량(ATK +5%)

[태엽 공방] T1
구성: 태엽 거미 ×2, 톱날 구동체 ×1        (3유닛, ATK 8, HP 80)
카드 태그: #스팀펑크 #강화
트리거: 제조 시
효과: 제조된 카드 내 #태엽 유닛 전체 개량(ATK +5%)
발동: 최대 2/R
★2: +#전기도 대상, 개량(ATK +7.5%)
★3: 개량 시 인접 카드 유닛 1기 제조 (개량→제조 브릿지), 최대 4/R

[증기 대장간] T2
구성: 철갑 게 ×1, 톱날 구동체 ×1          (2유닛, ATK 9, HP 110)
카드 태그: #스팀펑크 #강화
트리거: 개량 시
효과: 개량된 카드 내 #기갑 유닛 추가 개량(ATK +5%)
발동: 최대 2/R
★2: #기갑+#증기도 대상, 최대 3/R
★3: 추가 개량 시 대상 카드 유닛 1기 제조 (개량→제조 브릿지), 최대 4/R

[증기 이자기] T2
구성: 태엽 정찰기 ×2, 증기 쥐 ×1          (3유닛, ATK 6, HP 65)

트리거: 리롤할 때마다
효과: 자신 유닛 1기 제조
발동: 최대 3/R
★2: 인접 카드에도 유닛 1기 제조
★3: 리롤 시 50% 확률로 리롤 비용 환급

카드 태그: #스팀펑크 #생산
트리거: 다른 카드가 제조 시 (자신 제외)
효과: 자신 유닛 1기 제조, 인접 카드 유닛 1기 제조
발동: 최대 3/R
★2: 제조 2기씩, 최대 4/R
★3: 3회째 발동 시 전체 아군 유닛 1기씩 제조

[시한 보일러] T3
구성: 증기 전갈 ×1, 증기 쥐 ×2            (3유닛, ATK 10, HP 85)
카드 태그: #스팀펑크 #시간
트리거: 라운드 시작 (조건: 필드에 3라운드+ 체류)
효과: 자신 #증기 유닛 전체 개량(ATK +8%, HP +8%)
★2: +인접 카드 유닛 1기 제조
★3: 5라운드 체류 달성 시 전체 아군 유닛 수 2배 (1회 임계점)

[전쟁 기계] T4
구성: 증기 터렛 ×1, 황동 포차 ×1, 전기 드론 ×1  (3유닛, ATK 17, HP 85)
카드 태그: #스팀펑크 #전투
트리거: ⚔공격 실행 시

★3: #화기 → #기계 전체

[과부하 코일] T4
구성: 전기 드론 ×2                        (2유닛, ATK 8, HP 40)
카드 태그: #스팀펑크 #과부하

[제국 공장] T5
```

## 체인 흐름 (⚡충전 없이)

```
조립소 → 제조 이벤트 → 태엽공방 (개량) + 조립라인 (추가 제조)
                          ↓                    ↓
                     개량 이벤트          제조 이벤트
                          ↓                    ↓
                     대장간 (추가 개량)    태엽공방 (추가 개량)
```

## 성장 보유 카드 (3/9)

```
태엽 공방:    제조 → 개량(ATK +5%)
증기 대장간:  개량 → 추가 개량(ATK +5%)
시한 보일러:  3R 체류 → 개량(ATK +8%, HP +8%)
```
