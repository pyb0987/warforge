# 테마 & 카드 풀 구성

## 카드 풀 배분 (50장)

```
중립:       14장  ← 굿스터프 덱 가능. 테마 간 접착제.
스팀펑크:    9장  ← 기계 군단. 공장 생산라인.
드루이드:    9장  ← 자연 군단. 생명의 폭주.
포식종:        9장  ← 벌레/파충 군단. 둥지의 무리.
군대:        9장  ← 정규군. 편성과 정예화.
```

## 4테마 메카니즘 프레임워크 (시뮬레이션 검증 기반)

```
          | 유닛 수 | 개체 강도 | 성장 방식       | 핵심 판타지
─────────|──────|──────--|──────────────|──────────────────
포식종    |  최다  |  최약    | 부화(대량 증식)   | "끝없이 밀려오는 떼"
스팀펑크   |  중    |  중     | 제조→개량 체인    | "효율적 공장"
군대      |  중    |  강     | 징집+훈련(강화)   | "훈련된 정예 부대"
드루이드   |  소    |  최강   | 번식+성장(시간)   | "고대의 거목이 깨어남"
```

### 테마별 파워 소스

```
포식종:   부화로 유충이 쏟아짐 → 일부가 변태로 상위 개체 진화.
          수의 압도 + 진화 잭팟. 약한 개체가 많되, 변태 시 폭발적.

스팀펑크:  카드 간 이벤트 체인 (제조→개량→제조→...).
          체인이 길수록 강함. 배치 순서가 전략. 안정적, 예측 가능.

군대:     징집으로 이미 강한 병력 확충 + 훈련으로 전체 강화.
          유닛 수 비례 집단 시너지. 전투 중 진형/버프 핵심.

드루이드:  소수 유닛이 매 라운드 자동 성장. 번식도 느리지만 강한 개체 복제.
          오래 키울수록 압도적. 인내가 전략. 체인 없이도 성장.
```

### 테마별 키워드 차이

```
유닛 추가:
  제조 (스팀펑크):  대상 카드의 랜덤 유닛 추가. 외부 체인 의존.
  번식 (드루이드):  자기 카드 내 같은 유닛 복제. 자급자족.
  부화 (포식종):   대량 스폰. 가장 약한 유닛 위주. 양 극대화.
  징집 (군대):    카드 구성과 무관한 별도 풀에서 추가. 유연성.

강화:
  개량 (스팀펑크):  스탯 % 증가 (유닛 유지). 이벤트 체인 생성.
  성장 (드루이드):  시간 기반 자동 스탯 증가. 체류할수록 강해짐.
  변태 (포식종):   약한 유닛 제거 → 상위 유닛으로 교체. 양→질.
  훈련 (군대):    유닛 수에 비례하는 전체 강화. 많을수록 세게.
```

## 테마별 정체성

```
┌─ 스팀펑크 ─────────────────────────────────────┐
│  판타지: 효율적 공장. 기계 생산 라인.            │
│  유닛:   중간 물량, 중간 개체 (평균 ATK 4.2)     │
│  유닛 추가: "제조" (생산라인에서 유닛 생산)       │
│  강화:     "개량" (기계 튜닝/업그레이드)          │
│  플레이:   카드 간 이벤트 체인. 배치 순서 최적화.  │
│  범용 태그: #기계 #소형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 드루이드 ─────────────────────────────────────┐
│  판타지: 고대의 숲이 깨어남. 소수의 거목.         │
│  유닛:   소수 정예, 최강 개체 (평균 ATK 7.5)     │
│  유닛 추가: "번식" (같은 유닛 복제, 자급자족)     │
│  강화:     "성장" (시간 기반 자동 스탯 증가)      │
│  플레이:   느리지만 압도적 후반. 인내 보상.       │
│  범용 태그: #생체 #중형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 포식종 ─────────────────────────────────────────┐
│  판타지: 끝없이 밀려오는 벌레 떼.                 │
│  유닛:   최다 물량, 최약 개체 (평균 ATK 3.3)     │
│  유닛 추가: "부화" (대량 스폰, 약한 유닛 위주)    │
│  강화:     "변태" (약한 유닛 소모→상위 개체 변환)  │
│  플레이:   물량 러시 + 일부 진화. 양→질 트레이드.  │
│  범용 태그: #생체 #소형~#중형                    │
└────────────────────────────────────────────────┘

┌─ 군대 ─────────────────────────────────────────┐
│  판타지: 훈련된 정예 부대. 조직적 전투.           │
│  유닛:   중간 물량, 강한 개체 (평균 ATK 5.9)     │
│  유닛 추가: "징집" (별도 풀에서 유닛 추가)        │
│  강화:     "훈련" (유닛 수 비례 전체 강화)        │
│  플레이:   물량+질 균형. 전투 중 진형/버프 핵심.   │
│  범용 태그: #생체+#기계 혼합                     │
└────────────────────────────────────────────────┘

┌─ 중립 ─────────────────────────────────────────┐
│  판타지: 테마에 속하지 않는 범용 카드             │
│  키워드:  Layer 1 결과 범주 직접 사용             │
│           ("유닛 추가 시", "강화 시" — 테마 무관)  │
│  플레이:   어떤 테마에든 들어감. 굿스터프 덱 코어  │
│  범용 태그: 혼합                                 │
└────────────────────────────────────────────────┘
```

## 테마별 키워드와 Layer 구조

```
Layer 1 (결과 범주, 테마 무관)         Layer 2 (키워드, 테마별)
───────────────────────────         ─────────────────────────
🔧 유닛 추가                   ←── 제조(스팀) / 번식(드루) / 부화(포식종) / 징집(군대)
📈 강화                        ←── 개량(스팀) / 성장(드루) / 변태(포식종) / 훈련(군대)
⚡ 충전 (테마 무관)             ←── 모든 테마 공유. 발동 상한 +1.
⬆ 합성                        ←── 시스템 이벤트 (테마 무관)
⚔ 공격 / 💀 사망 / 🛡 버프    ←── 전투 이벤트 (테마 무관)

키워드 독립 원칙:
  "제조 시" → 제조에만 반응 (번식/부화/징집에 반응 안 함)
  "유닛 추가 시" → 모든 테마의 유닛 추가에 반응 (결과 범주)
  → 테마 내 시너지 vs 테마 간 시너지의 선택

중립 카드:
  Layer 1 결과 범주를 직접 사용 ("유닛 추가 시", "강화 시")
  → 어떤 테마 카드의 키워드에도 반응 = 범용 접착제
```

## 빌드 아키타입 (테마 기반)

```
순수 테마:      스팀펑크 9장 + 중립 보조 → 테마 내 키워드 시너지 극대화
굿스터프:       중립 14장 중심 + 테마 1~2장 → Layer 1 반응으로 범용 시너지
크로스 테마:    스팀펑크 + 포식종 혼합 → "제조로 유닛 추가 → 포식종 카드가 유닛 추가에 반응"
                Layer 1 결과 범주가 다른 테마를 연결
```
