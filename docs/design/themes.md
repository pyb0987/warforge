# 테마 & 카드 풀 구성

## 카드 풀 배분 (50장)

```
중립:       14장  ← 굿스터프 덱 가능. 테마 간 접착제.
스팀펑크:    9장  ← 기계 군단. 공장 생산라인.
드루이드:    9장  ← 자연 군단. 생명의 폭주.
충군:        9장  ← 벌레/파충 군단. 둥지의 무리.
군대:        9장  ← 정규군. 편성과 정예화.
```

## 테마별 정체성

```
┌─ 스팀펑크 ─────────────────────────────────────┐
│  판타지: 공장이 찍어내는 기계 군단               │
│  유닛 추가: "제조" (생산라인에서 유닛 생산)       │
│  강화:     "개량" (기계 튜닝/업그레이드)          │
│  플레이:   안정적·예측 가능한 성장               │
│  범용 태그: #기계 #소형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 드루이드 ─────────────────────────────────────┐
│  판타지: 숲이 살아나 뒤덮는다                    │
│  유닛 추가: "번식" (식물/생물이 퍼져나감)         │
│  강화:     "성장" (자연의 힘으로 강해짐)          │
│  플레이:   유닛 수 폭발, 개체는 약함. 물량 극대화 │
│  범용 태그: #생체 #소형                          │
└────────────────────────────────────────────────┘

┌─ 충군 ─────────────────────────────────────────┐
│  판타지: 둥지에서 쏟아지는 벌레/파충류 무리       │
│  유닛 추가: "부화" (알에서 유닛이 깨어남)         │
│  강화:     "변태" (탈피하여 상위 형태로 변환)      │
│  플레이:   초반 약→변태로 질적 도약. ★3 = 완전변태│
│  범용 태그: #생체 #소형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 군대 ─────────────────────────────────────────┐
│  판타지: 병력 충원과 정예화                      │
│  유닛 추가: "징집" (새 병사 모집)                 │
│  강화:     "훈련" (반복 훈련으로 정예화)          │
│  플레이:   균형잡힌 전투력. 전투 특화 (업그레이드↑)│
│  범용 태그: #생체 #소형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 중립 ─────────────────────────────────────────┐
│  판타지: 테마에 속하지 않는 범용 카드             │
│  키워드:  Layer 1 결과 범주 직접 사용             │
│           ("유닛 추가 시", "강화 시" — 테마 무관)  │
│  플레이:   어떤 테마에든 들어감. 굿스터프 덱 코어  │
│  범용 태그: 혼합                                 │
└────────────────────────────────────────────────┘
```

## 테마별 키워드와 Layer 구조

```
Layer 1 (결과 범주, 테마 무관)         Layer 2 (키워드, 테마별)
───────────────────────────         ─────────────────────────
🔧 유닛 추가                   ←── 제조(스팀) / 번식(드루) / 부화(충군) / 징집(군대)
📈 강화                        ←── 개량(스팀) / 성장(드루) / 변태(충군) / 훈련(군대)
⚡ 충전 (테마 무관)             ←── 모든 테마 공유. 발동 상한 +1.
⬆ 합성                        ←── 시스템 이벤트 (테마 무관)
⚔ 공격 / 💀 사망 / 🛡 버프    ←── 전투 이벤트 (테마 무관)

키워드 독립 원칙:
  "제조 시" → 제조에만 반응 (번식/부화/징집에 반응 안 함)
  "유닛 추가 시" → 모든 테마의 유닛 추가에 반응 (결과 범주)
  → 테마 내 시너지 vs 테마 간 시너지의 선택

중립 카드:
  Layer 1 결과 범주를 직접 사용 ("유닛 추가 시", "강화 시")
  → 어떤 테마 카드의 키워드에도 반응 = 범용 접착제
```

## 빌드 아키타입 (테마 기반)

```
순수 테마:      스팀펑크 9장 + 중립 보조 → 테마 내 키워드 시너지 극대화
굿스터프:       중립 14장 중심 + 테마 1~2장 → Layer 1 반응으로 범용 시너지
크로스 테마:    스팀펑크 + 충군 혼합 → "제조로 유닛 추가 → 충군 카드가 유닛 추가에 반응"
                Layer 1 결과 범주가 다른 테마를 연결
```
