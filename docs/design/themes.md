# 테마 & 카드 풀 구성

## 카드 풀 배분 (54장)

```
중립:       14장  ← 굿스터프 덱 가능. 테마 간 접착제.
스팀펑크:   10장  ← 기계 군단. 공장 생산라인.
드루이드:   10장  ← 자연 군단. 생명의 폭주.
포식종:     10장  ← 벌레/파충 군단. 둥지의 무리.
군대:       10장  ← 정규군. 편성과 정예화.
```

## 4테마 메카니즘 프레임워크 (시뮬레이션 검증 기반)

```
          | 유닛 수 | 개체 강도 | 성장 방식       | 핵심 판타지
─────────|──────|──────--|──────────────|──────────────────
포식종    |  최다  |  최약    | 부화(대량 증식)   | "끝없이 밀려오는 떼"
스팀펑크   |  중    |  중     | 제조→개량 체인    | "효율적 공장"
군대      |  중    |  강     | 징집+훈련(강화)   | "훈련된 정예 부대"
드루이드   |  소    |  최강   | 번식+성장(시간)   | "고대의 거목이 깨어남"
```

### 테마별 파워 소스

```
포식종:   부화로 유충이 쏟아짐 → 일부가 변태로 상위 개체 진화.
          수의 압도 + 진화 잭팟. 약한 개체가 많되, 변태 시 폭발적.

스팀펑크:  카드 간 이벤트 체인 (제조→개량→제조→...).
          체인이 길수록 강함. 배치 순서가 전략. 안정적, 예측 가능.

군대:     징집으로 이미 강한 병력 확충 + 훈련으로 전체 강화.
          유닛 수 비례 집단 시너지. 전투 중 진형/버프 핵심.

드루이드:  소수 유닛이 매 라운드 자동 성장. 번식도 느리지만 강한 개체 복제.
          오래 키울수록 압도적. 인내가 전략. 체인 없이도 성장.
```

### 테마별 고유 메커니즘

```
          | 고유 엔진       | 고유 자원        | 핵심 결정          | 재미의 축
─────────|──────────────|──────────────|─────────────────|──────────────
스팀펑크   | 이벤트 체인     | 체인 이벤트(무형) | 배치 순서 최적화    | 연쇄 반응의 쾌감
드루이드   | 나무 토큰 시스템 | 나무(비전투 토큰)  | 나무: 성장 vs 유닛  | 축적의 쾌감
포식종    | 유닛=자원       | 약한 유닛 자체    | 물량 vs 진화        | 변환의 쾌감
군대      | 계급 임계점     | 계급 카운터       | 임계점 도달+징집 선택 | 달성의 쾌감
```

### 테마별 키워드 — 고유 메커니즘

```
🔧 유닛 추가 (Layer 1):

  제조 (스팀펑크):
    대상: 자기 카드 또는 다른 카드 (카드 효과에 따라 다름)
    유닛 선택: 대상 카드의 랜덤 유닛
    부가 효과: 이벤트 체인 emit (다른 카드가 반응 가능)
    → "공장이 부품을 찍어낸다" (랜덤 + 체인이 핵심, 대상은 유동적)

  번식 (드루이드):
    대상: 인접 카드 (복제 유닛은 대상 카드에 배치)
    유닛 선택: 대상 카드의 가장 강한 유닛(CP 기준) 복제
    부가 효과: 나무 N개 소모. 복제 유닛은 대상 카드의 성장 modifier 즉시 적용.
    → "뿌리가 뻗어 인접 숲의 생명을 키운다" (핵심 카드의 유닛 보충)

  부화 (포식종):
    대상: 자기 카드만
    유닛 선택: 가장 약한 유닛
    부가 효과: 1회에 2~3기 대량 스폰
    → "유충이 쏟아진다" (양의 압도)

  징집 (군대):
    대상: 자기 카드
    유닛 선택: 별도 징집 풀에서 3택1 (플레이어 선택)
    부가 효과: 즉시 전투 버프(ATK +10%, 이번 전투)
    → "훈련된 병력이 배치된다" (선택권 + 즉시 전투력)

📈 강화 (Layer 1):

  개량 (스팀펑크):
    대상: 태그 필터 가능, 다른 카드 대상 가능
    내용: ATK만
    부가 효과: 이벤트 체인 emit
    → "부품을 튜닝한다" (정밀 강화 + 체인)

  성장 (드루이드):
    대상: 카드 전체, 자기 카드만
    내용: ATK + HP 동시
    부가 효과: 나무 수에 비례하여 강화량 증가 (나무 N개 × 성장 계수)
    → "뿌리가 깊어진다" (나무가 많을수록 1회 성장 가치↑)

  변태 (포식종):
    대상: 자기 카드
    내용: 약한 유닛 N기 소모 → 상위 유닛 1기로 교체
    부가 효과: 유닛 수 감소, 개체 질 상승
    → "유충이 성충이 된다" (양→질 변환)

  훈련 (군대):
    대상: 카드 전체
    내용: 카드의 계급 +1
    부가 효과: 계급 임계점 달성 시 질적 보너스 해금 (ATK/AS/리더 유닛)
    → "훈련을 거듭할수록 정예가 된다" (이산 임계점)
```

### 고유 자원 상세

```
나무 (드루이드):
  카드별 내부 카운터. 비전투. 카드 UI에 "🌳 N" 표시.
  생성: 카드 효과로 자기 카드에 +N (일부 효과는 인접/전체 카드에 기부)
  소모: 번식 시 N개 소모 → 인접 카드에 최강 유닛 복제
  스케일링: "N성장" = 이 카드의 🌳 × N% ATK+HP 증강 (연속 스케일링)
  활용: 성장(🌳 읽기), 번식(🌳 소모), 전투 방어막(🌳 비례), 경제(🌳 임계점)
  결정: 나무 축적(성장 가속/방어막) vs 소모(인접 카드 번식)

계급 (군대):
  카드 레벨 카운터. 전투 불참.
  생성: 훈련 발동 시 +1
  소모: 소모 없음 (누적 전용)
  스케일링: 임계점 달성 시 보너스 해금 (이산 스케일링)
    계급 3: ATK +15%
    계급 5: ATK +25%, AS 보너스
    계급 8: ATK +35%, AS 보너스, 리더 유닛 생성
  결정: 임계점까지 얼마나 빨리 도달하느냐 (훈련 카드 편성)

(스팀펑크/포식종은 별도 자원 토큰 없음 — 이벤트/유닛 자체가 자원)
```

## 테마별 정체성

```
┌─ 스팀펑크 ─────────────────────────────────────┐
│  판타지: 효율적 공장. 기계 생산 라인.            │
│  유닛:   중간 물량, 중간 개체 (평균 ATK 4.2)     │
│  고유 엔진: 이벤트 체인 (카드 간 연쇄 반응)      │
│  제조: 다른 카드에 랜덤 유닛 + 체인 emit         │
│  개량: 태그 필터 ATK + 체인 emit                │
│  플레이: 배치 순서 최적화. 체인이 길수록 강함.    │
│  범용 태그: #기계 #소형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 드루이드 ─────────────────────────────────────┐
│  판타지: 고대의 숲이 깨어남. 소수의 거목.         │
│  유닛:   소수 정예, 최강 개체 (평균 ATK 7.5)     │
│  고유 엔진: 나무 토큰 시스템 (비전투 자원 축적)   │
│  번식: 나무 임계점/소모 → 인접 카드에 최강 유닛 복제  │
│  성장: 나무 수 비례 ATK+HP 증강                    │
│  세계수: 유닛 ≤20 → 매 라운드 숲÷30×0.1 곱연산 누적  │
│  플레이: 나무를 키워 숲을 깊게, 유닛은 ≤20 유지.    │
│  범용 태그: #생체 #중형~#대형                    │
└────────────────────────────────────────────────┘

┌─ 포식종 ─────────────────────────────────────────┐
│  판타지: 끝없이 밀려오는 벌레 떼.                 │
│  유닛:   최다 물량, 최약 개체 (평균 ATK 3.3)     │
│  고유 엔진: 유닛=자원 (약한 유닛이 곧 재료)       │
│  부화: 최약 유닛 2~3기 대량 즉시 스폰             │
│  변태: 약한 유닛 N기 소모 → 상위 유닛 1기 교체    │
│  플레이: 물량으로 밀 것인가, 진화시킬 것인가.      │
│  범용 태그: #생체 #소형~#중형                    │
└────────────────────────────────────────────────┘

┌─ 군대 ─────────────────────────────────────────┐
│  판타지: 훈련된 정예 부대. 조직적 전투.           │
│  유닛:   중간 물량, 강한 개체 (평균 ATK 5.9)     │
│  고유 엔진: 계급 임계점 시스템                    │
│  징집: 공용 base pool 에서 랜덤 1 종 → 유닛별    │
│        기수 차등 (신병 3 기, 바이커/드론 2 기,    │
│        보병/방벽/플라즈마 1 기). 2026-04-21 자동. │
│  훈련: 계급 +1 → R4 강화 변환, R10 엘리트 보너스. │
│  플레이: 계급 도달 속도 + 유닛 공급 총량.         │
│  범용 태그: #생체+#기계 혼합                     │
└────────────────────────────────────────────────┘

┌─ 중립 ─────────────────────────────────────────┐
│  판타지: 테마에 속하지 않는 범용 카드             │
│  키워드:  Layer 1 결과 범주 직접 사용             │
│           ("유닛 추가 시", "강화 시" — 테마 무관)  │
│  플레이:   어떤 테마에든 들어감. 굿스터프 덱 코어  │
│  범용 태그: 혼합                                 │
└────────────────────────────────────────────────┘
```

## 테마별 키워드와 Layer 구조

```
Layer 1 (결과 범주, 테마 무관)         Layer 2 (키워드, 테마별)
───────────────────────────         ─────────────────────────
🔧 유닛 추가                   ←── 제조(스팀) / 번식(드루) / 부화(포식종) / 징집(군대)
📈 강화                        ←── 개량(스팀) / 성장(드루) / 변태(포식종) / 훈련(군대)
⚡ 충전 (테마 무관)             ←── 모든 테마 공유. 발동 상한 +1.
⬆ 합성                        ←── 시스템 이벤트 (테마 무관)
⚔ 공격 / 💀 사망 / 🛡 버프    ←── 전투 이벤트 (테마 무관)

키워드 독립 원칙:
  "제조 시" → 제조에만 반응 (번식/부화/징집에 반응 안 함)
  "유닛 추가 시" → 모든 테마의 유닛 추가에 반응 (결과 범주)
  → 테마 내 시너지 vs 테마 간 시너지의 선택

중립 카드:
  Layer 1 결과 범주를 직접 사용 ("유닛 추가 시", "강화 시")
  → 어떤 테마 카드의 키워드에도 반응 = 범용 접착제
```

## 빌드 아키타입 (테마 기반)

```
순수 테마:      스팀펑크 10장 + 중립 보조 → 테마 내 키워드 시너지 극대화
굿스터프:       중립 14장 중심 + 테마 1~2장 → Layer 1 반응으로 범용 시너지
크로스 테마:    스팀펑크 + 포식종 혼합 → "제조로 유닛 추가 → 포식종 카드가 유닛 추가에 반응"
                Layer 1 결과 범주가 다른 테마를 연결
```
