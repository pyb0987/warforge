# 커맨더 7종

> 런 시작 시 1명 선택. 게임 규칙 자체를 변경하여 최적 전략이 달라짐.
> 커맨더는 "런 전체 규칙 변경" — 가장 강력한 패시브.
> 부적과 조합하여 시작 조합 수백 가지.

## 해금 순서

```
초기 해금: 도박꾼, 양성가 (2종)
업적 해금: 나머지 5종 (replay.md 참조)
```

## 상세

```
[🎲 도박꾼] — ★3 올인, 공격적 리롤

시작 보너스: 시작 골드 +3 (13→16)
패시브1: 리롤 50% 확률 무료 (실패 시 1골드)
패시브2: ★합성 시 합성에 사용된 카드의 구매 비용 합계의 50% 골드 환급

  전략: 공격적 리롤 → 같은 카드 3장 수집 → ★3 올인.
  리롤 무료 확률 50%로 평균 리롤 비용 0.5골드. ★합성 환급으로 ★3 비용 부담 완화.
  위험: 무료 실패 시 골드 소모, ★3 실패 시 자원 낭비 → "도박" 컨셉.
  최적 빌드: 소수 카드를 ★3으로 극한 강화. T1/T2 카드 ★3 집중.

[🌱 양성가] — 물량 극대화, 유닛 상한 돌파

시작 보너스: 시작 카드 각각에 composition 무작위 유닛 +2기
패시브1: 카드당 유닛 상한 +20 (60→80)
패시브2: 유닛 추가 시 30% 확률로 추가 1기

  전략: 유닛 상한 돌파 + 추가 확률로 물량 극대화.
  상한 80 = 다른 커맨더 60에서 멈출 때 계속 성장. 후반 압도적 물량 차이.
  30% 추가 유닛 = R1부터 즉시 체감. 제조/부화/징집 모두에 적용.
  시작 +2기 = 초반 전투력 보강.
  최적 빌드: 포식종(부화), 중립 시작기+증폭기, 물량 보스 보상(R4-3 연쇄반응로, R12-6 물량의 법칙).

[⚒️ 단조사] — 업그레이드 정예

시작 보너스: 런 시작 시 커먼 업그레이드 3택1 → 선택한 카드 1장에 부착
패시브1: 업그레이드 슬롯 +1 (5→6)
패시브2: 커먼 업그레이드 비용 -1 테라진 (4→3)

  전략: 업그레이드를 빠르게 쌓아 소수 카드를 극한 강화.
  슬롯 6개 = 에픽 2개+레어 4개 가능. 타 커맨더 대비 업그레이드 1개 여유.
  커먼 3테라진으로 초반부터 업그레이드 부착 가능 → 빠른 질적 전환.
  최적 빌드: 스팀펑크(소수 정예), 드루이드(세계수), 에픽 업그레이드 집중.

[📐 전략가] — 배치 지휘, 체인 최적화

시작 보너스: 없음 (필드+1이 충분한 보너스)
패시브1: 필드 시작 크기 +1 (6→7)
패시브2: "인접" 효과 범위 2칸 확장 (양쪽 1칸→2칸)
영웅 능력: 빌드 페이즈에서 1회, 카드 2장의 배치를 교환 (tenure 유지)

  전략: 카드 1장 더 + 인접 범위 확장 + 배치 교환 = 체인 극대화.
  필드 7장 = 카드 1장 추가. 보스 보상 필드+1과 합치면 최대 9장 가능.
  인접 2칸 = 대지의 울림(인접 유닛 추가)이 2칸까지 도달. 체인 범위 2배.
  영웅 능력 핵심: 교환 시 tenure(체류 카운터) 유지.
    판매→재배치하면 성장/유닛/tenure 모두 리셋되지만, 교환은 전부 보존.
    인내 카드(고대의 잔해 2R, 고대의 각성 4R)와 시너지 극대.
  최적 빌드: 중립 시작기+브릿지 풀배치, 인내 카드, 스팀펑크(제조→개량 긴 체인).

[📚 수집가] — 다양성, 합성 안 함

시작 보너스: 시작 상점에 T2+ 카드 4장 보장
패시브1: 필드 위 서로 다른 카드 종류 1개당 전체 아군 ATK +4% (영구 누적 아님, 현재 상태 기준)
패시브2: 필드 위 서로 다른 카드 종류 5개 이상이면 매 라운드 +1 테라진

  전략: 다양한 카드를 1장씩 수집. ★합성 안 함.
  필드 6장이 모두 다른 카드 = ATK +24%. 7장(전략가/보스)이면 +28%.
  합성을 안 하므로 ★2/★3 파워 없음 → 대신 카드 슬롯 효율 극대화.
  5종 이상 시 매 라운드 테라진 → 업그레이드 경제 확보.
  최적 빌드: 굿스터프(다양한 테마 혼합), 차원 행상인(테마 다양성 보상).

[⚔️ 약탈자] — 연승, 전투 중심

시작 보너스: 시작 카드 전투 스탯 ATK +20% (영구)
패시브1: 전투 승리 시 추가 +2골드
패시브2: 전투 승리 3회 누적마다 무료 커먼 업그레이드 획득

  전략: 초반부터 전투에서 이겨서 경제+업그레이드 확보.
  ATK +20% 시작 = R1~R3 승리 보장. 승리 보너스 + 2골드 = 경제 우위.
  승리 3회마다 커먼 업그레이드 = 테라진 없이도 업그레이드 축적.
  연패 후에도 누적 카운터 유지 → 복구 경로 존재.
  최적 빌드: 군대(훈련+전투 버프), 야생의 힘(전투 ATK), 강한 전투 카드 중심.

[💰 연금술사] — 에픽 구매, 테라진 경제

시작 보너스: 없음 (패시브가 충분한 보너스)
패시브1: 에픽 업그레이드가 상점에 등장 (가격: 16 테라진)
패시브2: 매 라운드 시작 시 +1 테라진

  전략: 다른 커맨더는 에픽을 보스 보상으로만 얻지만, 연금술사는 "산다".
  에픽 16테라진: 매 라운드 +1테라진 + 전투 보상(승3/패2) → 약 10~12R에 1개 구매 가능.
  시작 테라진은 기본(2)이므로 초반 업그레이드는 여유 없음 → 중반 이후 에픽으로 역전.
  런당 에픽 1~2개 추가 구매 가능 → 보스 에픽과 합치면 총 3~4개.
  최적 빌드: 에픽 업그레이드 집중 (광폭화+연쇄폭발 등 조합), 테라진 경제 보스 보상.
```

## 커맨더 간 선택 긴장

```
🎲 도박꾼 vs 📚 수집가:  ★3 올인 vs 합성 안 함 = 완전히 반대 전략
🌱 양성가 vs ⚒️ 단조사:  물량(상한 돌파) vs 정예(업그레이드) = 유닛 수 vs 유닛 질
📐 전략가 vs ⚔️ 약탈자:  배치 최적화 vs 전투 극대화 = 전략 vs 공격
💰 연금술사:             "에픽을 산다" = 테라진 경제

커맨더 × 테마 최적 조합:
  🎲 × 포식종: 알 둥지 ★3 → 부화 5기 + 양쪽 부화 3기 = 물량 폭발
  🌱 × 포식종/중립: 상한 80 + 30% 추가 유닛 → 물량 극한. 물량의 법칙(R12) 시너지.
  ⚒️ × 스팀펑크: 병기창에 에픽 6개 장착 → 소수 정예 극한
  📐 × 중립: 7장 필드 + 인접 2칸 + 배치 교환 → 인내 카드 + 체인 극대화
  📚 × 굿스터프: 서로 다른 카드 6종 = ATK +24% + 차원 행상인 시너지
  ⚔️ × 군대: 초반 ATK +20% → 연승 → 훈련으로 누적 강화
  💰 × 아무 테마: 에픽 상점 구매 → 에픽 3~4개 장착. 광폭화+연쇄폭발 조합.
```
