# 보스 보상 (27개)

> R4/R8/R12 보스 격파 시 전용 풀에서 4개 표시, 1개 선택. 스킵 불가.
> 보스별 9개 (⚡즉시 2 / 🔄영구 4 / 💥직접 2 / 📐구조 1).
> 유형별 설계 원칙은 game-loop.md 참조.

## R4 보스 — "기반"

빌드 방향을 설정하는 수준의 보상.

```
[R4-1] ⚡ 긴급 보급
카드 1장 선택: ★승급 1회 + 4 테라진
  초반 핵심 카드를 ★2로 올려 R5~R7 안정화.

[R4-2] ⚡ 용병단 초청
즉시 상점에 T3+ 카드 6장 표시 + 10 골드
  높은 티어 카드를 저렴하게 수집할 기회.

[R4-3] 🔄 연쇄 반응로
영구: 유닛 추가 시 50% 확률로 추가 1기
  물량 빌드의 핵심 규칙. 모든 제조/부화/징집에 적용.

[R4-4] 🔄 이중 리롤
영구: 리롤 시 상점 카드 6→8장 표시
  상점 효율 대폭 증가. 원하는 카드를 찾을 확률↑.

[R4-5] 🔄 강화 증폭기
영구: 강화(ATK%) 효과 +50% 증폭 (3%→4.5%)
  성장 체인의 효율을 1.5배. 후반 누적 차이가 큼.

[R4-6] 🔄 자동 징집
영구: 매 라운드 시작 시 전체 아군 카드에 유닛 1기 추가
  빌드 무관 물량 증가. R5부터 매 라운드 +8유닛.

[R4-7] 💥 정예 각성
카드 1장 선택: 해당 카드 유닛 전체 ATK +30%, HP +30%
  핵심 카드 즉시 강화. 대상 선택이 의미 있는 결정.

[R4-8] 💥 비상 증원
카드 1장 선택: 해당 카드에 유닛 5기 추가
  초반 물량 확보. T3+ 카드의 유닛 기반 빠르게 구축.

[R4-9] 📐 확장 벤치
영구: 벤치 +2칸
  카드 보관 여유 → 합성 재료 확보 용이.
```

## R8 보스 — "전환"

R4보다 확실히 강력한 파워 스파이크.

```
[R8-1] ⚡ 대규모 징집
카드 1장 선택: ★승급 1회 + 무료 에픽 업그레이드 3택1 부착
  ★승급 + 에픽 동시 획득. 단일 카드 극대화.

[R8-2] ⚡ 전쟁 기금
즉시 15골드 + 8 테라진
  경제 대량 주입. 상점 + 업그레이드 동시 투자.

[R8-3] 🔄 과부하 엔진
영구: 전체 카드 발동 상한 +1
  체인 길이 확장. 모든 카드의 max_activations +1.

[R8-4] 🔄 전리품 회수
영구: 전투 승리 시 +1골드 → +3골드, 패배 시 +2 테라진
  경제 규칙 변경. 승리하면 부유, 패배해도 테라진 확보.

[R8-5] 🔄 광역 강화장
영구: 강화 시 인접 카드에도 50% 강화 적용
  단일 강화가 범위 강화로. 배치 순서가 더 중요해짐.

[R8-6] 🔄 승전 의지
영구: 전투 승리 시 전체 아군 카드 강화(ATK +3%, 영구 누적)
  승리할수록 강해지는 눈덩이 효과.

[R8-7] 💥 유전자 강화
카드 1장 선택: 유닛 수 2배 + 무료 레어 업그레이드 부착
  단일 카드 물량 + 업그레이드 동시 획득.

[R8-8] 💥 일괄 증원
전체 아군 카드에 유닛 3기씩 추가
  전체 물량 확보. 분산형 보상.

[R8-9] 📐 전선 확장
영구: 필드 +1칸
  카드 1장 추가 배치. 빌드 무관 범용 이득.
```

## R12 보스 — "완성"

단독으로 게임을 바꾸는 수준.

```
[R12-1] ⚡ 궁극 진화
카드 2장 선택: 각각 ★승급 1회
  2장 동시 ★승급. ★3 2장 또는 ★2 2장 선택.

[R12-2] ⚡ 제국의 유산
즉시 전체 아군 카드 강화(ATK +20%, HP +20%) + 15 테라진
  전체 즉시 강화 + 테라진. R13~R15 대비.

[R12-3] 🔄 과부하 연쇄
영구: 전체 카드 발동 상한 +2
  R8-3의 강화 버전. 체인 길이 대폭 확장.

[R12-4] 🔄 이중 합성
영구: ★합성 시 보너스 2배 (×1.30→×1.60, 무료 업그레이드 2개)
  합성 가치 극대화. 남은 합성이 있을수록 강력.

[R12-5] 🔄 전장의 메아리
영구: 전투 중 아군 사망 시 남은 아군 ATK +3%
  물량 빌드에서 사망이 곧 강화. 전멸 직전이 가장 강함.

[R12-6] 🔄 물량의 법칙
영구: 유닛 50기 이상인 카드는 ATK ×1.5, 공격속도 ×1.3
  물량 빌드 전용 극강 보너스. 50기 조건이 빌드 방향을 확정.

[R12-7] 💥 신의 일격
카드 1장 선택: ATK ×2, 에픽 업그레이드 무료 부착
  단일 카드 최강화. ATK 곱연산 + 에픽.

[R12-8] 💥 군단 복제
카드 1장 선택: 유닛 수 3배
  물량 극대화. 30유닛 카드 → 90유닛.

[R12-9] 📐 차원 균열
영구: 필드 +1칸 + 업그레이드 슬롯 전체 +1
  구조 확장 복합 보상. 카드 1장 + 슬롯 1개.
```
