# 부적 12종

> 런 시작 시 커맨더와 함께 1개 선택. 게임 규칙을 소폭 변경하여 플레이 방식을 조절.
> 커맨더가 "방향"이라면, 부적은 "세기". 한 문장으로 설명 가능한 단순한 규칙 변경.
> 이름은 효과를 직접 설명하지 않는 구체적 오브젝트. 효과를 읽고 "그래서 이 이름이구나" 하는 느낌.

## 설계 원칙

```
1. 한 문장 규칙: 부적 1개 = 규칙 변경 1개. 복합 효과 금지.
2. 교차 시너지: 특정 커맨더 1종 전용이 아니라, 여러 커맨더와 다른 맛의 시너지.
3. 순간 생성:  매 라운드 "이 부적 덕분에!" 체감 순간이 있어야 함.
4. 선택 긴장:  12종 중 "항상 최고"인 부적이 없어야 함.
5. 커맨더 하위: 부적은 빌드를 정의하지 않음. 커맨더가 정의한 방향을 보조.
6. 행동 변형:  절반 이상의 부적이 플레이어 행동을 바꿔야 함 (수동 부스트만 금지).
```

## 해금 순서

```
초기 해금 (3종):
  부싯돌, 양면 동전, 금간 해골
  → 이해 쉽고 행동 변형이 명확. 첫 런부터 효과 체감 가능.

업적 해금 (5종):
  "성장 효과 50회 발동"         → 수은 방울
  "★2 카드 3장 보유"           → 깨진 알
  "전투에서 아군 유닛 수 우위 승리" → 전쟁 북
  "업그레이드 8개 장착"         → 녹슨 렌치, 터진 자루
  "카드 5장 판매"              → 영혼 항아리

난이도 클리어 해금 (4종):
  난이도 2 클리어 → 유리 눈
  난이도 3 클리어 → 구리 전선
  난이도 5 클리어 → 황금 주사위
  난이도 7 클리어 → 전쟁 북 (고난이도 보상 — 업적과 택1 선해금)
```

## 상세

### 성장 부적 (3종)

```
[부싯돌] — "시작부터 터진다"

효과: 매 라운드 첫 번째 성장 이벤트의 효과량 2배.
  - 첫 번째 카드(맨 왼쪽)의 첫 발동에 적용.
  - "효과량 2배": 유닛 추가면 추가 수 2배, 강화면 수치 2배.
  - 라운드당 1회만. 첫 이벤트가 무엇이든 적용.

전략: 맨 왼쪽에 가장 강한 성장 효과를 가진 카드를 배치.
  "어떤 카드를 첫 번째에 놓을 것인가"가 매 라운드 결정.
  → 배치 순서의 첫 자리 경쟁이 생김.

시너지:
  ◎ 양성가: 첫 부화 2배 = R1부터 큰 시작. 물량 가속.
  ◎ 전략가: 필드 7장 중 최적 첫 카드 선택지 넓음.
  ○ 도박꾼: ★2/★3 카드의 강화된 첫 효과가 2배 = 큰 스파이크.
  ○ 포식종: 부화 2배 (유충 4~6기 → 8~12기). 즉시 체감.
  ○ 드루이드: 성장 효과 2배 = 소수 유닛의 스탯이 크게 점프.

안티 시너지:
  · 수집가: 다양한 카드라 첫 효과가 약할 수 있음. 하지만 첫 카드를
    성장 특화로 두면 충분히 활용 가능.
```

```
[수은 방울] — "강화가 진해진다"

효과: 강화 효과(개량/성장/변태/훈련) 발동 시 효과량 +25%.
  - 모든 강화 키워드에 범용 적용.
  - "효과량": ATK +2 → +2.5(반올림 +3), HP +10 → +12 등.
  - 유닛 추가에는 미적용 (강화 이벤트만).

전략: 강화 중심 빌드에 안정적 부스트.
  화려한 순간은 없지만, 매 발동마다 조금씩 더 강해짐.
  → "15라운드 누적" 관점에서 가장 안정적인 부적.

시너지:
  ◎ 드루이드: 성장 효과가 핵심. 느린 성장이 +25% 가속.
  ◎ 군대: 훈련(전체 강화)이 +25% = 집단 버프 강화.
  ○ 단조사: 정예 카드의 강화가 더 강해짐.
  ○ 스팀펑크: 개량 효과 +25%.

안티 시너지:
  △ 양성가: 유닛 추가 중심이라 강화 발동 빈도가 상대적으로 낮음.
    하지만 강화가 발동하면 많은 유닛에 적용되므로 나쁘지 않음.
```

```
[깨진 알] — "진화한 카드가 더 낳는다"

효과: ★2 이상 카드의 유닛 추가 효과 발동 시, 추가 유닛 +1기.
  - ★2, ★3 모두 적용.
  - 모든 유닛 추가 키워드(제조/번식/부화/징집)에 적용.
  - ★1 카드에는 미적용. ★합성이 gate 역할.

전략: ★2를 빨리 만들수록 일찍 혜택.
  "★합성 동기 강화" = 도박꾼/합성 빌드와 자연스러운 시너지.
  ★3까지 가면 유닛 추가 효과가 매우 커짐.

시너지:
  ◎ 도박꾼: ★3 올인 + 유닛 추가 +1 = 합성할수록 강해지는 성장.
  ○ 양성가: ★2 만들면 부화+1. 30% 추가와 독립 적용.
  ○ 포식종: 부화(2~3기) + 1기 = 대량 스폰 강화.
  ○ 스팀펑크: 제조+1. 안정적 유닛 확보.

안티 시너지:
  ✕ 수집가: 합성 안 하는 전략이므로 ★2 자체가 드묾.
    하지만 "핵심 카드 1장만 ★2" 변칙 전략 가능.
```

### 전투 부적 (2종)

```
[금간 해골] — "한 번은 버틴다"

효과: 각 유닛의 첫 치사 피격을 HP 1로 생존 (유닛당 1회/전투).
  - 전투 시작 시 모든 아군 유닛에 "불사" 상태 부여.
  - HP가 0 이하가 되는 첫 번째 피격에만 적용.
  - 이미 HP 1인 유닛이 다시 맞으면 정상 사망.

전략: 사실상 모든 유닛에 "추가 생명 1회".
  물량 빌드에서는 전체 생존 시간이 대폭 증가.
  소수 정예에서는 핵심 유닛의 생존이 보장.

파워 조절:
  - 유닛당 1회/전투 (라운드 아님).
  - 카드당 상한 검토 필요 (예: 카드당 최대 20기까지 적용).
    → 양성가(80기 상한)에서 전원 불사는 과잉 가능.
    → 플레이테스트 후 상한 조정.

시너지:
  ◎ 양성가: 대군 전원 1회 생존 = 물량의 압도적 생존력.
  ◎ 드루이드: 소수 정예 유닛이 한 번 더 생존 = 시간 벌기.
  ○ 군대: 강한 유닛이 생존하면 훈련 버프 유지.
  ○ 포식종: 약한 유충도 1회 생존 → 변태 시간 확보.

안티 시너지:
  · 약탈자: 나쁘진 않지만, 이미 초반 ATK가 높아 빠르게 이기는
    전략이라 생존 부적보다 공격 부적이 더 효율적.
```

```
[전쟁 북] — "수가 곧 힘이다"

효과: 전투 시작 시 아군 유닛 수가 적보다 많으면, 적 전체 ATK -10%.
  - 전투 시작 시 1회 판정. 전투 중 유닛 수 변동은 미반영.
  - 아군 ≤ 적이면 효과 없음 (0 or -10%, 중간 없음).
  - 적 ATK 감소는 전투 종료까지 유지.

전략: 물량을 유지하면 전투에서 유리해지는 bridge 부적.
  성장 페이즈에서 유닛을 많이 키울수록 전투가 안정적.
  → 성장(물량) → 전투(ATK 감소) 연결이 생김.
  물량 우위가 "더 강하다"가 아니라 "적이 약해진다"라서
  스케일링이 캡됨 (최대 -10%). 과잉 파워 우려 없음.

시너지:
  ◎ 양성가: 유닛 상한 80 + 30% 추가 = 거의 항상 수적 우위.
    물량 극대화의 전투 보상이 됨.
  ◎ 포식종: 부화로 유충이 쏟아짐 = 수적 우위 용이.
  ○ 군대: 징집으로 병력 확충. 중후반 수적 우위 가능.
  ○ 전략가: 필드 7장 = 카드 1장 더 = 유닛 수 우위.

안티 시너지:
  △ 드루이드: 소수 정예라 유닛 수가 적보다 적을 수 있음.
    번식이 느려 수적 우위 달성이 어려움.
  △ 단조사: 정예 빌드라 유닛 수 보다 질에 투자.
    수적 우위 조건 충족이 불확실.
```

### 경제 부적 (3종)

```
[유리 눈] — "원하는 카드가 보인다"

효과: 리롤 시 이미 보유 중인 카드의 등장 확률 ×1.15.
  - "보유 중": 필드 + 벤치에 있는 카드.
  - 기존 확률에 1.15 곱셈. 확률 자체가 15%p 증가가 아님.
  - 리롤할 때마다 적용.

전략: ★합성 목표 카드를 찾기 쉬워짐.
  "이 카드 한 장만 더 있으면 ★2인데..." 상황에서 빛남.
  리롤 효율 증가 = 골드 절약 효과.

시너지:
  ◎ 도박꾼: ★3 올인 + 카드 찾기 가속 = 가장 직관적 조합.
    리롤 50% 무료 + 찾기 쉬움 = ★3 달성률 대폭 증가.
  ○ 양성가: 핵심 카드 ★2 빠르게 확보.
  ○ 단조사: 정예 카드 ★2 → 무료 레어 업그레이드 빠르게 획득.

안티 시너지:
  ✕ 수집가: 합성 안 하는 전략이므로 같은 카드 찾기가 무의미.
    보유 카드가 다양할수록 "아무 카드나" 확률이 올라가긴 하지만
    의도적 시너지는 아님.
```

```
[양면 동전] — "운이 좋으면 득, 나쁘면 실"

효과: 매 라운드 상점 카드 중 랜덤 1장이 50% 할인, 다른 1장이 50% 할증.
  - 6장 중 할인 1장 + 할증 1장 + 정상가 4장.
  - 할인/할증 대상은 랜덤 (플레이어 선택 불가).
  - 리롤 시 할인/할증 대상도 변경.
  - 할인된 카드가 마침 원하는 카드면 큰 이득,
    원하는 카드가 할증이면 추가 비용.

전략: 매 라운드 "할인 카드가 쓸 만한가?" 판단.
  "원래 안 살 카드인데 싸니까 사볼까?" = 새로운 탐색 행동.
  할증 카드가 원하는 카드면 → 울며 겨자 먹기 or 리롤.
  → 매 라운드 상점에서 의사결정이 풍부해짐.

시너지:
  ◎ 수집가: 다양한 카드를 사므로 할인 카드를 활용할 기회가 많음.
    "원래 안 쓸 카드인데 싸니까" = 다양성 확대.
  ○ 도박꾼: 리롤이 잦으므로 할인 대상을 여러 번 리롤할 수 있음.
  ○ 전략가: 7장 필드를 채울 때 할인 카드 활용.
  ○ 양성가: 싼 카드 구매 → 유닛 키우기.

안티 시너지:
  △ 단조사: 특정 정예 카드만 필요. 할인 카드가 맞을 확률 낮음.
    할증이 정예 카드에 걸리면 손해.
  △ 약탈자: 이미 골드가 풍부해 할인의 가치가 상대적으로 낮음.
```

```
[녹슨 렌치] — "떼었다 다시 붙인다"

효과: 업그레이드를 카드에서 분리(제거) 가능. 분리 시 테라진 50% 환급.
  - 기본 시스템에는 업그레이드 분리가 불가능.
  - 분리한 업그레이드는 소멸 (다른 카드에 재부착 불가).
  - 빌드 페이즈에서만 분리 가능.

전략: 업그레이드 실험의 자유.
  "이 커먼 빼고 레어로 교체"가 가능해짐.
  잘못 붙인 업그레이드도 손해를 줄이며 수정 가능.
  → 새로운 행동(분리)이 생기는 유일한 부적.

시너지:
  ◎ 단조사: 6슬롯 + 분리 가능 = 업그레이드 최적화 극한.
    커먼으로 초반 버티다가 레어/에픽으로 교체.
  ◎ 연금술사: 에픽 상점 구매 → 기존 커먼 분리 → 에픽으로 교체.
  ○ 약탈자: 승리 보상 커먼을 후반에 레어로 교체.

안티 시너지:
  △ 수집가: 업그레이드를 많이 쌓는 편이 아니라 활용도 낮음.
    하지만 5종+ 테라진 보너스로 업그레이드 실험 가능.
```

### 빌드 부적 (2종)

```
[터진 자루] — "선택지가 넘친다"

효과: 매 라운드 업그레이드 상점 표시 +1개 (2→3).
  - 상점에 업그레이드가 3개 표시.
  - 리롤 시에도 3개씩 표시.
  - 등급 분포는 기존 규칙 유지 (70% 커먼 / 30% 레어).

전략: 원하는 업그레이드를 찾을 확률 50% 증가.
  업그레이드 중심 빌드에 안정적 이득.
  특정 태그 한정 업그레이드를 찾기 쉬워짐.

시너지:
  ◎ 단조사: 6슬롯 + 선택지 3개 = 최적 업그레이드 조합 확보.
  ◎ 연금술사: 에픽 포함 3개 표시 = 에픽 등장 빈도 증가.
  ○ 약탈자: 커먼 업그레이드 보상 + 상점 3개 = 업그레이드 풍부.

안티 시너지:
  · 없음. 범용 이득이지만 업그레이드를 덜 사는 빌드에서는 기회비용.
```

```
[구리 전선] — "충전이 옆으로 퍼진다"

효과: 업그레이드 슬롯을 모두 채운 카드의 업그레이드 효과가
  인접 카드에도 30% 적용.
  - 기본 5슬롯 기준: 5개 모두 부착 시 발동.
  - 단조사(6슬롯): 6개 모두 부착 시 발동.
  - "30% 적용": ATK +3 업그레이드 → 인접 카드에 ATK +0.9(반올림 +1).
  - 특수효과(스플래시, 관통 등)는 전파되지 않음. 수치 효과만.
  - 풀슬롯 카드가 필드 양 끝이면 한쪽에만 전파.

전략: 핵심 카드의 업그레이드가 양쪽으로 퍼지는 배치 퍼즐.
  "풀슬롯 카드를 어디에 놓을 것인가?" = 새로운 배치 결정.
  풀슬롯 카드를 중앙에 놓으면 양쪽 모두에게 전파.
  양 끝에 놓으면 한쪽에만 집중 전파.
  → 업그레이드 투자 + 배치 순서가 동시에 중요해짐.

시너지:
  ◎ 단조사: 6슬롯 + 커먼 할인 = 빠른 풀슬롯 달성. 인접 카드까지 강화.
    정예 카드 1장의 업그레이드가 양옆에 퍼짐 = "정예 허브" 배치.
  ◎ 연금술사: 에픽 업그레이드의 수치 부분이 인접에 전파.
    에픽(ATK +5 등) × 30% = 인접에 +1~2.
  ○ 전략가: 필드 7장 + 인접 2칸 = 전파 효과가 더 넓게 느껴짐.
    풀슬롯 카드를 중앙에 놓고 양쪽 2칸씩 영향.
  ○ 약탈자: 승리 커먼으로 빠른 풀슬롯 → 인접 강화.

안티 시너지:
  △ 수집가: 다양한 카드에 분산 투자라 풀슬롯 달성이 느림.
  △ 양성가: 업그레이드보다 유닛 수에 투자. 풀슬롯 우선순위 낮음.
```

```
[황금 주사위] — "한 번 더 던진다"

효과: 보스 보상 선택지 +2 (4→6).
  - R4/R8/R12 보스 클리어 후 보상 선택에 적용.
  - 풀 크기(9개)는 변하지 않음. 표시만 6개.
  - 풀의 67%를 볼 수 있음 = 원하는 보상을 거의 확실히 찾을 수 있음.
  - 🔄영구규칙 등장 확률: 96% → 100% (4개 중 최소 1개 확정).
  - 📐구조 등장 확률: 44% → 67%.

전략: 15라운드 중 3회만 효과가 있지만, 보스 보상이 런을 정의.
  "15라운드 연속 효과를 포기하고, 보스 3회에서 최적 보상을 고른다."
  = 명확한 트레이드오프. 고수일수록 보상 가치를 정확히 평가하므로
  이 부적의 진가가 올라감.

시너지:
  ◎ 전략가: 📐구조(필드+1) 확률 44%→67%. 필드 9장 가능성 대폭 증가.
  ◎ 도박꾼: ⚡즉시(★승급) + 🔄영구규칙을 원하는 대로 선택.
    보스 보상 최적화 = ★3 빌드의 완성도 극대화.
  ○ 단조사: 💥직접강화(에픽 부착) 확률 증가.
  ○ 모든 커맨더: 보스 보상은 범용적으로 강력.

안티 시너지:
  · 없음. 하지만 보스 라운드 외에는 빈 슬롯 = 기회비용이 높음.
    "3번의 최고 선택 vs 15라운드 매 라운드 효과" — 진짜 고민.
```

```
[영혼 항아리] — "떠난 자의 힘이 남는다"

효과: 라운드당 첫 번째 카드 판매 시, 판매 카드 유닛의 절반을
  필드 위 모든 카드에 균등 배분.
  - 라운드당 1회만 적용. 두 번째 판매부터는 일반 판매.
  - 배분: 유닛 절반 ÷ 필드 카드 수 (반올림). 각 카드에 랜덤 유닛 추가.
  - 카드 상한(60기) 초과 시 초과분은 소멸.
  - 판매 골드 환급은 정상 적용 (추가 효과).

전략: 판매가 "성장 행위"가 됨.
  "이 카드를 팔아서 나머지를 키운다" = 새로운 결정 축.
  R1~R3에 싼 카드를 사서 유닛을 키운 뒤, 판매해서 배분하는 전략.

시너지:
  ◎ 수집가: 다양한 카드를 돌려가며 구매→성장→판매→배분.

안티 시너지:
  △ 도박꾼: 카드를 판매보다 합성에 사용하므로 판매 빈도 낮음.
    하지만 합성 재료 외 카드를 정리할 때 활용 가능.
```
